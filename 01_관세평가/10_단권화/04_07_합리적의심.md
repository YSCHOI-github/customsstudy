# 관세평가 단권화 교과서

---

```
┌──────────────────────────────────────────────────────────────┐
│ 제4장 제7절  기타 제1방법을 적용할 수 없는 경우와 합리적 의심      │
│ 출제 티어: ★★ B                                               │
│ 출제율: ~27% (제38회·2016 출제 / 50점 문제의 부분 논점으로 빈출)  │
│ 핵심 법령: 관세법 제30조 제4항·제5항, 시행령 제24조              │
│ 연계 절: → 제4장 제3~6절 [배제사유 4가지], → 제2장 제1절 [평가방법 적용순위] │
└──────────────────────────────────────────────────────────────┘
```

## [1] 핵심 요약 (3줄 이내)

- **제4장 제7절**은 제30조 제3항(법정 배제사유 4가지) 이외에 제1방법을 적용할 수 없는 **추가 상황**과, 신고가격에 **합리적 의심(Reasonable Doubt)**이 있는 경우의 세관장 조치를 규율한다.
- 세관장은 시행령 제24조 소정의 사유가 있으면 납세의무자에게 소명자료를 요구하고(제30조 제4항), 소명 후에도 인정이 곤란하면 제31조~제35조 방법으로 전환한다(제30조 제5항).
- 빈출 패턴: ①합리적 의심 3가지 사유 열거, ②세관장 절차(소명 요구 → 미소명 시 전환), ③수출판매 불성립 등 기타 적용 불가 유형 서술형 출제.

---

## [2] 구조화된 이론 분석

### 2-1. 체계 개관 — 제1방법 적용 불가 사유의 전체 지형

제1방법을 적용할 수 없는 사유는 크게 세 층위로 구분된다.

| 층위 | 근거 조문 | 내용 | 본 절 해당 여부 |
|------|----------|------|----------------|
| ① 법정 배제요건 | 관세법 제30조 제3항 1~4호 | 처분·사용 제한 / 조건·사정 영향 / 사후귀속이익 / 특수관계 영향 | ✗ (제4장 제3~6절) |
| ② 기타 적용 불가 | 관세법 제30조 제1항·제2항 해석 | 수출판매 불성립 등 구조적 불가 사유 | ✓ **본 절** |
| ③ 합리적 의심 | 관세법 제30조 제4항·제5항, 시행령 제24조 | 신고가격의 신빙성 의심 → 소명 요구 → 미소명 시 전환 | ✓ **본 절** |

---

### 2-2. 기타 제1방법 적용 불가 사유 (구조적 불가)

다음에 해당하는 경우에는 법정 배제사유(제30조 제3항) 여부를 검토하기 전에, 제1방법 자체를 적용하는 것이 **구조적으로 불가**하다.

#### (1) 수출판매(Sale for Export) 불성립

> **의의**: 관세법 제30조 제1항은 "우리나라에 수출하기 위하여 판매된 물품"을 전제하므로, **수출판매가 성립하지 않으면** 거래가격이 존재하지 않아 제1방법 적용 불가.

**수출판매 불성립 유형 정리**

| 유형 | 설명 | 적용 가능 방법 |
|------|------|----------------|
| 임차(Lease) 수입 | 소유권 이전 없는 대여 | 제6방법(시행규칙 제7조의4) |
| 무상 공급 | 증여·견본·샘플 등 대가 없음 | 제2~6방법 순차 검토 |
| 위탁판매 수입 | 수입 시점 미판매, 조건부 | 판매 확정 후 신고 가능; 그 전엔 제1방법 불가 |
| 보세구역 내 거래(B/L 양수도) | 수입항 도착 후 제3자에게 양도 | 양수가격이 수출판매가격 요건 충족 시 예외 인정 (→ 제3장 참조) |
| 반품·수리 후 반입 | 최초 수출판매 이미 완료; **관세법 제101조** 등 특수 규정 적용 사례 (수출판매 불성립과 구별) | 수리비(실제지급가격)를 기초로 제1방법 |
| 부품·재료 무상 제공 후 완제품 수입 | **법정 가산요소(관세법 제30조 제1항 제3호)** — 수출판매 불성립이 아님; 생산지원비용 가산 후 제1방법 적용 | 생산지원비용 가산(시행령 제20조) 후 제1방법 |

> [WTO협정 제1조 주해]: "판매된(sold)"은 **반드시 소유권 이전을 수반하는 상업적 거래**를 의미하며, 임차·증여·공탁은 이에 해당하지 않는다.

---

#### (2) 실제지급가격이 금액으로 특정되지 않는 경우

- 가격이 **물물교환(Barter)** 등 비금전적 방식으로 결정된 경우 (금액으로 환산 불가한 경우)
- 단, 금액으로 **환산·수량화 가능**하면 제1방법 적용 가능 (관세법 제30조 제1항 단서)

---

#### (3) 납세의무자가 제1방법에 의한 가격신고를 하지 않은 경우

- 납세의무자 스스로 제2방법 이하로 신고한 경우 세관장은 이를 검토하여 수용 여부 결정
- 단, 신고 순위를 임의로 건너뛸 수는 없으며(평가방법 적용순위 → 제2장 제2절 참조), 세관장은 적용 가능성을 순차 심사

---

### 2-3. 합리적 의심 (Reasonable Doubt)

#### (1) 의의 및 법적 근거

합리적 의심이란, 세관장이 납세의무자가 신고한 가격이 **사실과 다를 수 있다는 합리적 근거가 있는 경우**, 납세의무자에게 입증자료 제출을 요구하는 제도이다.

- 근거: 관세법 제30조 제4항, 제5항 / 시행령 제24조
- WTO협정 제17조: "이 협정의 어떤 규정도 세관이 신고된 가격의 진실성 또는 정확성을 의심할 권리를 제한하는 것으로 해석되어서는 아니된다."

> [WTO협정 제17조 해설(WCO)]: 합리적 의심은 세관의 **직권적 심사권**이며, 이를 행사하기 위해서는 단순한 의심을 넘어 **합리적이고 구체적인 근거**가 있어야 한다. 세관이 부당하게 거래가격을 부인하는 수단으로 남용되어서는 안 된다.

---

#### (2) 합리적 의심 사유 (시행령 제24조) — ★ 빈출 열거 대상

세관장이 신고가격을 과세가격으로 인정하기 곤란하여 납세의무자에게 소명자료 제출을 요구할 수 있는 경우:

| 번호 | 사유 | 비고 |
|------|------|------|
| ①(제1호) | 신고된 가격이 **동종·동질물품 또는 유사물품**의 가격과 **현저한 차이**가 있는 경우 | 가장 빈출 사유 |
| ②(제2호) | **동일한 공급자**로부터 계속하여 수입하고 있음에도 신고한 가격에 **현저한 변동**이 있는 경우 | 동일 공급자 거래 연속성 체크 |
| ③(제3호) | 신고된 물품이 원유·광석·곡물 등 **국제거래시세**가 공표되는 물품인 경우, 신고가격이 시세와 **현저한 차이**가 있는 경우 | 국제시세 공표 물품에 적용 |
| ④(제4호) | 납세의무자가 **거래처를 변경**한 경우로서 신고한 가격이 종전의 가격과 **현저한 차이**가 있는 경우 | 거래처 변경 시 적용 |
| ⑤(제5호) | 그 밖에 제1호부터 제4호까지에 준하는 사유로서 세관장이 신고된 가격이 **의심스럽다고 인정할 만한 합리적인 사유**가 있는 경우 | 포괄적 사유, 구체적 근거 요함 |

> **"현저한 차이"의 판단**: 관세법령상 수치 기준이 명시되어 있지 않으며, **거래의 상업적 맥락**, 시장가격 변동, 특수관계 여부 등을 종합적으로 고려하여 판단한다 (WCO 지침 참조).

---

#### (3) 합리적 의심의 절차 — 세관장 조치 단계

```
[STEP 1] 합리적 의심 사유 확인 (시행령 제24조 ①②③호)
       ↓
[STEP 2] 납세의무자에게 신고가격의 사실 부합 여부를 
         증명할 수 있는 자료 제출 요구 (관세법 제30조 제4항)
         ※ 제출 기한: 요구일로부터 **15일** 이내 (시행규칙 제5조의2)
       ↓ 자료 제출 및 소명
  ┌────────────────────────────────┐
  │ 소명 성공 (신빙성 입증)         │ → 제1방법 유지 (거래가격 수용)
  └────────────────────────────────┘
       ↓ 소명 불충분 또는 자료 미제출
[STEP 3] 세관장, 제1방법 적용 배제
         제31조~제35조 방법으로 과세가격 결정 (관세법 제30조 제5항)
```

> ※ **핵심**: 세관장이 먼저 의심 사유를 특정해야 하며, 납세의무자는 소명의 기회를 반드시 부여받아야 한다. 소명 기회 부여 없이 제1방법을 배제하는 것은 위법.

---

#### (4) 납세의무자의 소명 방법 및 자료

납세의무자가 거래가격의 진실성을 입증하기 위해 제출할 수 있는 자료:

1. **상업서류**: 계약서, 인보이스, 신용장, 은행 결제서류
2. **가격결정 자료**: 가격결정 방법 설명서, 이전가격 연구(Transfer Pricing Study)
3. **시장자료**: 동종업계 가격표, 공개 가격 데이터베이스
4. **회계자료**: 구매자·판매자의 재무제표, 원가 분석 자료
5. **특수관계 자료**: APA(사전가격확인) 승인서, 정상가격 산출 근거

---

#### (5) 합리적 의심과 거래가격 배제사유의 관계 — 중요 비교

| 구분 | 배제사유 (제30조 제3항) | 합리적 의심 (제30조 제4항·5항) |
|------|----------------------|-------------------------------|
| 성격 | **법정 배제요건** (해당 시 자동 배제) | **절차적 심사권** (소명 기회 부여 후 판단) |
| 적용 순서 | 배제사유 해당 여부 먼저 판단 | 배제사유 해당 여부 불분명할 때 보완 수단 |
| 납세의무자 권리 | 배제요건에 해당하면 거래가격 불인정 | **소명 기회 반드시 부여** |
| 입증 책임 | 원칙: 세관 / 납세의무자도 반박 가능 | 세관이 의심 사유 특정 → 납세의무자가 소명 |
| 배제 후 방법 | 제31조~제35조 순차 적용 | 동일 (제30조 제5항) |

## [3] 판단 흐름도

```
[수입신고 접수]
      ↓
[STEP 1] 수출판매 성립 여부 확인
      ↓ 불성립 → 제2방법부터 제6방법까지 순차 적용
      ↓ 성립
[STEP 2] 제30조 제3항 법정 배제사유(①~④) 해당 여부
      ↓ 해당 → 제4장 제3~6절 각 배제사유별 처리
      ↓ 비해당
[STEP 3] 시행령 제24조 합리적 의심 사유(①~⑤) 해당 여부
      ↓ 해당 없음 → 거래가격 수용 (제1방법 적용)
      ↓ 해당 있음
[STEP 4] 납세의무자에게 소명자료 제출 요구 (제30조 제4항)
      ↓ 소명 성공 → 거래가격 수용 (제1방법 유지)
      ↓ 소명 실패 / 자료 미제출
[STEP 5] 제1방법 배제 (제30조 제5항)
         → 제31조~제35조 순차 적용
```

## [4] 계산 공식 및 실수 포인트

> 본 절은 주로 **서술형·판단형** 문제로 출제되며, 직접 계산 문제는 드물다.
> 단, "현저한 차이" 판단의 전제로 비교 수치를 제시하는 사례 문제는 출제됨.

■ **합리적 의심 적용 판단 체크 포인트**

```
┌────────────────────────────────────────────────────────────┐
│  합리적 의심 성립 여부 판단 (시행령 제24조 제1항)               │
│                                                            │
│  ① (제1호) 동종·동질/유사물품 가격 대비 현저한 차이? (Y/N)     │
│  ② (제2호) 동일 공급자 수입가격에 현저한 변동? (Y/N)           │
│  ③ (제3호) 국제거래시세 공표 물품의 시세 대비 현저한 차이? (Y/N)│
│  ④ (제4호) 거래처 변경 후 종전 가격 대비 현저한 차이? (Y/N)    │
│  ⑤ (제5호) 기타 합리적·구체적 의심 근거 존재? (Y/N)            │
│                                                            │
│  → ①~⑤ 중 하나라도 Y이면: 소명자료 제출 요구 가능             │
└────────────────────────────────────────────────────────────┘
```

■ ⚠ 실수하기 쉬운 포인트 체크리스트

- □ 합리적 의심은 거래가격을 **자동 배제**하지 않는다 — 반드시 소명 기회를 부여해야 함
- □ 세관장이 배제사유(제30조 제3항)와 합리적 의심(제30조 제4항)을 **혼동**하지 말 것
  - 배제사유 → 법정 요건 해당 시 직접 배제
  - 합리적 의심 → 절차(소명 요구) 거쳐야 배제 가능
- □ WTO협정 제17조의 취지: 세관의 심사권을 **인정**하되, 자의적 부인을 **제한**하는 양방향 균형 규정임
- □ "현저한 차이"의 수치 기준은 **법령에 없음** — 종합적 판단이므로 답안에 수치를 임의로 쓰지 말 것
- □ 소명 성공 후에는 제1방법 **유지** — 세관이 배제사유에 해당한다고 재결정하려면 별도 근거 필요

`📌 최근 심화 경향 (2022~2025)`
- 2021년 제38회에서 합리적 의심 관련 요건을 **독립 소문항**(물음3·4)으로 출제하여 7점+8점 배점
- 최근 특수관계 심화 경향과 맞물려, 특수관계 거래에서 세관이 합리적 의심을 근거로 소명을 요구하는 복합 패턴 출제 증가 가능성
- WTO협정 제17조(세관의 권리) 조문을 직접 인용하며 논증을 전개하는 답안이 고득점 패턴
