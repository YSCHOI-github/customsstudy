# 관세평가 단권화 교과서

---

```
┌──────────────────────────────────────────────────────────────────┐
│ 제5장 제4절  용기 및 포장비용 (容器費用 및 包裝費用)                   │
│ 출제 티어: ★ C  (단독 출제 희소, 복합문제 내 세부 쟁점으로 반복 등장)    │
│ 출제율: 약 36% (2017·2019·2020년 복합문제 내 포함)                  │
│ 핵심 법령: 관세법 제30조 제1항 제2호, 관세평가 운영에 관한 고시 제18조   │
│ 연계 절: → 제5장 제5절 [생산지원비용] (용기·포장재 무상 제공 시 교차)    │
│          → 제3장 제2절 [실제지급가격] (가격 포함 여부 기준)             │
└──────────────────────────────────────────────────────────────────┘
```

## [1] 핵심 요약 (3줄 이내)

- **구매자가 부담한 용기비용·포장비용으로서 실제지급가격에 포함되지 않은 것은 과세가격에 가산한다.** (관세법 제30조 제1항 제2호)
- **용기비용**은 수입물품과 **관세목적상 일체로 취급되는 용기**에 한하고, **포장비용**은 포장재료비와 포장 노무비를 모두 포함한다. (관세법 제30조 제1항 제2호, 관세평가 운영에 관한 고시 제18조)
- 시험 빈출 패턴: 복합 거래문제에서 ①생산지원비용(시행령 제18조)과의 구분, ②파견 인력 부대비용(항공료·체류비)의 가산 여부가 핵심 쟁점이다.

---

## [2] 구조화된 이론 분석

### 2-1. 법적 근거 및 의의

1. **가산요소로서의 근거** (관세법 제30조 제1항 제2호)
   - 의의: 거래가격(실제지급가격)에 포함되지 않은 용기비용·포장비용은 과세가격 결정 시 별도 가산
   - 취지: 수입물품이 수입자에게 인도되기까지의 실질적 비용을 과세가격에 반영
   - [WTO 관세평가협정 제8조 제1항 (b)(ii)]: *"the cost of packing, whether for labour or materials"* — 포장비용은 재료비와 노무비를 모두 포함하는 것으로 명시

2. **가산 기본 요건**
   - 요건 ①: **구매자가 부담**한 비용일 것 (판매자 부담분은 실제지급가격에 통상 반영)
   - 요건 ②: 실제지급가격(송품장 가격)에 **포함되지 아니한** 것일 것
   - 요건 ③: 수입물품과 **관련된** 용기·포장일 것 (수입 전 발생 비용)

---

### 2-2. 용기비용 — 관세목적상 일체 취급 여부가 핵심

1. **용기비용의 정의** (관세평가 운영에 관한 고시 제18조)
   - 수입물품과 **관세목적상 일체로 취급되는 용기**에 관한 비용
   - 판단 기준: 해당 수입물품의 HS 품목분류 시 용기를 물품과 함께 분류하는지 여부

2. **관세목적상 일체 취급의 판단**

| 구분 | 예시 | 과세가격 가산 여부 |
|------|------|------------------|
| **일체 취급 O** (가산 O) | 정밀기계 전용 케이스, 향수병, 독자 사용 불가 특수 포장재 | ✅ 가산 |
| **일체 취급 X** (가산 X) | 반복 사용 가능한 컨테이너, 별도 HS 분류되는 포장 용기 | ❌ 미가산 (별도 신고) |

> **⚠ 주의**: 재사용 가능한 운송용 컨테이너는 일반적으로 수입물품과 별도로 취급 → 용기비용이 아니라 운임의 일부 또는 별도 물품으로 처리

---

### 2-3. 포장비용 — 재료비와 노무비 모두 포함

1. **포장비용의 범위** (관세법 제30조 제1항 제2호)
   - **포장재료비**: 포장에 사용된 재료(박스, 완충재, 포장지 등)의 구입비용
   - **포장 노무비**: 포장 작업에 투입된 인력의 인건비

2. **포장비용 가산의 핵심 판단: 수입 전/후 구분**

| 발생 시점 | 과세가격 포함 여부 |
|----------|-----------------|
| **수입 전** (수출국, 제3국에서 발생) | ✅ 가산 대상 |
| **수입 후** (수입항 도착 이후 발생) | ❌ 제외 (공제요소 또는 국내 비용) |

---

### 2-4. ★★★ 핵심 쟁점 — 생산지원비용과의 구분

**구매자가 포장재·용기를 무상으로 판매자에게 제공한 경우**, 어느 조항이 적용되는가?

| 구분 | 적용 조항 | 근거 |
|------|---------|------|
| 구매자가 **무상 제공**한 용기·포장재료 | **생산지원비용** (시행령 제18조) | 판매자에게 무상 또는 인하된 가격으로 제공한 물품·재료 |
| 구매자가 **직접 비용 지급**한 포장비용 | **용기·포장비용** (관세법 제30조 제1항 제3호) | 실제지급가격에 미포함된 별도 지급 비용 |

> **핵심 논리**: 포장재를 **무상 제공**하면 생산지원비용, **현금으로 별도 지급**하면 포장비용. 단, **양쪽 모두 과세가격에 가산**된다는 결론은 동일함.

---

### 2-5. 파견 인력 부대비용의 가산 여부 (반복 출제 쟁점)

구매자가 포장을 위해 수출국에 인력을 파견하는 경우 발생 비용:

| 비용 항목 | 성격 | 과세가격 가산 여부 |
|----------|------|-----------------|
| 포장재료비 | 직접적 포장비용 | ✅ 가산 (관세법 제30조 제1항 제2호) |
| 포장 노무비 (인건비) | 직접적 포장비용 | ✅ 가산 (관세법 제30조 제1항 제2호) |
| 파견 왕복 항공료 | 간접 부대비용 | ⚠ 쟁점 → 포장비용의 일부로 가산 가능 |
| 파견 체류비용 | 간접 부대비용 | ⚠ 쟁점 → 포장비용의 일부로 가산 가능 |

> **실무·시험 처리**: 파견 항공료·체류비는 포장작업을 수행하기 위해 **불가피하게 발생한 비용**으로 보아 포장 노무비에 준하여 **가산하는 것이 일반적 해석**. 단, 답안 작성 시 쟁점임을 명시하고 근거 조문과 함께 논거를 제시하는 것이 고득점 전략.

## [3] 판단 흐름도

```
[구매자가 부담한 용기·포장 관련 비용 발생]
          │
          ▼
[STEP 1] 실제지급가격(송품장)에 포함되어 있는가?
          │ YES → 이미 과세가격에 반영됨 → 별도 가산 불필요
          │ NO
          ▼
[STEP 2] 포장재·용기를 판매자에게 무상 제공한 것인가?
          │ YES → 생산지원비용 (시행령 제18조) 적용 → 가산
          │ NO (구매자가 현금으로 직접 지급)
          ▼
[STEP 3] 용기비용인가, 포장비용인가?
          │
          ├─ [용기비용] 관세목적상 수입물품과 일체 취급되는가?
          │         │ YES → 가산 (고시 제18조)
          │         │ NO  → 별도 물품 또는 미가산
          │
          └─ [포장비용] 수입 전(수출국·제3국)에서 발생하였는가?
                    │ YES → 재료비 + 노무비 가산 (관세법 제30조 제1항 제2호)
                    │ NO (수입항 도착 이후) → 미가산 (국내 비용)
```

## [4] 계산 공식 및 실수 포인트

### ■ 용기·포장비용 가산액 산출

```
┌─────────────────────────────────────────────────────────┐
│  가산액 = 포장재료비 + 포장 노무비 + 파견 부대비용(가산 시)  │
│                                                         │
│  ※ 이미 실제지급가격에 포함된 금액은 제외                   │
│  ※ 수입 후 발생분은 제외                                   │
└─────────────────────────────────────────────────────────┘
```

**계산 절차**
- STEP 1. 관련 비용 항목을 열거하고 수입 전/후 발생 여부 확인
- STEP 2. 실제지급가격 포함 여부 확인 → 포함이면 제외, 미포함이면 가산 대상
- STEP 3. 무상 제공인지 직접 지급인지 확인 → 생산지원 또는 포장비용으로 분류
- STEP 4. 최종 가산액 합산

### ■ ⚠ 실수하기 쉬운 포인트 체크리스트

- □ **재사용 가능 컨테이너**를 용기비용으로 가산하지 말 것 → 별도 취급
- □ **수입 후 발생한 포장비용**을 가산하지 말 것 → 국내 비용
- □ **판매자가 부담한 포장비용**이 이미 송품장 가격에 반영된 경우 **이중 가산** 주의
- □ 구매자가 포장재를 **무상 제공**했다면 **생산지원비용** 조항 적용 → 조문 혼동 주의
- □ 포장 노무비는 **인건비(노무비)** 포함이지만, 파견 항공료·체류비는 **별도 논거 필요**
- □ WTO 협정 제8조 제1항 (b)(ii) 인용 시 "labour **or** materials" → **OR이 아닌 AND** (모두 포함)

## [5] 기출 기반 사례 응용

| 회차(연도) | 쟁점 (Issue) | 판단 근거 (Rationale) |
|-----------|-------------|---------------------|
| 제34회('17) | 구매자가 수출국에 포장전문 노무자 3인 파견: 항공료 USD 5,000, 체류비 USD 4,000, 노무비 USD 6,000, 포장재료비 USD 3,000 | 포장재료비·노무비 → 관세법 제30조 제1항 제2호 포장비용 가산 / 항공료·체류비 → 포장을 위한 간접비용으로 가산 가능성 있음 |
| 제36회('19) | 구매자가 특수포장재 자체개발 후 판매자에게 무상 제공 (자체개발비 USD 4,000 / 포장작업비 USD 3,000) | 자체개발비 → **생산지원비용** (시행령 제18조, 용기·포장재 무상 제공) / 포장작업비 → **포장비용** (관세법 제30조 제1항 제2호) / 양쪽 모두 가산 |
| 제37회('20) | 풍력 발전세트 복합거래: 구매자가 포장 노무자 파견 관련 다수 비용 발생 | 포장재료비 + 노무비 → 가산 O / 파견 항공료·체류비 → 쟁점, 포장비용으로 포함 시 가산 |

`📌 시험 대응 전략`
- 용기·포장비용은 독립 문제로 출제되지 않으며, **복합 거래 문제(1번 30~50점)**의 세부 비용 항목으로 등장
- 핵심은 **생산지원비용과의 구분**: 무상 제공(생산지원) vs. 현금 지급(포장비용)
- 결론(가산 여부)은 같더라도 **적용 조문이 다름**을 반드시 명시해야 감점 없음
- 파견 부대비용(항공료·체류비)은 **쟁점임을 전제하고 논거 제시** 형식으로 서술
