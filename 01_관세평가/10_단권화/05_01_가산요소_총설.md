# 관세평가 단권화 — 제5장 제1절 가산요소 총설

---

```
┌──────────────────────────────────────────────────────────────────┐
│ 제5장 제1절  총설 및 개론         출제 티어: D (제2~9절의 도입 프레임) │
│ 핵심 법령: 관세법 제30조 제1항 제1~6호, 시행령 제20조 등              │
│ 연계 절: → 제5장 제3~9절 [각 가산요소 상세]                          │
└──────────────────────────────────────────────────────────────────┘
```

---

### 1. 가산요소의 의의

가산요소란 실제지급가격에 **포함되어 있지 않으나**, 과세가격 산정 시 **반드시 더해야 하는 비용·금액**이다.
수입물품의 **진정한 거래가격**을 반영하기 위해 구매자가 별도로 부담한 항목을 과세가격에 편입한다.
(관세법 제30조 제1항 제1~6호)

---

### 2. 가산요소의 종류 — ★ 4가지 완전 암기

| 호수 | 가산요소 | 세부 항목 | 상세 절 |
|------|---------|---------|--------|
| **제1호** | **수수료·중개료** | 판매수수료·중개료 (구매수수료 제외) | → 제5장 제3절 |
| **제2호** | **용기·포장비용** | 수입물품과 일체로 취급되는 용기·포장 | → 제5장 제4절 |
| **제3호** | **생산지원비용** | 구매자가 무료·인하가로 제공한 물품·용역 | → 제5장 제5절 |
| **제4호** | **권리사용료** | 특허·상표·저작권 등 무형재산권 사용 대가 | → 제5장 제6·7절 |
| **제5호** | **사후귀속이익** | 재판매·처분·사용 수익 중 판매자 귀속분 (금액 계산 가능 시) | → 제5장 제8절 |
| **제6호** | **운임·보험료** | 수입항까지의 운송·보험 비용 | → 제5장 제9절 |

> ※ 운임·보험료는 관세법 제30조 제1항 **제6호**에 명시된 법정 가산요소이며, 구체적인 산정 기준은 **시행령 제20조**에 따름

---

### 3. 가산요소의 공통 요건

가산요소로 인정되려면 아래 3가지 요건을 **모두** 충족해야 한다.

1. **실제지급가격에 미포함**: 이미 실제지급가격에 포함된 금액은 가산 불가 (이중 계상 금지)
2. **객관적·수량화 가능**: 일반적으로 인정된 회계원칙에 따른 자료로 금액 확정 가능
3. **수입물품과의 관련성**: 당해 수입물품의 생산·판매와 직접 관련된 비용이어야 함

---

### 4. 가산 불가 항목 — 자주 혼동되는 항목

| 항목 | 이유 |
|------|------|
| **구매수수료** | 구매자 자신을 위한 대리 비용 → 가산 제외 (시행령 제17조의2) |
| **수입항 도착 후 운임·보험료** | 과세 기준점(수입항) 이후 발생 → 가산 제외 |
| **국내 설치·조립·유지비용** | 수입 후 국내에서 발생 → 가산 제외 |
| **수입관세·내국세** | 국내 부과 세금 → 가산 제외 |
| **연불이자** | 요건 충족 시 공제 대상 (→ 제3장 제4절) |

---

### 5. 가산요소 답안 서술 순서 (★ 2025년 기출 물음1 출제 패턴)

> "실제지급가격(공제요소 제외)과 가산요소에 대하여 기술하시오"

```
① 실제지급가격의 의의
② 가산요소 제1호 (수수료·중개료) / 제2호 (용기·포장)
③ 가산요소 제3호 (생산지원비용)
④ 가산요소 제4호 (권리사용료)
⑤ 가산요소 제5호 (사후귀속이익)
⑥ 가산요소 제6호 운임·보험료 (법 §30①6호, 시행령 제20조)
```
