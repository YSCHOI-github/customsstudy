## [1] 핵심 요약 (3줄 이내)

생산지원비용이란 **구매자가 수입물품의 생산을 위해 판매자에게 무료 또는 인하된 가격으로 직·간접으로 공급한 물품·용역·설계·도구의 가치**로서, 실제지급가격에 포함되지 않은 경우 과세가격에 가산한다 (관세법 제30조 제1항 제3호, 시행령 제18조).

**4가지 유형** (물품·용역·설계·도구)의 **가격 산출방식**(무료/인하/임차/구매자 소유)과 **배분(안분)방식**(최초 선적·총예정수량·세관장 승인)이 시험의 핵심이며, 2022·2025년 심화 출제 기준으로 **임가공 계약·분할수입·손모율** 계산이 반드시 동반된다.

> **빈출 패턴**: 복합거래 1번 문항의 구성요소 분류 + 독립 계산문제 (분할수입 안분, 인하공급 가격 산출) 혼합 출제 — 73% 출제율로 A티어 필수 마스터 영역

## [2] 구조화된 이론 분석

### 1. 의의 및 법적 근거

- **의의**: 구매자가 수입물품의 생산·수출을 위해 판매자에게 무료 또는 인하된 가격으로 공급하는 물품·용역 등으로서, 실제지급가격에 미반영된 경제적 가치 (관세법 제30조 제1항 제3호)
- **WTO협정 근거**: [WTO 관세평가협정 제8조 제1항 (b)호]: 구매자가 무료 또는 인하된 가격으로 직접 또는 간접으로 공급한 물품·용역으로서 수입물품의 생산·판매에 사용된 경우 과세가격에 가산한다
- **[WTO협정 제8조 주해 (b)항]**: 구매자가 공급하는 생산지원물품의 가격이란, 공급자가 지급한 취득가격을 말하며, 해당 물품이 구매자에 의해 생산된 경우에는 해당 물품의 생산원가를 말한다
- **가산 전제**: 실제지급가격에 이미 포함된 경우에는 가산 불가 (이중산입 금지)

---

### 2. 생산지원비용 4가지 유형 (시행령 제18조)

| 유형 | 내용 | 대표 사례 | 공급지 제한 |
|------|------|---------|-----------|
| ① **물품** | 수입물품에 결합되는 재료·부품·부분품, 수입물품 생산에 소비되는 물품 | 원자재, 부자재, 반제품, 포장재 | 없음 |
| ② **용역** | 수입물품의 생산을 위한 기술·설계·고안·공학적 작업·예술적 작업 | 기술지원용역, 엔지니어링 용역 | **우리나라(수입국) 외에서 개발된 것** |
| ③ **설계** | 수입물품 생산을 위해 필요한 설계·고안·기술적 작업 | 설계도면, 사양서, 도면 작성 | **우리나라(수입국) 외에서 개발된 것** |
| ④ **도구** | 수입물품의 생산에 사용되는 공구·금형·주조물, 기타 생산에 사용하는 유형물 | 금형, 지그, 치공구, 기계·장비 | 없음 |

> ⚠ **핵심 구분**: 시행령 제18조 제4호 단서 — 기술·설계·고안·공예·디자인은 원칙적으로 가산하되, **"우리나라에서 개발된 것은 제외한다"**. 즉 **국내(우리나라)에서 개발된 설계·기술은 가산 제외**, 수출국이나 제3국에서 개발된 것은 가산 대상이다.

---

### 3. 가격 산출방법 (시행규칙 제4조 제3항)

#### (A) 무료로 공급하는 경우

1. **구매(취득)한 물품·도구 등**: 취득가격 + (수출국까지의 운송비용)
2. **구매자가 직접 생산한 물품**: 생산원가 + (수출국까지의 운송비용)
3. **임차한 경우**: 임차료 (운송비 별도 가산 여부 검토)

#### (B) 인하된 가격으로 공급하는 경우

```
가산금액 = (정상가격 − 실제지급가격) + 수출국까지 운송비용
```

> ※ 정상가격: 시장에서 통상적으로 거래되는 가격 기준

#### (C) 손모분(Waste) 처리 — 물품 유형에 한정

- **손모율이 존재하는 경우**: 실제 사용량(손모 포함)을 기준으로 가산
- 즉, `가산 대상 수량 = 수입물품에 실제 결합된 수량 + 손모분 수량`

#### (D) 국내 부가가치세 포함 여부

- 기계·장비의 임차료에 국내 부가가치세가 포함된 경우, **부가가치세 제외 후 가산** (관세 과세표준에 국내세 불포함 원칙)

---

### 4. 배분(안분)방법 (시행령 제18조의2, 고시 제20조)

생산지원비용이 복수의 수입신고에 걸쳐 분할 수입되는 경우:

| 배분방법 | 내용 | 적용 우선순위 |
|---------|------|------------|
| ① **최초 선적분 전액 배분** | 전체 생산지원비용을 최초 수입신고 시 전액 가산 | 수입자가 선택 |
| ② **총예정수입물량 기준 안분** | 전체 비용 ÷ 총예정수입물량 × 금번 수입수량 | 수입자가 선택 |
| ③ **세관장 승인 방법** | 납세의무자가 합리적 안분방법을 제출, 세관장이 승인 | 기타 방법 필요 시 |

> **핵심 공식 (②)**:
> ```
> 금번 수입분 가산액 = 총 생산지원비용 × (금번 수입수량 ÷ 총예정수입물량)
> ```

---

### 5. 가산 요건 요약

1. **요건 ①** 구매자가 직접 또는 간접으로 공급할 것 (판매자에게 제공)
2. **요건 ②** 무료 또는 인하된 가격으로 공급할 것 (정상가격 이하)
3. **요건 ③** 수입물품의 생산·수출에 사용될 것 (생산 목적성)
4. **요건 ④** 실제지급가격에 포함되지 않은 금액일 것 (미반영분만 가산)
5. **판단기준**: 객관적이고 수량화할 수 있는 자료로 가격이 결정될 것 (관세법 제30조 제2항)
6. **효과**: 요건 충족 시 실제지급가격에 가산하여 과세가격 산출

---

### 6. 가산 제외 항목 (비교 정리)

| 항목 | 가산 여부 | 근거 |
|------|---------|------|
| 우리나라(국내)에서 개발한 설계·기술지원 | **제외** | 시행령 제18조 제4호 단서 — "우리나라에서 개발된 것은 제외" |
| 수입 후 국내에서 이루어지는 설치·시운전 비용 | **제외** | 수입항 도착 후 발생 비용 (공제요소 → 제3장 참조) |
| 구매자가 자신의 계산으로 행한 활동의 비용 (검사비용 등) | **제외** | 간접지급액에 해당하지 않음 (시행령 제20조의2 제2항) |
| 정상가격으로 공급된 물품·도구 | **제외** | 인하 없이 정상가 공급 시 가산요건 미충족 |
| 실제지급가격에 이미 포함된 부자재비 | **제외** | 이중산입 금지 |

## [3] 판단 흐름도

```
[STEP 1] 구매자가 판매자에게 물품·용역·설계·도구를 제공했는가?
         (관세법 제30조 제1항 제3호)
              ↓ 아니오 → 가산 불가
              ↓ 예
[STEP 2] 무료 또는 인하된 가격으로 제공했는가?
              ↓ 정상가격 공급 → 가산 불가
              ↓ 무료 또는 인하
[STEP 3] 4가지 유형 해당 여부 확인 (시행령 제18조)
          - ① 물품 (재료·부분품 등)
          - ② 용역 (우리나라 외에서 개발된 기술지원 등)
          - ③ 설계 (우리나라 외에서 개발된 설계도면 등)
          - ④ 도구 (금형·공구 등)
          ※ 기술·설계·고안·공예·디자인의 경우: 우리나라(국내) 개발이면 가산 불가
              ↓ 비해당 → 가산 불가
              ↓ 해당
[STEP 4] 실제지급가격에 이미 포함되었는가?
              ↓ 포함 → 가산 불가 (이중산입 금지)
              ↓ 미포함
[STEP 5] 가격 산출 (시행규칙 제4조 제3항)
          - 무료 공급: 취득가격 + 수출국까지 운송비
          - 인하 공급: (정상가격 - 지급가격) + 수출국까지 운송비
          - 임차: 임차료 기준
          - 직접생산: 생산원가 + 운송비
              ↓
[STEP 6] 배분방법 결정 (시행령 제18조의2, 고시 제20조)
          - 분할수입이 없는 경우: 전액 가산
          - 분할수입의 경우:
            ① 최초 선적분 전액 OR
            ② 총예정수량 기준 안분 OR
            ③ 세관장 승인 기타방법
              ↓
[최종] 과세가격 = 실제지급가격 + 생산지원비용(안분액) + 기타 가산요소
```

## [4] 계산 공식 및 실수 포인트

### ■ 공식 1: 무료 공급 시 가산금액

```
┌────────────────────────────────────────────────────────┐
│ 가산금액 = 취득가격 + 수출국까지의 운송비용               │
│                                                        │
│ ※ 직접 생산한 경우: 취득가격 → 생산원가으로 대체         │
│ ※ 임차한 경우: 취득가격 → 임차료로 대체                  │
└────────────────────────────────────────────────────────┘
```

### ■ 공식 2: 인하된 가격으로 공급 시 가산금액

```
┌────────────────────────────────────────────────────────┐
│ 가산금액 = (정상가격 − 실제 지급가격) + 수출국까지 운송비  │
└────────────────────────────────────────────────────────┘
```

### ■ 공식 3: 분할수입 시 금번 수입분 배분액 (안분방법)

```
┌────────────────────────────────────────────────────────────┐
│                    금번 수입수량                             │
│  금번 가산액 = 총 생산지원비용 × ─────────────────          │
│                    총 예정 수입수량                          │
└────────────────────────────────────────────────────────────┘
```

### ■ 공식 4: 손모율 있는 원자재 가산

```
┌─────────────────────────────────────────────────────────────┐
│ 가산 수량 = 물품에 결합된 순 수량 ÷ (1 − 손모율)             │
│ 가산 금액 = 단위가격 × 가산 수량                              │
│                                                             │
│ ex) 수입물품 1단위에 재료 10kg 결합, 손모율 5%               │
│     → 실제 투입량 = 10 ÷ 0.95 = 10.526kg → 이 기준으로 가산 │
└─────────────────────────────────────────────────────────────┘
```

### ■ 계산 절차

- **STEP 1.** 4가지 유형별 분류 및 가산 대상 여부 판단
- **STEP 2.** 각 항목별 가산금액 산출 (무료/인하/임차/직접생산 구분)
  - 운송비: 수출국까지(수입국 → 수출국 발생 운송비 포함 여부 확인)
  - 부가가치세: 국내세는 제외
  - 손모율: 주어진 경우 반드시 적용
- **STEP 3.** 분할수입 여부 확인 → 배분방법 결정
- **STEP 4.** 최종 과세가격 = 실제지급가격 + 생산지원가산액 + 기타 가산요소(운임·보험료 등)

### ■ ⚠ 실수하기 쉬운 포인트 체크리스트

- □ **우리나라(국내)에서 개발된 설계·기술은 가산 불가**: 시행령 제18조 제4호 단서 — "우리나라에서 개발된 것은 제외". 국내 설계사에게 의뢰한 설계도면은 **가산 제외**
- □ **수출국 또는 제3국에서 개발된 설계·기술용역은 가산 대상**: 우리나라 외에서 개발된 것이므로 시행령 제18조 제4호 본문 적용
- □ **구매자가 자기의 계산으로 행한 활동의 비용은 가산 불가**: 간접지급액에 해당하지 않음 (시행령 제20조의2 제2항)
- □ **임차료에 포함된 국내 부가가치세는 차감**: 임차비용에 VAT 포함 시 VAT 제외 금액만 가산
- □ **분할수입 안분 시 총예정물량 vs 금번 수량 혼동 주의**: 분모는 총예정수입물량, 분자는 금번 수입수량
- □ **운송비 방향 확인**: 생산지원물품을 구매자 소재지(한국)에서 판매자 소재지(수출국)로 보내는 운송비가 가산 대상 — 역방향(수입항까지)은 관세법 제30조 제1항 제6호 및 시행령 제20조 운임·보험료 가산요소
- □ **실제지급가격 포함 여부 선확인**: 인보이스에 부자재비가 이미 포함된 경우 추가 가산 불가

`📌 최근 심화 경향 (2022~2025)`
- **임가공 계약 구조 심화**: 무상제공·현지구매·임차 등 복수 방법이 혼재하는 복합거래 출제
- **손모율(Waste) 개념 적용 요구**: 2025년 문제4에서 명시적으로 손모율 적용 계산 출제
- **부가가치세 제외 처리**: 국내 임차료 등에 포함된 VAT 처리 요구
- **배분방법 서술 + 계산 혼합**: 배분방법 이론 서술(10점 수준) + 수입신고별 금액 계산(5점 수준) 동시 요구
- **WTO 협정 조문 인용**: 협정 제8조 제1항 (b)호 직접 인용 요구 증가

## [7] 단권화 암기 키워드 & 체크리스트

### ✏ 고득점 키워드 (답안 필수 포함)

1. **"물품·용역·설계·도구"** — 4가지 유형 명칭 (시행령 제18조)
2. **"무료 또는 인하된 가격으로 공급"** — 가산 전제조건
3. **"취득가격 + 수출국까지 운송비용"** — 무료공급 가산금액 산출식 핵심 표현
4. **"우리나라에서 개발된 것은 제외"** — 기술·설계 유형의 지역 요건 핵심 표현 (시행령 제18조 제4호 단서)
5. **"총예정수입물량 기준 안분"** — 분할수입 배분방법
6. **"최초 수입신고 시 전액 가산"** — 배분방법 ① 핵심 표현
7. **"손모분(waste) 포함한 실제 투입량"** — 손모율 적용 가산 근거
8. **"직접 또는 간접으로"** — 공급 형태 (직접공급·제3자 경유 모두 포함)
9. **"실제지급가격에 포함되지 아니한 경우"** — 이중산입 금지 요건
10. **"일반적으로 인정된 회계원칙"** — 가격 산출 자료의 증거력 요건
