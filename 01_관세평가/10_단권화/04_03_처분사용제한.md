# 제4장 제3절 당해물품에 대한 처분 또는 사용의 제한

```
┌──────────────────────────────────────────────────────────────────┐
│ 제4장 제3절  당해물품에 대한 처분 또는 사용의 제한                    │
│ 출제 티어: ★★ B (중요)                                             │
│ 출제율: 45% (5/11개년) — 제33·35·36·39·40회 출제                   │
│ 핵심 법령: 관세법 제30조 제3항 제1호, 시행령 제21조(배제범위)·제22조 제1항(허용)  │
│ 연계 절: → 제4장 제4절 조건·사정에 의한 영향 (금액 계산 가능 여부)    │
│           → 제2장 제2절 평가방법 적용순위                            │
└──────────────────────────────────────────────────────────────────┘
```

## [1] 핵심 요약 (3줄 이내)

- **처분·사용 제한이 거래가격에 "실질적으로 영향을 미치는" 경우**, 수입물품의 거래가격(제1방법)을 과세가격으로 사용할 수 없고, 제2방법 이하로 순차 적용한다 (관세법 제30조 제3항 제1호).
- 단, **거래가격에 영향을 미치지 않는 제한**에 해당하면 배제요건이 아니므로 제1방법 적용이 가능하다 (시행령 제22조 제1항).
- **출제 핵심 패턴**: ① 제한의 유형 3가지(허용 제한) 암기, ② 독점유통계약 조건 항목별 제한 해당 여부 판단, ③ 금액 계산 가능성에 따른 제4장 제4절과의 구분.

---

## [2] 구조화된 이론 분석

### 1. 처분·사용 제한의 배제 원칙

- **의의**: 구매자의 물품 처분·사용에 제한이 부가되어 그 제한이 거래가격에 영향을 미치는 경우, 해당 거래가격은 과세가격으로 인정되지 않는다 (관세법 제30조 제3항 제1호).
- **배제 효과**: 제1방법(거래가격) 적용 불가 → 제2방법(동종·동질)부터 순차 적용

> **[WTO협정 제1조 제1항(a)]**: 구매자의 물품 처분 또는 사용에 제한이 부가된 경우, 단 그러한 제한이 ①법률이나 공공당국에 의한 것이거나, ②물품 판매 가능 지역에 관한 것이거나, ③가격에 실질적으로 영향을 미치지 않는 것에 해당하면 과세가격 적용을 방해하지 않는다고 해석함.

> **[WTO협정 주해(Interpretative Note), 부속서 I]**: "실질적으로 영향을 미친다"는 의미는 거래가격이 그 제한이 없었을 경우와 실질적으로 다른지를 판단하는 것. 제한으로 인해 판매자가 가격을 조정한 경우 영향을 미친 것으로 본다.

---

### 2. 거래가격에 영향을 미치지 않는 제한 (배제요건 해당 X) — 시행령 제22조 제1항

아래 3가지에 해당하면 처분·사용 제한이 있어도 제1방법 적용 가능:

| 유형 | 내용 | 예시 |
|------|------|------|
| **① 법률·공공당국에 의한 제한** | 수입국의 법령 또는 공공당국이 부과하는 제한 | 안전기준, 환경규제, 위생허가 요건 |
| **② 지역 제한** | 물품을 판매할 수 있는 지역에 관한 제한 | 독점판매 지역 지정 (한국 내에서만 판매) |
| **③ 가격에 실질적 영향 없는 제한** | 당해 물품의 가격에 실질적으로 영향을 미치지 않는 것으로 인정되는 제한 | 재판매 전 재포장 의무, 보관 방법 지정 등 |

> ⚠️ **핵심**: 유형 ①②는 제한이 존재하더라도 **자동으로 허용**. 유형 ③은 **실질적 가격 영향 여부를 별도 판단**해야 함.

---

### 3. 거래가격에 영향을 미치는 제한 — 시행령 제21조 (배제요건 해당 O)

아래의 경우 처분·사용 제한이 거래가격에 실질적 영향을 미친 것으로 보아 제1방법 배제 (시행령 제21조가 정하는 제한의 범위에 해당):

1. **판매자에 의한 처분·사용 지시·제약**
   - 구매자가 물품의 사용방법, 처분방법, 판매가격 등을 판매자의 지시에 따라야 하는 경우
   - 예) 판매자가 구매자의 재판매 가격을 직접 통제하는 경우

2. **국내 판매 후 이익 환수 조건** *(→ 실제로는 관세법 제30조 제3항 제3호 사후귀속이익 배제사유에 해당)*
   - 구매자가 당해 물품을 처분하거나 사용한 후 발생한 이익의 전부 또는 일부를 직접·간접적으로 판매자에게 귀속시키는 조건
   - 예) 재판매 이익의 일정 비율을 판매자에게 송금하기로 한 경우
   - ※ 법령상 §30③3호·시행령 제19조의2 소관; 처분·사용 제한(§30③1호)과 구분 필요

3. **구매 조건과 연계된 제한** *(→ 실제로는 관세법 제30조 제3항 제2호 조건·사정 배제사유에 해당; 시행령 제22조 제2항)*
   - 구매자가 특정 물품·서비스를 구매하거나 특정 행위를 이행하는 조건으로 가격이 결정된 경우
   - 예) 10% 할인을 받는 대신 품질검사를 구매자가 이행해야 하는 조건 (→ 2018년 기출, 이 경우 할인가격으로 인정 가능 여부 검토 필요)

### 4. 판단 실무에서의 핵심 쟁점

#### (1) 독점유통계약상의 제한 조건 분석 (2018년 기출 핵심)

| 독점유통 조건 | 처분·사용 제한 해당 여부 | 근거 |
|-------------|----------------------|------|
| 특정 지역에서만 판매 (ⓐ) | **해당 없음** | 시행령 제22조 제1항 제2호 지역 제한 — 허용 |
| 재고 유지 의무 (ⓓ) | **해당 없음** | 가격에 실질적 영향 없음 — 통상적 유통 관행 |
| 전시장·직원 고용 의무 (ⓔ) | **해당 없음** | 가격에 실질적 영향 없음 — 유통망 구축 요건 |
| 광고 활동 의무 (ⓕ) | **검토 필요** | 광고비 부담이 가격 조정과 연동되면 영향 있음 |
| 구매자의 자기 계산 사업 (ⓘ) | **해당 없음** | 손익 독립 → 처분·사용 제한 아님 |
| 품질검사 이행 조건부 할인 (ⓒ) | **조건부 허용** | 할인 연동 조건이지만, 할인가격이 독립적으로 결정 가능하면 제1방법 적용 가능 |

#### (2) 가격인하 원인이 규제 회피인 경우 (2014년 기출)

- **사실**: 개별소비세 부과 회피를 위해 단위당 가격을 $2,000 → $1,400으로 인하
- **쟁점**: 이 인하된 가격을 거래가격으로 인정할 수 있는가?
- **판단**: 조건 또는 사정(수입국 조세부담 회피)에 의해 가격이 영향을 받은 경우 → **시행령 제22조 제2항** 해당 가능 (→ 제4장 제4절 참조)
- 단, 금액으로 계산 가능하면 제1방법 적용 가능, 계산 불가능하면 배제

## [3] 판단 흐름도

```
  [STEP 1] 구매자의 물품 처분·사용에 제한이 있는가?
       ↓ 없음 → 처분·사용 제한 배제요건 해당 없음 → 제1방법 적용 계속
       ↓ 있음
  [STEP 2] 해당 제한이 시행령 제22조 제1항 3가지 중 하나인가?
       ↓ 해당 → 거래가격에 영향 없는 제한 → 배제요건 해당 없음 → 제1방법 적용 가능
       ↓ 해당 없음 (제22조 제1항 해당 불가)
  [STEP 3] 제한이 거래가격에 실질적으로 영향을 미쳤는가? (시행령 제21조 제한 범위 해당)
       ↓ 영향 없음 → 제1방법 적용 가능
       ↓ 영향 있음
  [STEP 4] 해당 제한(조건·사정)의 영향을 금액으로 계산할 수 있는가?
       ↓ 계산 가능 → 제1방법 적용 (실제지급가격 조정)    [→ 제4장 제4절]
       ↓ 계산 불가능
  [최종] 제1방법(거래가격법) 배제 → 제2방법(동종·동질)부터 순차 적용
```
