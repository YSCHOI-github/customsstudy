```
┌──────────────────────────────────────────────────────────────────────┐
│ 제4장 제5절  제1방법 배제요건으로서의 사후귀속이익                         │
│ 출제 티어: ★ C~D (희소·연계 출제)                                       │
│ 출제율: 18% (2/11개년) — 제33회('16)·제39회('22) 직접·복합 출제         │
│ 핵심 법령: 관세법 제30조 제3항 제3호, 제30조 제1항 제3호                 │
│ 연계 절: → 제5장 제8절 사후귀속이익 (가산요소) ← 반드시 함께 학습         │
│           → 제4장 제4절 조건·사정에 의한 영향 (조정 가능 여부 구조 공유)   │
```

## [1] 핵심 요약 (3줄 이내)

- **사후귀속이익(Subsequent Proceeds)** 이란 수입물품의 처분·사용으로 발생한 수익의 일부를 구매자가 직·간접으로 판매자에게 귀속시키기로 한 금액이다 (관세법 제30조 제1항 제5호, 제3항 제3호).
- **적절히 조정(금액 계산) 가능하면 가산요소로 처리(제1방법 유지)**, 금액 계산이 불가능하면 제1방법 배제 요건에 해당한다는 2분법이 이 절 전체의 핵심이다.
- **출제 핵심 패턴**: ① 사후귀속이익의 정의 + 요건 2가지 서술, ② 조정 가능/불가능 판단, ③ 제5장 제8절(가산요소)과의 연결 논술.

---

## [2] 구조화된 이론 분석

### 1. 사후귀속이익의 정의 및 법적 구조

- **의의**: 구매자가 수입물품을 처분(재판매)하거나 사용함으로써 발생하는 **수익(proceeds)의 전부 또는 일부**가 직접 또는 간접으로 판매자에게 귀속되도록 약정된 금액 (관세법 제30조 제1항 제5호, 제3항 제3호)
- **법적 이중성**: 동일한 사후귀속이익이 **조정 가능 여부**에 따라 전혀 다른 법적 결론을 낳는다.

| 조정 가능 여부 | 적용 조문 | 처리 방법 | 결론 |
|-------------|---------|---------|------|
| **가능** (금액 계산 O) | 관세법 제30조 제1항 제5호 | **가산요소**로 실제지급가격에 가산 | 제1방법 **적용 유지** |
| **불가능** (금액 계산 X) | 관세법 제30조 제3항 제3호 | **배제요건** 해당 | 제1방법 **배제** |

> ⚠️ **관세법 제30조 제3항 제3호 단서**: "다만, 제1항에 따라 적절히 조정할 수 있는 경우는 제외한다." → 즉, 조정 가능한 경우는 배제요건에서 **자동으로 제외**되어 가산요소로만 처리됨.

---

### 2. 사후귀속이익의 성립 요건

사후귀속이익으로 인정되려면 아래 **2가지 요건을 모두 충족**해야 한다:

#### 요건 ① 처분·사용으로부터 발생한 수익일 것

- 수입물품을 **재판매**하거나 **사용**함으로써 발생하는 수익이어야 함
- 단순히 물품 대금 외 부수 수익이라도 물품의 처분·사용과 인과관계가 있어야 함
- **제외**: 주식배당금 및 금융서비스의 대가 등 수입물품과 관련이 없는 금액은 제외 (관세법 시행령 제19조의2)
- 예) 재판매 이익, 재임대료 수입, 물품을 활용한 서비스 매출의 일부

#### 요건 ② 직·간접으로 판매자에게 귀속될 것

- 구매자가 수익의 일부를 **판매자에게 직접 지급**하거나, **계약상 귀속 의무**를 부담하는 경우
- **간접 귀속**도 포함: 판매자 지정 제3자에 대한 지급, 가격 조정 형식으로 귀속시키는 경우 등
- 귀속이 **계약에 의해 약정**되어 있어야 함 (사후 임의 지급과 구별)

> **[WTO협정 제1조 제1항(c)]**: 구매자에 의한 물품의 사후 처분 또는 사용으로부터 발생하는 수익의 일부가 직·간접으로 판매자에게 귀속되는 경우, 단 그러한 금액에 대해 제8조에 따라 적절히 조정이 이루어질 수 있는 경우는 제외한다.

> **[WTO협정 제8조 제1항(d)]**: 사후귀속이익이 금액으로 계산 가능한 경우, 실제지급가격에 가산하여 과세가격을 구성한다.

---

### 3. "적절히 조정 가능"의 판단 기준

**조정 가능 = 금액으로 계산 가능 = 객관적·수량화 가능**

| 구분 | 조정 가능 (가산요소) | 조정 불가능 (배제요건) |
|------|-------------------|---------------------|
| **금액 확정 시점** | 수입 시 또는 합리적 기간 내 확정 가능 | 수익 발생 전까지 금액 불확정 |
| **계산 방법** | 계약서에 비율·금액이 명시 | 비율·기준이 없거나 사후 임의 결정 |
| **객관성** | 일반적으로 인정된 회계원칙 등 기반 | 주관적 판단에 의존 |
| **수량화** | 과세 시점에 수치 특정 가능 | 수치 특정 불가 |
| **처리 결론** | 가산 후 제1방법 적용 (→ 제5장 제8절) | 제1방법 배제 → 제2방법 이하 순차 |

**조정 가능의 대표 사례**:
- 재판매 금액의 일정 **비율(%)**이 판매자에게 귀속되도록 계약서에 명시 → 계산 가능
- 연간 총 국내판매액의 5%를 판매자에게 지급 → 계약서 기재 + 회계 자료로 계산 가능 (2022년 기출)

**조정 불가능의 대표 사례**:
- 재판매 후 "이익의 상당 부분"을 귀속시킨다는 막연한 조건 → 계산 불가
- 귀속 금액이 사후 협상으로만 결정되는 경우 → 수입 시점에 수량화 불가

### 4. 사후귀속이익과 유사 개념 구별

#### (1) 사후귀속이익 vs 사후 가격조정(Price Adjustment)

| 구분 | 사후귀속이익 | 사후 가격조정 |
|------|------------|------------|
| **발생 원인** | 물품 처분·사용 후 발생 수익의 귀속 | 수입 시 잠정 결정된 가격의 사후 확정 |
| **귀속 방향** | 구매자 → 판매자 | 양방향 (추가 납부 또는 환급) |
| **법적 처리** | 제30조 제1항 제5호 가산 또는 배제요건 | 잠정가격 신고 후 확정가격으로 정산 (관세법 제28조) |
| **예시** | 재판매 이익의 일부 귀속 계약 | 원자재 가격 연동 조정 조항 |

#### (2) 사후귀속이익 vs 권리사용료 (로열티)

| 구분 | 사후귀속이익 | 권리사용료 |
|------|------------|---------|
| **지급 근거** | 물품 처분·사용으로 발생한 수익의 분배 | 지식재산권(특허·상표·저작권 등) 사용 대가 |
| **법적 근거** | 관세법 제30조 제1항 제5호 | 관세법 제30조 제1항 제4호, 시행령 제19조 |
| **가산 요건** | 적절히 조정 가능할 것 (거래조건 여부와 관계없이 가산, 고시 제23조) | 물품관련성 + 거래조건성 |
| **배제 요건** | 조정 불가능 시 제3항 제3호 | 별도 배제 조항 없음 (요건 미충족 시 불가산) |

## [3] 판단 흐름도

```
  [STEP 1] 구매자가 수익의 일부를 판매자에게 귀속시키기로 한 약정이 있는가?
       ↓ 없음 → 사후귀속이익 해당 없음 → 제1방법 적용 계속
       ↓ 있음
  [STEP 2] 요건 확인
       ① 수입물품의 처분·사용으로부터 발생한 수익인가?
       ② 직·간접으로 판매자에게 귀속되는가?
       ↓ 어느 하나 불충족 → 사후귀속이익 해당 없음
       ↓ 모두 충족 → 사후귀속이익 성립
  [STEP 3] 적절한 조정(금액 계산)이 가능한가? ← 핵심 분기
       ↓ 가능 (금액 수량화 O)
       → 실제지급가격에 가산 → 제1방법 적용 유지
         [→ 제5장 제8절 사후귀속이익(가산요소) 로 처리]
       ↓ 불가능 (금액 수량화 X)
  [최종] 관세법 제30조 제3항 제3호 배제요건 해당
       → 제1방법 배제 → 제31조(제2방법)부터 순차 적용
```

## [4] 제5장 제8절과의 연계 구조 (가산요소로서의 사후귀속이익)

> 이 절(제4장 제5절)은 **배제 여부 판단**, 제5장 제8절은 **가산 방법·계산식** 을 다룬다.
> 시험에서는 두 절을 하나의 논술로 연결하는 경우가 많으므로 반드시 함께 정리할 것.

```
사후귀속이익 성립 확인 (제4장 제5절)
       ↓
  조정 가능? → YES
       ↓
  가산요소 처리 (제5장 제8절)
  ┌─────────────────────────────────────────┐
  │ 가산 원칙: 실제지급가격 + 사후귀속이익     │
  │ 계산 공식: 수입 시 확정 가능 금액 또는     │
  │           계약상 비율 × 국내판매금액       │
  │ 세관 신고: 잠정가격으로 신고 후            │
  │           사후귀속 금액 확정 시 정산        │
  └─────────────────────────────────────────┘
       ↓
  조정 가능? → NO → 제1방법 배제 (제4장 제5절)
```

## [7] 단권화 암기 키워드 & 체크리스트

1. **"처분 또는 사용으로부터 발생하는 수익"**

6. **"금액으로 계산 가능한지 여부"**
7. **"제1방법 배제 → 제2방법부터 순차 적용"**
