## [1] 핵심 요약 (3줄 이내)

- 과세가격 결정방법은 **제1→2→3→4→5→6방법 순서**로 적용하되, 각 방법의 **적용 불가 요건**이 구체적으로 규정되어 있으며 요건 미충족 시에만 다음 방법으로 이행한다.
- 제2·3방법은 **선적일 전후 60일(총 120일) 이내**, **동일 생산국**, **동일 거래단계·수량** 조건을 원칙으로 하며, 조건 차이가 있으면 **가격 조정 후 적용**한다.
- **빈출 패턴**: 종합문제에서 "제1~6방법 순차 적용 가능성 전부 검토"(2017년 50점), 또는 제2·3방법의 **거래단계·수량 차이 조정 계산**(2020년 10점, 2024년 10점)으로 출제.

## [2] 구조화된 이론 분석

### 1. 순차 적용 원칙의 법적 구조

- **원칙**: 과세가격 결정방법은 반드시 **낮은 번호의 방법부터 순서대로** 적용을 검토해야 하며, 앞 방법의 적용이 불가능한 경우에 한하여 다음 방법으로 이행. (관세법 제31조~35조 각 제1항 "제○조의 방법으로 과세가격을 결정할 수 없는 경우")
- **예외**: 납세의무자 신청 시 **제4방법과 제5방법의 순서 교환** 가능. (관세법 제33조 제1항 단서)
- **목적**: 어떤 방법도 적용 불가인 경우 최종적으로 제6방법(합리적 기준) 적용.

### 2. 제2방법 — 동종·동질물품의 거래가격 (관세법 제31조)

#### 2-1. 동종·동질물품의 정의 (시행령 제25조 제1항)

> **동종·동질물품(Identical Goods)**: 평가대상 물품과 **모든 면에서 동일**한 물품. 단, 외관상 경미한 차이가 있더라도 그 차이가 물품의 상업적 가치에 영향을 미치지 않는 경우 동종·동질물품으로 본다.

구체적 요건:

| 요건 | 내용 |
|------|------|
| 물리적 특성 | 재질·구조·성능·디자인 등 모든 면에서 동일 |
| 품질 | 동일한 품질 기준 충족 |
| 명성 | 동일한 상업적 명성·소비자 인식 |
| 생산국 | **동일 국가**에서 생산된 물품 |
| 생산자 | 원칙적으로 **동일 생산자** 제품 우선; 동일 생산자 제품 없으면 다른 생산자 제품도 인정 |

> ⚠ **수입국에서 생산된 물품은 동종·동질물품에서 제외** (관세법 시행령 제25조 제1항 단서)

#### 2-2. 적용 요건 (관세법 제31조, 시행령 제25조)

1. **시기 요건**: 평가대상 물품의 **선적일과 같은 날 또는 거의 같은 시기** (원칙 선적일 전후 **60일**, 총 120일 이내)에 수입된 동종·동질물품의 거래가격. (시행령 제25조 제3항)
2. **거래단계 동일**: 평가대상 물품과 **같은 거래단계(도매·소매 등)** 및 **실질적으로 동일한 수량**으로 판매된 물품의 거래가격.
3. **차이 조정 가능**: 거래단계 또는 수량이 다를 경우 **그 차이를 조정**하여 적용 가능. (→ 제2절 4항 '조정 방법' 참조)
4. **복수 거래가격**: 조건이 동일한 거래가격이 복수인 경우 **가장 낮은 가격** 적용. (관세법 제31조 제3항)

#### 2-3. 적용 불가 사유

- 동종·동질물품의 수입 실적 자체가 없는 경우
- 선적일 전후 60일(총 120일) 이내 수입 실적 없는 경우
- 거래가격 배제요건(제30조 제3항)에 해당하는 경우
- 가산요소에 대한 객관적 자료 부재로 거래가격을 확정할 수 없는 경우 (제30조 제1항 단서)
- 신고가격의 부정 등으로 세관장이 배제하는 경우 (제30조 제5항)

### 3. 제3방법 — 유사물품의 거래가격 (관세법 제32조)

#### 3-1. 유사물품의 정의 (시행령 제26조 제1항)

> **유사물품(Similar Goods)**: 평가대상 물품과 **모든 면에서 동일하지는 않으나**, ① 기능과 용도가 유사하고 ② 상업적으로 대체 가능하며 ③ 구성 재료 및 특성이 유사한 물품.

구체적 요건:

| 요건 | 내용 |
|------|------|
| 기능·용도 | 평가대상 물품과 유사한 기능 및 목적 수행 |
| 상업적 대체 가능성 | 구매자가 상호 대체하여 구매할 수 있는 수준 |
| 구성 재료·특성 | 유사한 재질·성분·제조방식 |
| 생산국 | **동일 국가** 생산 필요 |
| 생산자 | 동일 생산자 우선; 없으면 다른 생산자 인정 |

#### 3-2. 동종·동질물품과 유사물품 비교

| 구분 | 동종·동질물품 (제2방법) | 유사물품 (제3방법) |
|------|---------------------|-----------------|
| WTO협정 | Identical Goods (제1조 제2항) | Similar Goods (제1조 제2항) |
| 물리적 동일성 | **완전 동일** (경미한 차이 제외) | **동일 불필요**, 기능·용도 유사로 충분 |
| 상업적 대체 | 당연 인정 | **명시적 충족 필요** |
| 생산국 요건 | 동일국 생산 | 동일국 생산 |
| 적용 순위 | 제2방법 (우선) | 제3방법 (차순위) |
| 예시 | 동일 모델·규격의 A사 스마트폰 | 다른 모델이지만 동급 성능의 B사 스마트폰 |

#### 3-3. 적용 요건

- 제2방법(동종·동질물품)과 **동일한 시기·거래단계·수량 요건** 적용.
- 복수 거래가격 존재 시 **가장 낮은 가격** 적용. (관세법 제32조 제3항)
- 수입국 생산물품 제외 원칙 동일 적용.

### 4. 제2·3방법 공통 — 거래단계·수량 차이 조정 (시행령 제25조 제4항) ★★★

#### 4-1. 조정의 필요성

동종·동질물품 또는 유사물품이 존재하더라도, 평가대상 물품과 **거래단계(도매↔소매) 또는 수량이 다른 경우** 그 차이를 조정해야 함.

#### 4-2. 조정 가능 요건

조정은 아래 **두 가지 조건을 모두 충족**할 때만 가능:

1. **가격 차이의 원인**: 거래단계 또는 수량의 차이에 **기인**한다고 인정되어야 함.
2. **객관적 자료 존재**: 조정의 근거가 되는 **공표된 가격표(price list)** 등 객관적이고 수량화 가능한 자료에 의해 조정 금액이 명백히 입증되어야 함. (시행령 제25조 제4항)

> ⚠ **핵심**: 가격표 등 객관적 자료가 없으면 조정 자체가 불가 → 해당 방법 적용 불가로 다음 방법으로 이행.

#### 4-3. 거래단계 차이 조정 계산

```
┌─────────────────────────────────────────────────────────┐
│  조정된 과세가격 = 동종(유사)물품 거래가격                   │
│                 × (1 ± 거래단계 차이 조정률)               │
└─────────────────────────────────────────────────────────┘
```

**조정 방향**:
- 기준물품이 **소매 단계** → 평가대상이 **도매 단계**: 가격 **하향** 조정 (도매가 < 소매가)
- 기준물품이 **도매 단계** → 평가대상이 **소매 단계**: 가격 **상향** 조정

#### 4-4. 수량 차이 조정 계산

```
┌─────────────────────────────────────────────────────────┐
│  조정된 과세가격 = 동종(유사)물품 거래가격                   │
│                 × (1 ± 수량 차이 조정률)                  │
└─────────────────────────────────────────────────────────┘
```

**조정 방향**:
- 기준물품 수량이 **평가대상보다 많음** → 기준물품에 수량할인 적용 → 가격 **상향** 조정
- 기준물품 수량이 **평가대상보다 적음** → 가격 **하향** 조정

#### 4-5. 기출 계산 사례 (2020년 제37회 문제3)

**(사례 1)** 거래단계 차이 조정 — 소매→도매

- 평가대상: 판매자 A, 2,800개, 단가 150원 CIF, **도매** 단계
- 동종물품: 판매자 B, 2,800개, 단가 250원 CIF, **소매** 단계 (할인율 15%)

**조정 과정**:

```
[STEP 1] 거래단계 차이 확인: 도매 vs 소매 → 조정 필요
[STEP 2] 객관적 가격표 존재 확인:
          B의 공표 가격표 — 도매 20% 할인, 소매 15% 할인
          ∴ 도매가 = 소매가 × (1 - 20%) / (1 - 15%)
                    → 소매가 기준 도매 환산 가능
[STEP 3] 동종물품의 도매 환산가격 계산:
          소매가(할인 후) = 250 × (1 - 15%) = 212.5원
          도매가(할인 후) = 250 × (1 - 20%) = 200원
          ∴ 조정된 거래가격 = 200원
[STEP 4] 과세가격 결정: 200원 (CIF) 적용 가능
```

> ※ 수량은 동일(2,800개)하므로 수량 조정 불필요.

**(사례 2)** 수량 차이 조정 — 도매 동일, 수량 상이

- 평가대상: 판매자 C, 1,700개, 단가 400원, **도매**
- 유사물품: 판매자 D, 2,300개, 단가 475원, **도매** (2,000개 이상 475원, 미만 500원)

**조정 과정**:

```
[STEP 1] 거래단계 확인: 모두 도매 → 거래단계 조정 불필요
[STEP 2] 수량 차이 확인: 1,700개(평가대상) vs 2,300개(유사물품) → 조정 필요
[STEP 3] 객관적 가격표 확인:
          D의 가격표: 2,000개 미만 → 500원, 2,000개 이상 → 475원
          ∴ 1,700개에 해당하는 D의 가격 = 500원
[STEP 4] 수량 조정 후 유사물품 거래가격 = 500원
[STEP 5] 과세가격 결정: 500원 (CIF) 적용
```

### 5. 제4방법 — 국내판매가격 역산 (관세법 제33조)

#### 5-1. 적용 요건 요약

- **수입 후 국내 판매 실적** 존재 필요.
- **최초 상업적 거래단계**에서 비특수관계자에게 판매된 가격 기준.
- **단위가격 결정**: 수입신고일 또는 그와 거의 동시에 **최대수량**으로 판매된 단위가격. (→ 제6장 제3절 상세)
- **90일 규정**: 수입신고일로부터 **90일** 이내 판매 실적 기준. 90일 초과 시 세관장 인정을 통해 적용 가능.

#### 5-2. 공제 항목

국내판매가격에서 아래 항목을 **역산으로 공제**하여 과세가격 산출:

```
┌──────────────────────────────────────────────────────────┐
│  과세가격 = 국내판매 단위가격                                │
│           − 수입 후 국내 통상의 이윤 및 일반경비            │
│           − 수입항 도착 후 국내 발생 통상의 운임·보험료와 그 밖의 관련 비용 │
│           − 수입 및 국내판매 관련 조세·공과금 (관세 포함)    │
└──────────────────────────────────────────────────────────┘
```

---

### 6. 제5방법 — 산정가격 (관세법 제34조)

#### 6-1. 적용 요건 요약

- 수출자(생산자)로부터 **생산원가 관련 자료** 입수 가능해야 함.
- 수출국 내 동종·동류 물품 생산자의 **통상적인 이윤 및 일반경비** 자료 필요.

#### 6-2. 산정 공식

```
┌────────────────────────────────────────────────────────────┐
│  과세가격 = 생산원가 (재료비 + 노무비 + 제조경비)              │
│           + 수출국 동종·동류 물품 생산자의 통상 이윤·일반경비  │
│           + 수입항까지의 운임·보험료 및 기타 비용             │
└────────────────────────────────────────────────────────────┘
```

> ⚠ 우리나라에서 개발된 기술·설계·디자인 등의 비용은 **생산자(수출자)가 부담하는 경우 산정가격에 포함**된다. 단, 구매자(수입자)가 무상으로 제공하여 생산자가 별도로 부담하지 않은 경우에는 포함하지 않는다 (이중 계산 방지). (시행령 제28조 제2항)

### 7. 제6방법 — 합리적 기준 (관세법 제35조)

#### 7-1. 의의 및 원칙

- 제1~5방법으로 과세가격 결정이 불가한 경우 **최후 수단**으로 적용.
- **합리적 기준**: 제1~5방법의 기초 자료를 **합리적으로 변용·조정**하여 적용하는 방법. 완전히 새로운 기준 창출이 아님.
- [WTO협정 제7조 제1항]: 합리적 기준에 의한 과세가격은 WTO협정의 원칙 및 일반 규정에 부합해야 하며, 수입국에서 입수 가능한 자료에 기초해야 한다.

#### 7-2. 합리적 기준의 예시 (시행령 제29조, 시행규칙 제7조)

| 활용 가능한 자료 | 내용 |
|--------------|------|
| 국내 생산 동종·동류물품의 판매가격 | 단, 직접 기준으로 삼는 것은 금지 (변용 필요) |
| 수출국 내국 판매가격 | 금지 방법이지만, 변용된 형태로는 참고 가능 여부 논란 |
| 제3국 수출 가격 | 참고자료로 활용 가능 (직접 기준 금지) |
| 공표된 가격표·통계 | WCO, 무역통계, 관세청 통계 등 |
| 유사물품 과세가격 | 조정을 통해 변용 적용 |

#### 7-3. 절대적 금지 방법 (WTO협정 제7조 제2항, 관세법 제35조 제2항)

> 제6방법 적용 시에도 아래 방법으로는 **과세가격을 결정할 수 없음**:

| 번호 | 금지 방법 |
|------|---------|
| ① | 수출국에서 생산·제조된 물품의 **수출국 내 국내판매가격** 기준 |
| ② | 두 가격 중 **높은 것** 을 기준으로 하는 이원적 선택 방식 |
| ③ | 수출국의 **생산원가** (단, 제5방법과 달리 변용 없이 직접 사용 금지) |
| ④ | 수입국 내 **동종·동류 국내생산물품**의 가격 |
| ⑤ | **최저 관세가격(minimum customs value)** 제도 |
| ⑥ | **자의적(arbitrary) 또는 허구적(fictitious) 가격** |
| ⑦ | **우리나라 외의 국가에 수출하는 물품의 가격** (제3국 수출 가격) |

---

### 8. 제2·3방법 적용 시 '최근접 시기' 원칙

- 조건에 부합하는 동종·동질 또는 유사물품이 **여러 시기에 수입**된 경우:
  → **평가대상 물품의 선적일에 가장 근접한 시기**에 수입된 물품의 거래가격을 우선 적용. (관세법 제31조 제1항, 시행령 제25조 제3항)
- 최근접 시기 물품이 복수인 경우: **가장 낮은 가격** 적용.

---

### 9. 동종·동질물품 및 유사물품의 공통 제외 사항

- **수입국 생산물품**: 동종·동질물품·유사물품에서 **항상 제외**. (시행령 제25조)
- **생산자 우선 원칙**: 동일 생산자 제품이 존재하면 **우선 적용**, 없는 경우 다른 생산자 제품도 인정.
- **동일 생산국 요건**: 제2·3방법 모두 **생산국이 동일**해야 함.

```
│           + 동종·동류 생산자의 통상 이윤·일반경비          │

└────────────────────────────────────────────────────────┘
```

### ■ ⚠ 실수하기 쉬운 포인트 체크리스트

- □ **제2·3방법 시기 요건은 선적일 전후 60일(총 120일)이 원칙** — 90일은 제6방법(합리적 기준) 신축 해석 기준(시행규칙 제7조 제1항 제2호)이므로 혼동 주의
- □ **조정 근거 자료(가격표 등) 없으면 조정 자체 불가** → 해당 방법 포기하고 다음 방법으로
- □ **복수 거래가격 존재 시 가장 낮은 가격 적용** (제2·3방법 공통) — 높은 가격 선택 불가
- □ **생산국 동일 요건**: 제2·3방법 모두 평가대상 물품과 **동일 생산국** 물품이어야 함
- □ **수입국 생산 물품은 제2·3방법 기준물품에서 항상 제외**
- □ 제5방법에서 **수입국에서 개발된 설계도·디자인 비용은 생산원가에서 제외** (이중 가산 방지)
- □ **제4·5방법 순서 교환**은 세관장이 직권으로 할 수 없고 **납세의무자 신청**이 반드시 필요
- □ 제6방법의 합리적 기준은 **변용**이지 **자의적 새 기준 창출**이 아님 — 협정 기초 유지
