# 관세평가 단권화 교과서

```
┌──────────────────────────────────────────────────────────────────────┐
│ 제5장 제7절  소프트웨어(Software)와 권리사용료                          │
│ 출제 티어: ★ C  (27%, 3/11개년)                                       │
│ 출제율: 27% (제36회·제38회·제42회 출제)                                │
│ 핵심 법령: 관세법 시행규칙 제3조, 관세법 시행령 제19조                    │
│ 연계 절: → 제5장 제6절 권리사용료 [물품관련성·거래조건성 판단]           │
│          → 제5장 제5절 생산지원비용 [마스터 미디어 처리]                 │
│          → 제6장 제5절 제6방법 [보세공장 내 소프트웨어 관련 물품]        │
└──────────────────────────────────────────────────────────────────────┘
```

## [1] 핵심 요약 (3줄 이내)

- **캐리어 미디엄 룰**: 컴퓨터 소프트웨어가 수록된 **담체(매체)**를 수입할 때, 담체 자체의 가격과 소프트웨어 가치를 분리하여 **담체 가격만**을 과세가격으로 할 수 있다 (관세법 시행규칙 제3조). 단, 음악·영화·게임 등 오락용 콘텐츠에는 **적용 불가**.
- **재현권(Right to Reproduce)**: WTO 관세평가협정 주해상 '재현하는 권리' 자체는 권리사용료 가산 대상이 아니나, **재현된 복제물이 수입물품**인 경우 마스터 미디어 비용은 **생산지원비용**으로 가산한다.
- **빈출 패턴**: 마스터테이프·마스터CD·포토마스크가 관련된 복제물 수입에서 ① 재현권 해당 여부, ② 마스터 미디어의 생산지원비용 가산 여부, ③ 권리사용료 가산 여부를 복합 판단하는 문제.

## [2] 구조화된 이론 분석

### A. 캐리어 미디엄 룰 (Carrier Medium Rule)

#### 1. 의의 및 법령 근거

소프트웨어가 수록된 담체(플로피디스크·CD-ROM·USB 등)를 수입하는 경우, 담체 자체의 가치와 소프트웨어의 가치를 분리하여 **담체 가격만**을 과세가격으로 결정할 수 있는 특례 규정이다.

> (관세법 시행규칙 제3조) 컴퓨터 소프트웨어가 수록된 담체를 수입하는 경우에는 해당 소프트웨어의 가치를 제외한 담체 자체의 가격을 과세가격으로 결정할 수 있다.

> [WCO 권고의견 4.1]: 컴퓨터 소프트웨어를 담은 담체의 과세가격 결정 시, 담체 자체의 가격(Cost of the carrier medium)만을 과세가격으로 할 수 있다. 단, 음악, 시네마, 비디오 녹음물 및 그 유사물에는 이 규정이 적용되지 않는다.

#### 2. 적용 범위와 적용 제외

| 구분 | 적용 여부 | 이유 |
|------|----------|------|
| **업무용 소프트웨어** (ERP, CAD, OS 등) | **적용 가능** | 담체와 소프트웨어 가치 분리 가능 |
| **음악 CD·음반** | **적용 불가** | 오락·콘텐츠 자체가 물품의 본질적 가치 |
| **영화·영상 DVD/BD** | **적용 불가** | 오락·콘텐츠 자체가 물품의 본질적 가치 |
| **게임 CD·DVD** | **적용 불가** | 오락·콘텐츠 자체가 물품의 본질적 가치 |
| **데이터베이스 담체** | 사안별 판단 | 데이터 자체가 핵심 가치 → 원칙 적용 불가 |
| **기계에 내장된 S/W** | **적용 불가** | 기계의 불가분한 구성요소 → 기계 가격에 포함 |

> **핵심 판단 기준**: 수입물품의 본질적 가치가 **담체(매체) 자체**인가, 아니면 그 안에 담긴 **콘텐츠·데이터·지시사항**인가?
> - 담체가 본질: 업무용 소프트웨어 → 캐리어 미디엄 룰 적용
> - 콘텐츠가 본질: 음악·영화·게임 → 적용 불가, 전체 가격 과세

### B. 소프트웨어 권리사용료의 가산 여부

#### 1. 일반 원칙: 제5장 제6절 3요건 적용

소프트웨어 관련 권리사용료도 일반 권리사용료와 동일하게 아래 3요건을 충족해야 과세가격에 가산된다 (관세법 시행령 제19조). (**→ 제5장 제6절 상세 내용 참조**)

| 요건 | 내용 |
|------|------|
| ① **물품관련성** | 권리사용료가 수입물품과 관련되어 있을 것 |
| ② **거래조건성** | 권리사용료 지급이 수입거래의 조건일 것 |
| ③ **직·간접 지급** | 구매자가 직접 또는 간접으로 지급하는 금액일 것 |

#### 2. 소프트웨어 특유의 물품관련성 판단

소프트웨어 권리사용료는 "수입물품에 체화(體化)되어 있는가"에 따라 판단이 달라진다.

```
┌─────────────────────────────────────────────────────────────────┐
│ [HS 8523호 담체(CD-ROM·USB 등)에 수록된 컴퓨터소프트웨어 권리사용료] │
│  → 시행령 제19조 제4항: 해당 담체와 관련되지 않는 것으로 의제     │
│  → 가산 불가 (캐리어 미디엄 룰의 법적 근거)                      │
├─────────────────────────────────────────────────────────────────┤
│ [수입물품에 저작권이 체화된 경우 (오락콘텐츠·기계 내장 S/W 등)]   │
│  예) 기계에 내장된 제어 S/W, 게임CD의 게임, 음반의 음악           │
│  → 물품관련성 충족 가능 → 거래조건성 검토 후 가산 여부 결정       │
├─────────────────────────────────────────────────────────────────┤
│ [수입 후 별도 사용되는 소프트웨어 권리사용료]                      │
│  예) 수입 기계의 국내 사용 라이센스, 수입 후 다운로드 소프트웨어  │
│  → 물품관련성 불충족 → 가산 불가                                │
└─────────────────────────────────────────────────────────────────┘
```

### C. 재현권(Right to Reproduce)과 마스터 미디어 처리

#### 1. WTO협정상 재현권 정의 및 비가산 원칙

> [WTO 관세평가협정 제8조 제1항(c) 주해]: 이 항에서의 권리사용료는 **수입물품을 재현하는 권리(right to reproduce)**를 포함하지 않는다.
>
> [관세법 시행령 제19조 제2항]: 우리나라에서 고안이나 창안을 다른 물품에 **재현하는 권리를 사용하는 대가**는 가산 대상 권리사용료에서 제외한다.

즉, 마스터테이프·마스터CD·원판(스탬퍼) 등이 수입물품을 **재현(복제)하는 데 사용되는 권리** 자체에 대한 대가는 권리사용료로 가산하지 않는다.

#### 2. 재현권과 복제물 수입의 구분 — 핵심 구조

| 상황 | 처리 방법 |
|------|----------|
| **재현권 자체** (마스터 복제할 권리) | 권리사용료 가산 대상 **아님** |
| **재현된 결과물(복제물)이 수입물품** | 마스터 제작비용 등 → **생산지원비용**으로 가산 |
| **수입물품에 체화된 저작권** (음반의 음악 등) | 거래조건 충족 시 권리사용료로 **가산** |

---

### D. 마스터 미디어 관련 복합 거래 유형 — 기출 핵심 구조

시험에서 반복 출제되는 마스터 미디어 관련 복합 거래를 유형별로 분석한다.

#### 유형 1: 마스터테이프 → 압인기 → 레이저디스크 (2015·2014 기출)

```
[LCO(미국)] ──── 마스터테이프 무상 제공 ────▶ [SCO(중국)]
                                                    │
                           마스터테이프 → (재현) → 압인기
                                                    │
                           압인기 → (복제) → 레이저디스크(수입물품)
[甲상사(한국)] ──── 구매 ◀───────────────── SCO
              ──── 권리사용료(판매액 5%) ──▶ LCO
```

**과세평가 처리:**

| 항목 | 금액 | 분류 | 과세 여부 |
|------|------|------|----------|
| 레이저디스크 구매대금 | $30,000 (CIF) | 실제지급가격 | **포함** |
| 압인기(Stamper) 제조비용 | $1,000 | 생산지원비용 (도구·금형 해당) | **가산** |
| 마스터테이프 제작비용 | 별도 산정 필요 | 생산지원비용 (설계·개발) | **가산** |
| 권리사용료 (판매액의 5%) | 미확정 | 저작권 사용료 | 요건 충족 시 **가산** |

> **마스터테이프 관련 판단:**
> - 마스터테이프는 레이저디스크 제조에 **필수적으로 사용**됨 (없으면 생산 불가)
> - 이는 **재현권** 자체가 아니라, **생산에 사용된 용역의 결과물**
> - → 시행령 제18조의 **생산지원비용** (개발·설계 비용)으로 가산
>
> **저작권 권리사용료 판단 (물품관련성):**
> - 레이저디스크 자체에 뮤직비디오가 체화됨 → **물품관련성 충족**
> - 판매액 기준 지급 약정 → **거래조건성 충족 여부 추가 검토 필요**
>
> **재현권 해당 여부:**
> - 마스터테이프의 내용을 복제할 권리(재현권) 자체 → 시행령 제19조 제2항에 따라 **가산 대상 아님**
> - 수입물품(레이저디스크)에 수록된 저작권에 대한 사용료 → 시행령 제19조 제3항 제4호에 따라 물품관련성 인정 → **가산 대상**
> - 결론: 재현권 대가는 제외되나, 수입물품에 수록된 저작권 사용료로서 거래조건 충족 시 **가산**

#### 유형 2: 게임소프트웨어 개발 → 마스터CD → 게임CD (2021 기출)

```
[K사(한국)] ──── 개발 의뢰·권리사용료 지급 ──▶ [J사(일본)]
                                                    │
                              마스터CD 무상 제공 ──▶ [C사(중국)]
                                                    │
                마스터CD → (재현) → CD Stamper → (복제) → 게임CD(수입물품)
[K사] ──── 복제비용(XX) + CD Stamper 비용(YY) 지급 ──▶ C사
```

**과세평가 처리:**

| 항목 | 분류 | 과세 여부 | 근거 |
|------|------|----------|------|
| 게임CD 복제비용 (XX) | **실제지급가격** | 포함 | 관세법 제30조 제1항 |
| CD Stamper 비용 (YY) | **생산지원비용** (도구·금형) | **가산** | 시행령 제18조 제2호 |
| 마스터CD 제작비용 (J사 권리사용료) | **생산지원비용** (개발비) | **가산** | 시행령 제18조 제4호 |

> **핵심 포인트**: K사가 J사에 지급한 금액은 "게임소프트웨어 개발 및 마스터CD 제작비용"
> - 이 금액의 성격: 생산에 투입된 **개발·설계 용역 비용** + 마스터CD 자체 제작비
> - 권리사용료보다는 **생산지원비용(시행령 제18조)으로 가산**하는 것이 타당
> - 단, 별도의 저작권 라이센스 조건이 있다면 권리사용료 병행 검토

> **CD Stamper(압인기) 처리:**
> - 시행령 제18조 제2호: "수입물품의 제조에 사용하는 틀·주형 및 이와 유사한 물품"
> - CD Stamper는 수입물품(게임CD) 제조에 사용되는 **금형·도구에 해당** → 생산지원비용

#### 유형 3: 포토마스크 → 보세공장 → ASIC 칩 (2019 기출)

```
[수입자 I] ──── 포토마스크 구매 (5,000 c.u.) ──▶ [수출자 X]
                     │
                무상 제공 ──▶ [보세공장 B]
                                   │
              포토마스크 설계 → 웨이퍼 재현 → ASIC 칩 100,000개
                                   │
                수입자 I ◀─────── 수입물품
```

**과세평가 처리:**

| 항목 | 분류 | 가산 여부 |
|------|------|----------|
| 포토마스크 구매가격 (5,000 c.u.) | 생산지원비용 (금형·도구 유사물) | **가산** |
| 5,000개 초과분 권리사용료 (개당 0.1 c.u.) | 권리사용료 (물품관련성·거래조건성 충족 여부) | 요건 충족 시 **가산** |

> **권리사용료 계산:**
> - 총 100,000개 생산, 5,000개까지 무료 → 초과분 95,000개 × 0.1 = 9,500 c.u.
> - 단, 이 권리사용료는 수출자 X에게 지급 + 수입물품(ASIC 칩)과 관련 + 거래조건 충족
> - → 권리사용료 가산 **가능**

> **포토마스크 경제적 내구연한 배분:**
> - 경제적 내구연한 = 100,000개 생산 가능
> - 포토마스크 비용 5,000 c.u. = 총 100,000개에 배분
> - 개당 생산지원비 = 5,000 ÷ 100,000 = **0.05 c.u./개**

### E. 기계에 내장된 소프트웨어 처리 (2019 기출 연계)

기계 자체에 소프트웨어가 **내장(built-in)**되어 함께 수입되는 경우:

| 구분 | 처리 방법 |
|------|----------|
| S/W가 기계의 불가분한 구성요소 | **기계 과세가격에 포함** (별도 분리 불가) |
| S/W 구입비를 기계 판매자에게 별도 지급 | **실제지급가격의 일부** (별도 지급이나 수입물품과 불가분) |
| 수입 후 다운로드·업그레이드 소프트웨어 | **과세 불가** (수입 후 국내 발생 비용) |

> **2019년 쟁점비용 ④**: S/W가 기계장치에 설치되어 작동되고, 구매자 A가 S/W 구입비 USD 60,000을 판매자 B에게 별도 지급
> - S/W 없이는 기계 작동 불가 → 기계의 필수 구성요소
> - 판매자에게 지급 → **실제지급가격에 포함**

## [3] 판단 흐름도

### 소프트웨어 관련 권리사용료 종합 판단

```
  [START] 소프트웨어 관련 비용 발생
       ↓
  [STEP 1] 수입물품의 성격 확인
       │
       ├─ 소프트웨어 담체(CD·USB 등) 수입
       │      ↓
       │   오락용(음악·영화·게임)? ─YES─▶ 캐리어 미디엄 룰 적용 불가
       │      ↓ NO                         → 전체 가격 과세
       │   캐리어 미디엄 룰 적용 가능
       │   → 담체 자체 가격만 과세가격
       │
       ├─ 기계에 내장된 S/W
       │      ↓
       │   기계의 불가분한 부분 → 기계 과세가격에 포함
       │   기계 판매자에게 별도 지급 → 실제지급가격 포함
       │
       └─ 복제물(레이저디스크·게임CD 등) 수입
              ↓
         [STEP 2] 마스터 미디어 비용 확인
              ↓
         마스터 미디어가 생산에 필수 사용?
              ↓ YES
         생산지원비용(시행령 제18조) 가산
              ↓
         [STEP 3] 별도 권리사용료 지급 있는가?
              ↓
         물품관련성 + 거래조건성 충족?
              ↓ 충족 → 권리사용료 가산
              ↓ 불충족 → 불가산
              ↓
         재현권(right to reproduce) 자체 대가?
              ↓ YES → 가산 불가 (WTO협정 주해)
              ↓ NO  → 체화된 저작권 사용료로 가산 가능
```

## [5] 기출 기반 사례 응용

| 회차(연도) | 쟁점 (Issue) | 판단 근거 (Rationale) |
|-----------|-------------|---------------------|
| 제36회('19) 문제6 | 보세공장 ASIC 칩 포토마스크 권리사용료 가산여부 | 물품관련성·거래조건성 충족 → 가산; 5,000개 초과분 95,000개 × 0.1 c.u. 산정 |
| 제38회('21) 문제4 | 게임CD 복제물: 마스터CD 비용·Stamper 비용 처리 | 마스터CD → 생산지원(개발비); Stamper → 생산지원(금형·도구); 수입물품과의 관련성·체화 여부 핵심 |
| 제36회('19) 문제1 ④ | 기계에 설치된 S/W 구입비 별도 지급 | 기계 작동 필수 구성요소 + 판매자에게 지급 → 실제지급가격 포함 |
| 제32회('15) 문제1 | 뮤직비디오 레이저디스크: 마스터테이프·압인기·권리사용료 | 마스터테이프 → 생산지원(개발); 압인기 → 생산지원(금형); 권리사용료(판매액 5%) → 체화 저작권 가산 |
| 제31회('14) 문제1 | 동일 거래사실 (레이저디스크) 반복 출제 | 동일 구조 적용 |

`📌 최근 심화 경향 (2022~2025)`
- **2025년 제42회**: 소프트웨어 직접 출제 없었으나, 특허권 가산구조(조정액·가산율 계산)가 소프트웨어 권리사용료 계산 구조와 동일 → 제5장 제6절 계산식 숙지가 전제조건
- **WTO협정 주해 직접 인용 강화**: "재현하는 권리"의 비가산 근거를 협정 제8조 제1항(c) 주해로 명시하는 답안 작성 요구
- **마스터 미디어의 이중 성격**: 생산지원비용 vs 권리사용료 중 어느 것으로 처리할지의 판단이 심화됨 → 생산지원 우선 판단 원칙

## [7] 단권화 암기 키워드 & 체크리스트

`✏ 고득점 키워드 (7개)`
1. **"캐리어 미디엄 룰"** (담체 자체 가격만 과세)
2. **"오락용 콘텐츠 제외"** (음악·영화·게임 → 전체 과세)
3. **"재현하는 권리"** (WTO협정 주해 — 가산 불가)
4. **"체화된 저작권"** (수입물품에 담긴 콘텐츠 → 가산 가능)
5. **"생산지원비용 우선 판단"** (마스터테이프·Stamper → 시행령 제18조)
6. **"경제적 내구연한 배분"** (포토마스크 등 도구 비용의 개당 배분)
7. **"기계와 불가분한 S/W"** (내장 S/W → 기계 과세가격에 포함)

`★ 절 핵심 체크리스트`
- ★★★ 1. 캐리어 미디엄 룰의 적용 범위와 **오락용 콘텐츠 제외 이유**는?
- ★★★ 2. 마스터테이프·마스터CD는 권리사용료인가, 생산지원비용인가? 판단 기준은?
- ★★  3. WTO협정상 '재현하는 권리'가 가산 대상이 아닌 이유와, 체화된 저작권 사용료와의 구분은?
- ★★  4. 기계에 내장된 S/W와 수입 후 사용되는 S/W의 과세가격 처리 차이는?
- ★   5. 포토마스크 비용의 경제적 내구연한 배분 계산 방법은?
