## [1] 핵심 요약 (3줄 이내)

- 제1방법(거래가격) 적용이 불가한 경우, **동종·동질물품** 또는 **유사물품**의 거래가격을 기초로 과세가격을 결정한다(관세법 제31조·제32조).
- 두 방법은 **비교 대상 물품의 유사도** 차이로만 구분되며, 적용 절차·조정 방법은 동일한 구조를 따른다.
- 출제 시 주로 **거래단계·수량차이 조정 계산** 및 **동종동질 vs 유사물품의 개념 구별**이 핵심 논점이다.

## [2] 구조화된 이론 분석

### 1. 적용 전제 및 순위

1. **적용 전제**
   - 관세법 제30조(제1방법)를 적용할 수 없는 경우에 한하여 순차 적용
   - 제2방법(동종·동질) → 적용 불가 시 → 제3방법(유사물품) 순 적용 (관세법 제2조 평가방법 순위)
   - 납세의무자 신청 시 제4방법과 제5방법의 순서를 바꾸어 적용 가능하나, 제2·3방법은 순서 변경 불가

2. **비교 대상 거래의 수입시기 요건** (관세법 제31조 제1항, 제32조 제1항)
   - 평가대상 물품의 **수출과 동시에 또는 거의 동시에** 우리나라에 수출하기 위하여 판매된 동종·동질(또는 유사)물품의 거래가격을 사용
   - '거의 동시'의 해석: **선적일 전후 60일 이내**를 원칙으로 하되, 없는 경우 기간을 확대하여 탐색 (시행령 제25조 제3항) ※ 농림축산물 등 계절성 물품은 예외적으로 30일 적용

### 2. 핵심 개념: 동종·동질물품 vs 유사물품

| 구분 | 동종·동질물품 | 유사물품 |
|------|-------------|---------|
| **법적 근거** | 관세법 제31조, 시행령 제24조 | 관세법 제32조, 시행령 제25조 |
| **물리적 특성** | 외양을 포함한 모든 면에서 동일 | 기능·품질·상업적 교환가능성 유사 |
| **구성 재료** | 동일한 재료로 제조 | 유사한 재료로 제조 |
| **기능·특성** | 동일한 기능 수행 | 동일한 기능 수행 |
| **상업적 교환가능성** | 완전한 교환 가능 | 상업적으로 교환 가능 |
| **사소한 차이** | 사소한 외관 차이는 동종동질로 인정 | 적용 불가 시 → 유사물품으로 전환 |
| **WTO협정** | 제2조, 부속서I 주해 §2 | 제3조, 부속서I 주해 §3 |
| **적용 우선순위** | 먼저 적용 | 동종동질 없을 때 적용 |

> **[WTO협정 부속서 I 주해]**: 동종·동질물품을 결정할 때 사소한 외관의 차이는 동종·동질성 결정에 영향을 미치지 않는 것으로 간주한다. 동종·동질물품에는 평가대상물품을 생산한 자와 동일인이 생산한 물품이 우선 포함되나, 그러한 물품이 없는 경우 다른 생산자가 생산한 물품도 포함된다.

### 3. 거래가격 적용 요건 (관세법 제31조 제1항 각호)

적용 가능한 동종·동질물품(유사물품)의 거래가격이 되려면 다음 요건을 **모두** 충족하고, **세관에 의해 과세가격으로 인정된 사실이 있는** 거래가격이어야 한다.

1. **생산국 동일**: 평가대상물품과 같은 나라에서 생산된 것
2. **수출 우리나라**: 우리나라에 수출하기 위하여 판매된 것
3. **특수관계 없음 or 가격 무영향**: 구매자와 판매자 사이에 특수관계가 없거나, 특수관계가 있더라도 가격에 영향을 미치지 않은 것
4. **수입시기 근접**: 평가대상물품과 같은 시기 또는 거의 같은 시기에 수출된 것

> ⚠ 동일 생산자의 물품이 우선, 없으면 다른 생산자 물품 사용 가능 (시행령 제24조 제1항)

### 4. 거래단계·수량차이 조정 (핵심 계산 논점)

#### (1) 조정의 필요성

동종·동질(또는 유사)물품의 거래가격이 **평가대상물품과 거래단계 또는 수량이 다른 경우**, 그 차이를 반영하여 조정한다 (관세법 제31조 제2항, 제32조 제2항).

#### (2) 조정 가능 요건 — 3단계 충족 필요

```
[STEP 1] 비교 가능한 거래가격 존재 확인
         → 동종·동질(유사)물품의 거래가격이 하나 이상 존재
              ↓
[STEP 2] 거래단계·수량 차이 여부 확인
         → 차이 없음 → 그대로 적용
         → 차이 있음 → STEP 3으로
              ↓
[STEP 3] 조정의 근거 자료 확인
         → 조정 근거 있음(공표된 가격표 등) → 조정 후 적용
         → 조정 근거 없음                  → 해당 거래가격 사용 불가
                                             → 다른 거래가격 탐색
```

#### (3) 조정 방법 — 2가지 유형

| 구분 | 거래단계 차이 조정 | 수량 차이 조정 |
|------|-----------------|-------------|
| **조정 방향** | 비교물품의 거래단계를 평가대상 단계로 전환 | 비교물품 수량을 평가대상 수량 기준으로 전환 |
| **조정 근거** | 판매자가 각 단계별 가격을 공표하고 있을 것 | 판매자가 수량별 가격표를 공표하고 있을 것 |
| **핵심 요건** | 가격표가 **진실**되고 일관되게 적용될 것 | 가격표가 **진실**되고 일관되게 적용될 것 |
| **기출 사례** | 2020년 사례1 (소매 → 도매 조정) | 2020년 사례2 (2,300개 → 1,700개 조정) |

---

### 5. 복수의 비교가격이 존재하는 경우

- 여러 개의 동종·동질(유사)물품 거래가격이 발견된 경우: **최저가격**을 과세가격으로 결정 (관세법 제31조 제3항, 제32조 제3항)
- 단, 조정 후에도 복수의 가격이 남는 경우에도 동일하게 최저가격 적용

---

### 6. 사용 불가한 거래가격

다음에 해당하는 가격은 과세가격 결정에 사용할 수 없다 (시행령 제29조 제2항; 제6방법 적용 시 사용 금지 가격으로서 제2·3방법에도 준용):

1. 우리나라에서 생산된 물품의 국내 판매가격
2. 선택 가능한 가격 중 반드시 높은 가격을 과세가격으로 하여야 한다는 기준에 따라 결정하는 가격
3. 관세법 제35조(제6방법)에 따른 합리적 기준에 의해 설정된 가격
4. 수출국 내 국내시장 판매가격
5. 제3국에서 수출하기 위한 판매가격
6. 최저과세가격
7. 자의적 또는 가공의 과세가격

## [4] 계산 공식 및 실수 포인트

### ■ 거래단계 차이 조정 공식

```
┌────────────────────────────────────────────────────────────────┐
│  조정된 과세가격                                                │
│  = 비교물품 거래가격 × (1 − 단계 할인율)  [소매→도매 조정 예시] │
│  또는                                                          │
│  = 비교물품 거래가격 ÷ (1 + 단계 할증율)  [도매→소매 역방향]   │
└────────────────────────────────────────────────────────────────┘
```

- **할인율 근거**: 판매자의 공표된 가격표(Price List)에 명시된 단계별 할인율
- **조정 방향**: 비교물품의 단계 → 평가대상 물품의 단계로 변환

### ■ 수량 차이 조정 공식

```
┌────────────────────────────────────────────────────────────────┐
│  조정된 과세가격                                                │
│  = 비교물품 수량기준 단가 (평가대상 수량 구간에 해당하는 단가)  │
└────────────────────────────────────────────────────────────────┘
```

- 비교물품 판매자의 수량별 가격표를 기준으로 **평가대상물품의 수량 구간**에 해당하는 단가를 선택

### ■ 계산 절차 (기출 사례 유형 기준)

**[유형 A] 거래단계 차이 조정 (2020년 사례1)**

- STEP 1. 비교물품의 거래단계 확인 (소매/도매)
- STEP 2. 평가대상물품의 거래단계 확인
- STEP 3. 판매자의 단계별 공표 가격표 존재 여부·진실성 확인
- STEP 4. 가격표상 할인율 적용하여 조정
- STEP 5. 조정 후 과세가격 결정 (수량이 같으면 추가 조정 불요)

**[유형 B] 수량 차이 조정 (2020년 사례2)**

- STEP 1. 비교물품의 수량 확인
- STEP 2. 평가대상물품의 수량 확인
- STEP 3. 판매자의 수량별 공표 가격표 존재 여부·진실성 확인
- STEP 4. 평가대상 수량에 해당하는 단가로 조정
- STEP 5. 거래단계 동일 여부 추가 확인 → 동일이면 바로 적용

### ■ ⚠ 실수하기 쉬운 포인트 체크리스트

- □ 동종·동질물품과 유사물품을 혼용하지 말 것 — 제2방법과 제3방법은 **별개의 독립된 방법**
- □ 거래단계 조정은 **가격표가 공표되어 있고 진실할 것**이 전제 — 추정이나 계산에 의한 단계 차이 환산 불가
- □ 수량 조정도 동일: **공표된 수량별 가격표**가 있어야 조정 가능 — 없으면 해당 거래가격 사용 자체 불가
- □ 거래단계와 수량 모두 차이가 있는 경우: **두 조정을 모두 적용** 가능 (단, 각각의 근거 자료 필요)
- □ 복수 거래가격 존재 시 **최고가격 아닌 최저가격** 적용 — 혼동 주의
- □ 동일 생산자 물품 우선 탐색 → 없으면 다른 생산자 물품으로 확대 — 순서 건너뛰지 말 것
- □ 비교 대상 물품이 **수출국 내 국내판매** 가격이거나 **제3국 수출가격**이면 사용 불가
- □ 수입시기: '거의 동시' = **선적일 전후 60일 이내** 원칙 (시행령 제25조 제3항) — 농림축산물 등 계절성 물품은 30일로 단축; 원칙 기간 초과 시 제6방법(신축적 적용) 검토 필요

## [5] 기출 기반 사례 응용

### 기출 문제 직접 추출

#### 【2020년 제37회 문제3】 거래단계·수량차이 조정

**사례1 (거래단계 차이 조정)**

| 항목 | 평가대상물품 | 동종·동질물품(비교) |
|------|-----------|------------------|
| 판매자 | A | B |
| 수량 | 2,800 | 2,800 |
| 단가 | 150원(CIF) | 250원(CIF) × (1-15%) |
| 수입자 | X | Y |
| 거래단계 | 도매 | 소매 |

- B의 가격표: 도매 20% 할인, 소매 15% 할인 (공표된 가격표 존재)
- 수량은 동일(2,800개) → **거래단계 조정만** 필요

**풀이 및 결론**

```
조정 후 과세가격
= 250원 × (1 − 15%)     ← 소매 기준 단가
= 250원 × 0.85
= 212.5원               ← 소매 거래가격

그런데 평가대상은 도매단계이므로
도매 기준으로 재조정:
= 250원 × (1 − 20%) = 200원   ← 도매 기준 단가

∴ 조정된 과세가격 = 200원(CIF)
```

> 논거: B의 가격표는 공표되어 있고 진실하며, Y에 대한 판매에도 일관 적용됨이 확인됨 → 거래단계 조정 유효.

---

**사례2 (수량 차이 조정)**

| 항목 | 평가대상물품 | 유사물품(비교) |
|------|-----------|------------|
| 판매자 | C | D |
| 수량 | 1,700 | 2,300 |
| 단가 | 400원 | 475원(CIF) |
| 수입자 | Z | W |
| 거래단계 | 도매 | 도매 |

- D의 수량별 가격표: 2,000개 미만 → 500원, 2,000개 이상 → 475원 (공표·진실 확인)
- 거래단계는 동일(도매) → **수량 조정만** 필요

**풀이 및 결론**

```
평가대상 수량: 1,700개 → 2,000개 미만 구간
∴ 해당 단가 = 500원(CIF)

∴ 조정된 과세가격 = 500원(CIF)
```

> 논거: 평가대상물품의 수량(1,700개)은 2,000개 미만 구간에 해당하므로, D의 수량별 가격표에서 해당 구간 단가인 500원을 적용한다.

#### 【2024년 제41회 문제3】 제2방법 적용 가능성 검토 (양파 수입 사례)

**쟁점**: X(중국)가 7월에 한국 내 다른 수입자에게 USD 950(FOB)으로 수출한 거래를 4월 수입분(I의 잠정신고가격)의 과세가격 결정에 제2·3방법 기준으로 사용할 수 있는가?

**판단**

| 검토 항목 | 판단 | 근거 |
|---------|------|------|
| 생산국 동일 (중국) | ✔ 충족 | 동일 판매자 X |
| 우리나라 수출용 | ✔ 충족 | 한국 수입자에 판매 |
| 수입시기 근접 | ✔ or ✘ 검토 필요 | 4월 vs 7월 → **약 90일 차이** → 원칙(±60일) 초과 |
| 거래단계·수량 조정 | 추가 조정 필요 | FOB → CIF 환산 필요 |
| 품질 동일성 | 쟁점 | 성분분석 결과 99% 일치 → 동종동질 또는 유사물품 해당 가능 |

**결론**: 수입시기 요건(±60일)을 초과(약 90일 차이)하므로 엄격한 제2방법 적용은 불가. 다만, 시행규칙 제7조 제1항 제2호에 따라 **제6방법(제2방법의 신축적 적용)**으로서 기간 확대 적용 가능 여부 검토 필요. 운임(USD 200) 가산하여 CIF로 조정 후 사용.

```
조정된 과세가격 = USD 950(FOB) + USD 200(운임) = USD 1,150(CIF)
```

`📌 출제 경향`
- 단독 대문제로 출제되기보다 **제1방법 종합문제 내 소문항**으로 등장하는 경향
- 최근(2022~2025) 특수관계 비교가격의 일환으로 제2·3방법 거래가격이 간접 등장
- 거래단계·수량차이 조정은 **공표된 가격표의 진실성 확인**이 핵심 논거
