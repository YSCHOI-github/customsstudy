## [1] 핵심 요약 (3줄 이내)

- **수입항까지의 운임·보험료 및 운송 관련 비용은 과세가격의 가산요소**이며, **수입항 도착을 기점**으로 이전 비용은 포함, 이후 비용은 제외된다 (관세법 제30조 제1항 제6호).
- 항공으로 운송방법이 변경된 경우, **수입자 귀책 외 사유(제작지연·불가항력 등)이면 해상운임 상당액**으로 제한하고(수입자 귀책이면 실제 항공운임 전액 가산), 특수 비용(공선회조료·체선료·조출료·창고료)은 각각 수입물품 운송과의 **직접 관련성** 및 **발생 시점**으로 판단한다.
- A티어 핵심 패턴: ① 수입항 전후 분기점 판단 ② 특수 운임·보험료 유형별 처리 ③ 포괄예정보험 안분 계산이 반드시 출제된다.

## [2] 구조화된 이론 분석

### 1. 가산의 근거 및 범위

1. **가산 근거** (관세법 제30조 제1항 제6호)
   - 수입물품을 수입항까지 운송하는 데 필요한 **운임·보험료 및 그 밖에 운송과 관련되는 비용**
   - 실제지급가격에 포함되지 않은 경우에 한하여 가산

2. **수입항(port of importation)의 정의** (관세평가 운영에 관한 고시 제24조 제1항)
   - 수입물품이 **우리나라에 반입되는 항구·공항·역 기타의 장소**
   - CIF 조건: 계약서에 기재된 도착항이 수입항
   - FOB/EXW 조건: 실제 우리나라에 물품이 반입되는 항구·공항
   - 컨테이너 Door-to-Door 운송: 국내 CFS(컨테이너 화물 집결지) 또는 CY까지가 아닌 **수입항(도착 항구)** 까지만 포함

3. **과세가격 확정 시기(분기점)** (관세법 시행령 제20조 제5항)

   ```
   ──────────────────────────────────────────────────────
   [수출국]       [선적항]      [수입항]      [국내 목적지]
      │              │             │               │
      ←──────────────────────────→               │
        과세가격에 포함 (본선하역준비 완료 전까지)    │
                                  │←─────────────→
                                  과세가격에서 제외
                                  (→ 제4방법 공제요소)
   ──────────────────────────────────────────────────────
   ```

   - **본선하역준비 완료 이전** 발생 비용 → **과세가격에 포함 (가산)**
   - **본선하역준비 완료 이후** 발생 비용 → **과세가격에서 제외 (공제요소 해당)**

### 2. 운임의 결정 방법 (관세법 시행령 제20조·시행규칙 제4조의3)

1. **원칙**: 운송서류(B/L, AWB 등)에 기재된 운임 금액
2. **운임을 알 수 없는 경우**: 세관장이 확인한 통상의 운임
3. **운송방법 변경 — 항공운임 제한 규정** (관세법 시행규칙 제4조의3 제2항)

   | 변경 사유 | 과세가격 적용 운임 |
   |----------|----------------|
   | **구매자(수입자)의 편의·필요·귀책**으로 선박 → 항공 변경 | **실제 항공운임 전액** 가산 |
   | 제작지연 등 수입자 귀책 외 사유, 수출자 귀책, 천재지변 등 불가항력 | **해상운임 상당액**으로 제한 |

   > ⚠ 2023년 제40회 기출: FOB 조건 수입에서 수입자 귀책으로 항공 변경 → **실제 항공운임 전액** 과세가격에 산입

4. **Door-to-Door(문전배달) 컨테이너 운임 항목별 처리**

   | 운임 구성 항목 | 수입항 기준 | 과세가격 포함 여부 |
   |-------------|----------|----------------|
   | 수출국 내륙운송비 | 수입항 이전 | **포함** |
   | 해상운임 | 수입항 이전 | **포함** |
   | 컨테이너 임차료 | 수입항 이전 운송과 관련 | **포함** |
   | 수입항에서 하역비 | 수입항 도착 이후 발생 | **제외** |
   | 수입항 도착 이후 운송료 | 수입항 이후 | **제외** |

   > 📌 2024년 제41회 기출: 위 5개 항목 분류가 그대로 출제됨 (포함 합계 = USD 300)

### 3. 보험료의 결정 방법 (관세법 시행령 제20조·시행규칙 제4조의3)

1. **원칙**: 보험서류에 기재된 보험료
2. **보험료를 알 수 없는 경우**: 관세청장이 정한 요율로 계산한 금액
3. **가산 범위**: **수입항까지 운송을 담보**하는 보험료

   | 보험 유형 | 담보 범위 | 과세가격 포함 여부 |
   |---------|---------|----------------|
   | CIF 조건 보험료 (ICC(C) 등) | 수입항까지 | **포함** (이미 CIF 가격에 내재) |
   | 구매자가 별도 부보한 수입항까지 보험 | 수입항까지 | **포함** |
   | **협회전쟁약관(Institute War Clauses)** 등 추가 부보 | 수입항까지 담보 | **포함** |
   | 수입항 이후 국내 구간 보험료 | 수입항 이후 | **제외** |

   > 📌 2024년 제41회 기출: 구매자가 중동 전쟁위험을 별도 부보한 협회전쟁약관 보험료
   > → 수입항(부산항)까지의 운송을 담보 → **과세가격에 포함**

4. **포괄예정보험(Blanket / Open Cover Policy) 안분 계산**

   - **의의**: 장기간·다수 물품 수입을 위해 연간 일괄 부보하는 보험 방식
   - **가산 방법** (관세평가 운영에 관한 고시 제26조 제2항):
     1. **안분 방식**: 해당 수입 건에 귀속되는 보험료를 아래 비율로 산출 (원칙)
     2. **일괄 가산 방식**: 포괄예정보험료 전액을 **최초 수입물품의 수입신고 시에 잠정신고하여 일괄 가산**하는 방식도 허용

   ```
   ┌────────────────────────────────────────────────────────┐
   │  해당 건 보험료 = 총 보험료 ×                            │
   │               해당 건 CIF 가격 (또는 수량)               │
   │               ─────────────────────────                │
   │               전체 예상 CIF 가격 (또는 수량)             │
   └────────────────────────────────────────────────────────┘
   ```

   > 📌 2024년 제41회 기출: 포괄예정보험 USD 1,000 (연간 100톤 수입 예정),
   > 1차 수입 10톤 → 보험료 = USD 1,000 × (10/100) = **USD 100**

### 4. 특수 비용 유형별 처리 — 핵심 논쟁 항목

#### (1) 공선회조료(空船回助料) — 2023년 제40회 기출

| 구분 | 내용 |
|------|------|
| 의의 | 왕복항해 용선계약에서 **귀항(Returning Voyage) 시 화물 없이 운항**하여 손실이 발생할 경우, 수입자가 선박회사에 지급하는 손실 보전료 |
| 법적 성격 | 용선계약에 의하여 실제로 지급되는 금액으로, 해당 용선계약 범위 내 비용 |
| 과세가격 포함 여부 | **포함(가산)** |
| 근거 | 용선계약에 따라 지급되는 모든 금액에 해당 (관세평가 운영에 관한 고시 제24조 제3항 제2호) |

#### (2) 체선료(Demurrage) — 2017·2018·2020년 기출

| 발생 시점 | 의의 | 과세가격 포함 여부 |
|---------|------|----------------|
| **선적항(수입항 이전)** | 수출국 선적항에서 하역 지연 → 약정 정박일 초과 | **포함** |
| **수입항** | 수입항에서 하역 지연 → 약정 정박일 초과 | **제외** |

> ⚠ **판단 기준**: 수입항 도착 이전 발생 여부
> - 수입항에서 발생한 체선료는 수입항 도착 이후 비용 → 제외
> - 2017년 제34회, 2020년 제37회 기출: 수입항 체선료 → **제외**
> - 2020년 제37회 기출: 선적항 체선료 500,000원 → **포함**

#### (3) 조출료(Dispatch Money) — 2018년 제35회 기출

| 구분 | 내용 |
|------|------|
| 의의 | 약정 기간보다 **조기에 선적 완료**하여 선주로부터 받는 환급금(운임 성격의 감액) |
| 과세가격 처리 | 운임의 감액 → **과세가격에서 공제** (운임 자체에서 차감) |
| 근거 | 지급한 운임에서 조출료를 공제한 금액이 실제 운임 |

#### (4) 창고료(Storage Charge) — 2014·2018·2019년 기출

| 발생 원인 | 과세가격 포함 여부 | 근거 |
|---------|----------------|------|
| **운송수단 지연** 등 불가피한 사유로 수출국 부두·창고에 보관 | **포함** | 운송과 직접 관련된 부대비용 |
| **구매자의 판매일정 조정** 목적으로 보관 | **제외** | 운송 관련성 결여, 구매자 편의 비용 |
| 구매자 수입 일정 조절 목적으로 수출국 세관창고 보관 | **제외** | 구매자 사업상 편의 목적 |

> 📌 2018년 제35회 기출 사례 비교:
> - 사례1 (구매자 판매일정 → 세관창고) → **제외**
> - 사례2 (선박 지연으로 부두창고) → **포함**

## [4] 계산 공식 및 실수 포인트

### ■ 과세가격 기본 산식 (CIF 기준 거래)

```
┌──────────────────────────────────────────────────────────────┐
│  과세가격(CIF) = 실제지급가격(FOB 등)                           │
│               + 수입항까지 운임                                 │
│               + 수입항까지 보험료                               │
│               + 기타 운송 관련 비용 (수입항 이전)               │
│               − 조출료 (운임에서 차감)                          │
│               − 수입항 이후 비용 (CIF가격에 포함된 경우 공제)    │
└──────────────────────────────────────────────────────────────┘
```

### ■ 포괄예정보험 안분 계산식

```
┌────────────────────────────────────────────────────────┐
│  해당 건 보험료 = 연간 총 보험료 ×                        │
│               해당 건 수입량(또는 CIF가격)               │
│               ─────────────────────────                │
│               연간 예상 총 수입량(또는 CIF가격)          │
└────────────────────────────────────────────────────────┘
```

### ■ 2023년 제40회 기출 계산 예시

- FOB $1,000,000 (밀 10,000톤, $100/톤)
- 해상운임 및 보험료 $150,000 (몬트리올→부산, **수입항 이전**)
- 대구 보관창고까지 운송료 $50,000 → **수입항(부산항) 이후 제외**
- 공선회조료 → **용선계약상 지급비용, 포함(가산)** (고시 제24조제3항제2호)
- 환율 = 1USD = **1,250원** (관세청장 고시 환율 적용; 수입신고일 고시환율 아님)

```
과세가격 = ($1,000,000 + $150,000) × 1,250원
         = $1,150,000 × 1,250
         = 1,437,500,000원
```

> ⚠ **환율 주의**: 수입신고일 고시환율(1,200원)이 아닌,
> 수입신고일이 속하는 주의 **전주 월~금 평균 고시환율(1,250원)**을 적용
> → 관세법 시행규칙 제1조의2 (관세청장이 정한 환율 우선)

### ■ ⚠ 실수하기 쉬운 포인트 체크리스트

- □ **수입항 기준 혼동**: "수입항에서 하역비"는 언뜻 수입항과 관련된 것처럼 보이지만 **수입항 도착 이후** 발생이므로 제외
- □ **공선회조료 주의**: 왕복 용선계약에서 귀항 손실 보전료 → 용선계약에 따라 지급되는 모든 금액에 포함 → **포함(가산)** (고시 제24조제3항제2호)
- □ **체선료 발생 장소 구분**: 선적항 체선료 ≠ 수입항 체선료 → 선적항 발생은 포함, 수입항 발생은 제외
- □ **항공운임 제한 적용 주체**: 수입자 귀책 외 사유(제작지연·수출자귀책·불가항력 등) → 해상운임 상당액 적용 / 수입자 귀책·편의 → 실제 항공운임 전액 가산
- □ **포괄예정보험 가산 방식**: 안분 방식이 원칙이나, 최초 수입건에 전액 일괄 잠정신고도 허용 (고시 제26조제2항); 단순히 연간 총액을 여러 건에 중복 가산하면 오답
- □ **전쟁보험료**: 추가 특약이라도 **수입항까지 담보하면 포함** (수입항 이후 담보분은 제외)
- □ **환율 적용**: 수입신고일 당일 고시환율 아닌 **관세청장 고시환율(전주 월~금 평균)** 적용 원칙
