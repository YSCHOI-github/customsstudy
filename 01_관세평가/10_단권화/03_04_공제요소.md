## [1] 핵심 요약 (3줄 이내)

- **공제요소**란 구매자가 실제로 지급하였거나 지급할 총금액 중, **금액이 명백히 구분될 경우** 과세가격에서 제외할 수 있는 4가지 비용이다 (관세법 §30② 단서).
- 공제 인정의 **공통 전제**는 "금액이 명백히 구분(clearly distinguished)"되는 것이며, 연불이자는 이에 더하여 **서면 금융약정·이자율 입증**이라는 추가 2요건을 별도로 충족해야 한다.
- B티어 핵심 패턴: ① **공제요소 4가지 요건 서술** ② **연불이자 추가요건** ③ **수입항 정의 및 운임 분기점** — 3가지 논점이 독립 소문항(4~8점)으로 반복 출제.

### 2. "명백히 구분(clearly distinguished)"의 판단 기준

> **가장 먼저 확인해야 할 공통 전제** — 이를 충족하지 못하면 나머지 요건을 검토할 필요 없이 공제 불가

| 판단 방법 | 구체적 기준 | 효과 |
|----------|-----------|------|
| **인보이스 분리 기재** | 공제 대상 비용이 물품 대금과 별도 항목으로 기재 | 충족 — 공제 가능 |
| **계약서 명시** | 계약서에 비용 항목이 물품 가격과 구분하여 명기 | 충족 — 공제 가능 |
| **별도 청구서 발행** | 물품 인보이스와 별도의 청구서(별도 계약)로 분리 | 충족 — 공제 가능 |
| **가격에 혼합 포함** | 인보이스 총액에 일체로 포함, 분리 불가 | 미충족 — 공제 불가 |
| **구두 합의만 존재** | 서면 자료 없이 구두로만 구분 주장 | 미충족 — 공제 불가 |

> [WTO CVA 제1조 주해]:
> 실제로 지급했거나 지불할 가격으로부터 **구분될 수 있는** 부가금액 및 비용은
> 과세가격에 포함되지 아니한다.

> ⚠ "명백히 구분" = 단순히 비용이 별개로 존재한다는 주장만으로는 부족
> → **객관적 서면 증거**로 세관장이 인정할 수 있어야 함

### 3. 공제요소 ① — 수입 후 건설·설치·조립·유지·기술지원 비용

#### 3-1. 공제요소의 범위

- **"수입 후"** 의 의미: **수입항에 도착한 이후** 수입국 내에서 발생하는 비용
  - ≠ "수입신고 수리 이후" — 수입항 도착 이후라면 수입신고 전이라도 해당
- **5가지 비용 유형**:

| 비용 유형 | 설명 | 공제 가능 여부 |
|----------|------|--------------|
| 건설비용 | 물품이 수입 후 국내에서 구축·조성되는 공사 비용 | ✅ 명백 구분 시 |
| 설치비용 | 기계·설비를 수입 후 국내에서 부착·거치하는 비용 | ✅ 명백 구분 시 |
| 조립비용 | 수입 부품을 국내에서 결합·완성하는 비용 | ✅ 명백 구분 시 |
| 정비비용 | 수입 후 국내에서 발생하는 정비(repair/maintenance) 비용 | ✅ 명백 구분 시 |
| 유지비용 | 수입 후 국내에서 발생하는 유지보수(maintenance) 비용 | ✅ 명백 구분 시 |
| 기술지원비용 | 수입 후 국내에서 제공되는 기술적 조언·훈련 비용 | ✅ 명백 구분 시 |

#### 3-2. 수입 전 vs. 수입 후 비용 구분

| 구분 | 발생 장소·시점 | 과세 처리 |
|------|-------------|----------|
| 수출국 내 기술지원·조립비 (수입 전) | 수출국, 선적 전 | **포함** — 물품 대가 구성 |
| 수입항 도착 이후 기술지원비 | 수입국, 수입 후 | **불포함** — 명백 구분 시 |
| 수입 전 검사비 (판매자 청구) | 수출국 | **포함** — 물품 대가 구성 |
| 수입 후 기계 시운전비 (국내) | 수입국, 수입 후 | **불포함** — 명백 구분 시 |

> 📌 **WTO CVA와의 관계**:
> WTO CVA **제1조 주해(Interpretative Note to Article 1)**에도 수입 후
> 건설·설치·조립·유지·기술지원 비용을 과세가격에서 제외하는 항목으로 **명시적으로 규정**
> → 관세법 §30②①호는 WTO 협정을 충실히 반영한 규정
> ※ 단, 관세법에는 "정비" 비용이 명시적으로 포함되어 있어 협정보다 열거가 상세함

### 4. 공제요소 ② — 수입항 도착 이후 운임·보험료 및 관련 비용

#### 4-1. "수입항(Port of Importation)"의 정의 ★★★ (2025년 직접 출제)

> **관세평가 운영에 관한 고시 제24조제1항**:
> "수입항"이란 해당 수입물품이 외국에서 우리나라에 도착한 운송수단으로부터
> **양륙(일시 양륙은 제외한다)이 이루어지는 항구 또는 공항**을 말한다.

- 핵심: **물리적 양륙(하역)이 이루어지는 장소** 기준
- 세관 소재지가 아닌, 실제로 화물을 내리는(양륙하는) 항구·공항이 기준

#### 4-2. 수입항 기준 운임 분기점 (과세가격 확정 시기)

```
[수출항 → 수입항까지] ←─────────────→ [수입항 이후]
         ↓                                    ↓
    가산요소 (§30①⑥)               공제요소 (§30②②호)
    과세가격에 포함                  명백 구분 시 제외

────────────────────────────────────────────────────
수출항 선적 → 해상운송 → [수입항 도착] → 국내 내륙 운송
                                ↑
                         이 시점이 분기점
```

#### 4-3. 수입항 기준 세부 비용 처리

| 비용 항목 | 분기 기준 | 과세 처리 |
|----------|---------|----------|
| 수출항~수입항 간 해상운임 | 수입항까지 | **포함** (가산) |
| 수출항~수입항 간 항공운임 | 수입항까지 | **포함** (가산, 단 항공 특례 있음) |
| 수출항~수입항 간 적하보험료 | 수입항까지 | **포함** (가산) |
| 수입항 도착 후 국내 내륙 운임 | 수입항 이후 | **불포함** (명백 구분 시) |
| 수입항 도착 후 창고료 | 수입항 이후 | **불포함** (명백 구분 시) |
| 수입항에서의 **양하비(하역비)** | 수입항 도착 시 | **불포함** (금액 구분 시 공제 가능) |
| 수입항 도착 후 통관 비용 | 수입항 이후 | **불포함** (명백 구분 시) |

> ⚠ **양하비(하역비) 처리 핵심** (관세평가 고시 §24④②호):
> 수입항에서의 하역비가 운임에 포함되어 있더라도, **그 금액이 구분 표시된 경우에는
> 과세가격에 포함하지 아니한다** (공제 가능)
> → 금액이 혼합되어 구분 불가한 경우에만 운임에 포함되어 과세가격에 산입됨

#### 4-4. CIF·FOB 조건별 처리

| 거래 조건 | 구매자 부담 운임 범위 | 과세가격 처리 |
|----------|------------------|------------|
| **CIF** | 판매자가 수입항까지 운임·보험료 부담 | 인보이스 가격에 포함 → 별도 가산 불필요 |
| **FOB** | 구매자가 수입항까지 운임·보험료 별도 부담 | 구매자 부담분을 **가산** (§30①⑥) |
| **EXW** | 구매자가 모든 운임·보험료 부담 | 수입항까지 발생분 전부 **가산** |
| **DDP** | 판매자가 수입항 이후 비용까지 부담 | 수입 후 비용 명백 구분 시 **공제** 가능 |

### 5. 공제요소 ③ — 우리나라에서 부과되는 관세·세금

- **공제 대상 세목**: 관세, 부가가치세, 개별소비세, 교통·에너지·환경세, 교육세 등
- **공제 취지**: 과세가격에 관세를 포함시키면 관세에 관세를 부과하는 **순환 논리** 발생 — 방지
- **판단 기준**: 수입국(우리나라)에서 부과되는 세금에 한정
  - 수출국에서 부과된 세금 → 공제 대상 아님 (물품 대가 구성)

> [WTO CVA 제1조 주해]:
> "과세가격에는 수입국이 수입물품에 부과하는 세금은 포함하지 아니한다."

### 6. 공제요소 ④ — 연불이자 (Finance Charges) ★★★

#### 6-1. 연불이자의 개념

> **연불조건부 구매(deferred payment)**: 물품 대금을 수입 후 일정 기간에 걸쳐 분할 지급하는 거래에서 발생하는 금융비용(이자)

- 연불이자는 **물품 자체의 가치**가 아닌 **금융 서비스 대가**이므로 과세 불포함이 원칙
- 단, 공통 전제(명백히 구분) + **추가 2요건**을 모두 충족해야 함

#### 6-2. 연불이자 공제의 3요건

```
┌───────────────────────────────────────────────────────────────────┐
│  연불이자 공제 요건 (관세법 §30②④호 + 시행령 §20조의2③)          │
│                                                                   │
│  [공통 전제] ① 금액이 물품 대금과 명백히 구분될 것               │
│             (§30② 단서 — 모든 공제요소의 공통 요건)              │
│                                                                   │
│  [추가 요건] ② 금융약정이 서면으로 체결되어 있을 것              │
│                                                                   │
│             ③ 실제 적용 이자율이 금융이 제공된 국가에서           │
│                당시 금융거래에 통용되는 수준의 이자율을           │
│                초과하지 않음을 입증할 수 있을 것                  │
└───────────────────────────────────────────────────────────────────┘
```

#### 6-3. 각 요건의 상세

| 요건 | 내용 | 미충족 시 효과 |
|------|------|-------------|
| **① 명백히 구분** | 이자 금액이 물품 가격과 인보이스·계약서에서 별도 기재 | 공제 불가 — 전액 실제지급가격 포함 |
| **② 서면 금융약정** | 연불 조건·이자율·상환 일정 등이 서면(계약서·약정서)으로 체결 | 공제 불가 — 구두 합의만으로는 부족 |
| **③ 이자율 입증** | 실제 이자율 ≤ 금융이 제공된 국가의 당시 통용 금리 수준임을 입증 | 공제 불가 (통용 금리 **초과분**만 불가 아닌, **전액** 불가) |

> ⚠ 요건 ③의 핵심 포인트:
> - 이자율 기준: **금융이 제공된 국가(country where the finance is provided)**의 당시 통용 금리 (WTO CVA와 동일)
> - 입증 불가 시: 이자율 초과분만이 아닌 **연불이자 전액** 공제 불가
> - 통용 금리 수준 비교 시점: **금융이 제공된 당시** 기준

#### 6-4. WTO CVA 주해 vs. 국내 시행규칙 비교

| 구분 | WTO CVA 제1조 주해 (Finance charges) | 관세법 시행령 §20조의2③ |
|------|--------------------------------------|---------------------|
| **공통 전제** | 물품 가격과 명확히 구분 | 동일 (§30② 단서) |
| **서면 약정** | "서면으로 뒷받침될 것" | 동일 |
| **이자율 요건** | 해당 이자율이 금융 제공 당시 **금융 제공 국가**에서 통용되는 수준을 초과하지 않음을 입증 | **금융이 제공된 국가**의 당시 통용 금리 수준 (CVA와 동일) |
| **핵심 차이** | 없음 — 이자율 기준 일치 | — |
| **추가 요건** | 실제 판매가격이 신고 가격임을 입증 | 규정 없음 |

> ✅ **협정과 국내법의 관계**:
> WTO 주해와 국내 시행령 모두 **"금융이 제공된 국가"**의 이자율을 기준으로 함
> → 이자율 기준에서는 국내법과 WTO CVA 간 차이 없음

### 7. 공제요소 4가지 종합 비교

| 구분 | 공제요소 | 시점·장소 기준 | 추가 요건 | WTO CVA 명시 |
|------|---------|------------|---------|------------|
| **①** | 건설·설치·조립·정비·유지·기술지원 비용 | 수입 **후** 국내 | 없음 (명백 구분만) | ✅ (CVA 제1조 주해 명시) |

| **③** | 관세·세금 | 수입국 부과 | 없음 (명백 구분만) | ✅ |
| **④** | 연불이자 | — | 서면 약정 + 이자율 입증 | ✅ |

## [3] 판단 흐름도

### "특정 비용이 공제요소에 해당하는가?" 판단

```
[STEP 1] 공제요소 4가지 유형 중 하나에 해당하는가?
          ↓ 해당 없음 → 공제 불가 (실제지급가격 포함 또는 가산요소 검토)
          ↓ 해당 있음

[STEP 2] 금액이 명백히 구분되는가?
          (인보이스·계약서에 별도 기재 여부)
          ↓ 구분 불가 → 공제 불가
          ↓ 명백히 구분됨

[STEP 3] 연불이자인 경우 → 추가 2요건 검토
          ↓ ② 서면 금융약정 있는가?
               ↓ 없음 → 공제 불가
               ↓ 있음
          ↓ ③ 이자율이 금융 제공 국가의 당시 통용 금리 수준 이하임을 입증하는가?
               ↓ 입증 불가 → 전액 공제 불가
               ↓ 입증 가능

[STEP 4] 공제 인정 → 실제지급가격에서 제외
```

## [4] 계산 공식 및 실수 포인트

### ■ 공제 적용 과세가격 계산 구조

```
┌──────────────────────────────────────────────────────────────────────┐
│  과세가격 = 인보이스 금액 (물품 대금)                                  │
│            + 가산요소 (§30①)                                         │
│            − 공제요소 (§30②) [명백히 구분된 것만]                    │
│                                                                      │
│  ※ CIF 조건 인보이스인 경우: 운임·보험료가 이미 포함                  │
│     → 수입항 이후 운임은 명백 구분 시 별도 공제                       │
└──────────────────────────────────────────────────────────────────────┘
```

### ■ 계산 절차

- STEP 1. 인보이스 조건(CIF/FOB/EXW) 확인 → 운임·보험료 포함 여부 파악
- STEP 2. 인보이스 내 공제요소 해당 항목 식별 (별도 기재 여부)
- STEP 3. 연불이자 해당 시 추가 3요건 충족 여부 확인
- STEP 4. 공제 후 실제지급가격 확정 → 가산요소(§30①) 추가 → 최종 과세가격 산출

### ■ 기출 계산 예시 (2025년 제42회 문제2)

**거래 내용 (CIF 조건):**
```
본체 [principal machine]:            USD 300,000
연관 설비 [interconnected machines]: USD 120,000
조립·설치비 [assembling/installation]: USD  50,000  ← §30②①호 해당
시운전비 [trial cost]:               USD  30,000  ← §30②①호 해당
연불이자:  명백히 구분, 요건 충족     USD  별도   ← §30②④호 해당
                               합계  USD 500,000
```

**과세가격 산출:**
```
인보이스 총액:           USD 500,000
(−) 조립·설치비:        USD  50,000  [§30②①호, 수입 후 국내, 명백 구분]
(−) 시운전비:           USD  30,000  [§30②①호, 수입 후 국내, 명백 구분]
(−) 연불이자:           별도 금액    [§30②④호, 3요건 충족 가정]
────────────────────────────────────
과세가격:                USD 420,000  (연불이자 제외 전 기준)
```

### ■ ⚠ 실수하기 쉬운 포인트 체크리스트

- □ **"명백히 구분"은 모든 공제의 공통 전제** — 연불이자도 추가요건 이전에 이것부터 확인
- □ **연불이자 미충족 시 전액 불공제** — "초과분만" 불공제가 아님
- □ **수입항 기준 분기점** — 수입항에서의 하역비는 금액이 **구분 표시된 경우 공제 가능** (고시 §24④②호)
- □ **CIF 조건에서 수입항 이후 운임**: 인보이스에 별도 기재된 경우만 공제 — 혼합 총액이면 공제 불가
- □ **수입 전 기술지원비** (수출국 내): 공제 불가 — §30②①호는 수입 **후** 비용만 해당
- □ **WTO CVA와 ①번 공제요소**: CVA 제1조 주해에도 명시 — 국내법과 협정 모두 규정
- □ **연불이자 이자율 기준**: **금융이 제공된 국가**의 당시 통용 금리 (국내법 = WTO CVA, 기준 동일)
- □ **연불이자 관련 법령**: 시행령 §20조의2③ — 시행규칙 §3의2(확정가격 신고기간 연장)와 혼동 금지

## [7] 단권화 암기 키워드 & 체크리스트

### ✏ 고득점 키워드 (9개)

- `명백히 구분(clearly distinguished)` — 모든 공제의 공통 전제, 답안 첫 줄
- `수입 후 국내에서 발생` — ①번 공제요소의 핵심 시점 기준어
- `수입항` = `양륙이 이루어지는 항구 또는 공항` (관세평가 고시 §24①)
- `수입항 도착 이후` — ②번 공제요소의 시점 기준어 (수입항까지는 가산)
- `서면 금융약정` — 연불이자 추가요건 ②
- `금융이 제공된 국가의 통용 금리` — 연불이자 이자율 기준 (시행령 §20조의2③, WTO CVA와 동일)
- `전액 공제 불가` — 이자율 입증 실패 시 (초과분만이 아님)
- `WTO CVA에도 명시` — ①번 공제요소(건설·설치·조립·유지·기술지원), 협정과 국내법 일치
- `순환 논리 방지` — ③번 공제요소(관세 등) 취지
