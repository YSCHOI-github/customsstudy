## [1] 핵심 요약 (3줄 이내)

- 제1방법(거래가격 방법)은 **수입국으로의 수출을 위한 판매(수출판매)** 에서 실제 지급되었거나 지급될 가격을 기초로 과세가격을 결정하는 방법이다.
- 제1방법 적용의 전제는 ① **수출판매의 성립**, ② **거래가격 배제요건 해당 없음**, ③ **가격의 객관적 수량화 가능성**, ④ **특수관계가 있는 경우 가격에 영향 없음 입증**의 네 가지이다.
- **빈출 패턴**: 복합 거래 종합문제(50점)에서 "수출판매 성립 여부"·"일괄거래 평가"·"조건·사정 금액 계산 가능 여부"를 전제 쟁점으로 묻고, 이후 가산요소 계산으로 이어진다.

## [2] 구조화된 이론 분석

### 1. 제1방법(거래가격 방법) 개요

- **의의**: 수입물품의 과세가격은 **우리나라에 수출하기 위하여 판매되는 물품에 대하여 구매자가 실제로 지급하였거나 지급하여야 할 가격**에 법정 가산요소를 더한 거래가격으로 결정한다. (관세법 제30조 제1항)
- **적용 원칙**: 제1방법을 적용할 수 없는 경우에 한하여 순차적으로 제2~6방법을 적용한다. (관세법 제30조~제35조 순차 적용 원칙)
- **WTO 협정 근거**: [WTO협정 제1조]: 수입품의 관세가격은 거래가격, 즉 수입국으로의 수출을 위하여 판매되는 물품에 대하여 실제로 지급하였거나 지급할 가격으로 한다.

---

### 2. 수출판매(Sale for Export)의 개념 및 성립 요건

#### 2-1. 수출판매의 의의

- 수출판매란 **수입국으로의 수출을 위한 판매**를 의미하며, 단순한 물품의 이동이 아니라 '판매'라는 법률행위의 성립을 전제로 한다.
- [WTO협정 제1조 주해]: 관세가격은 거래가격에 기초한다. 거래가격은 수입국으로 수출되기 위해 판매된 물품에 지급하였거나 지급할 가격이다. "판매"는 일반 상법상 매매의 의미로, 당사자 사이에 물품과 대금의 교환을 내용으로 하는 계약을 의미한다.
- [관세평가 운영에 관한 고시 제15조 제1항]: '판매'란 **각각 자기 책임과 계산으로 해당 수입물품에 대한 대가를 지급하고 소유권 이전을 목적으로 하는 구매자와 판매자 간의 거래**를 말한다.

#### 2-2. 수출판매로 보지 않는 경우 (제1방법 적용 불가)

수출판매에 해당하지 않아 제1방법을 적용할 수 없는 거래 유형은 다음과 같다.

| 거래 유형 | 수출판매 해당 여부 | 근거 |
|----------|----------------|------|
| 무상 기증·증여 | ✗ (판매 아님) | 대금 없음 |
| 임차·리스 수입 | ✗ (소유권 이전 없음) | → 시행규칙 제7조의4 임차수입 과세가격 규정 적용 |
| 위탁판매 (수입 후 판매) | ✗ (수입 시점 판매 미성립) | 국내 도착 후 경매 등을 통해 판매가격이 결정되는 위탁판매물품 (시행령 제17조 제2호) |
| 별개의 독립된 법적 사업체가 아닌 지점 등과의 거래 (본·지점 내부이전) | ✗ (상업적 판매 아님) | 시행령 제17조 제4호 |
| 국내 보세구역 내 매매 (B/L 양수도) | △ (별도 판단 필요) | → 제3장 제1절 B/L양수도 참조 |
| 수출국에서의 판매 (수입국 판매 아닌 경우) | ✗ | 수입국 수출 목적 없음 |
| 반품·재수출 조건부 수입 | △ (조건·사정 판단) | → 제4장 제4절 참조 |
| 폐기 목적 수입물품 | ✗ (수출판매 아님) | 수출자 부담으로 국내 폐기하기 위해 도착하는 물품 (시행령 제17조 제7호) |

> ⚠ **핵심 판별 기준**: 수출판매 해당 여부는 **수입국으로의 수출 목적** 및 **소유권(title)의 이전**을 기준으로 판단한다. 소유권이 이전되지 않은 채 물품만 이동하는 경우 수출판매가 성립하지 않는다.

### 3. 제1방법 적용을 위한 전제조건 (4가지)

제1방법을 적용하여 과세가격을 결정하기 위해서는 **아래 4가지 조건을 모두 충족**해야 하며, 하나라도 미충족 시 제1방법 배제 후 순차 방법 적용.

```
전제조건 ① 수출판매의 성립
전제조건 ② 거래가격 배제요건 비해당 (관세법 제30조 제3항)
전제조건 ③ 실제지급가격의 확정 가능성 (객관적·수량화 가능)
전제조건 ④ 특수관계 있는 경우 → 가격에 영향 없음 입증 (관세법 제30조 제3항 제4호)
```

#### 3-1. 거래가격 배제요건 4가지 (관세법 제30조 제3항) — 핵심 암기

| 번호 | 배제요건 | 내용 |
|------|---------|------|
| ① | 처분·사용 제한 | 수입 후 처분·사용에 제한이 있는 경우 (→ 제4장 제3절) |
| ② | 조건·사정 영향 | 금액으로 계산할 수 없는 조건·사정에 의해 가격이 영향을 받은 경우 (→ 제4장 제4절) |
| ③ 사후귀속이익 | 적절히 조정 불가한 사후귀속이익이 있는 경우 (→ 제4장 제5절) |
| ④ 특수관계 영향 | 특수관계가 가격에 영향을 미친 경우 (→ 제4장 제6절) |

> ⚠ 단서: ①의 경우에도 **해당 제한이 거래가격에 실질적 영향을 미치지 않은 경우**, ②의 경우에도 **조건·사정의 영향이 금액으로 계산 가능한 경우**에는 제1방법 배제에서 제외. (관세법 제30조 제3항 단서)

---

### 4. 일괄거래(Lump-sum Transaction)의 과세가격 결정

#### 4-1. 의의

- **일괄거래**: 여러 물품을 하나의 가격으로 일괄 거래하는 경우로, 물품별 개별가격이 구분되지 않은 형태.
- 물품별 가격이 구분되지 않더라도, 합리적인 방법으로 안분 가능하면 제1방법 적용 가능.

#### 4-2. 처리 방법

**[STEP 1]** 계약서·인보이스 등에 **물품별 개별가격이 구분 명기**되어 있는가?
  - 구분 명기 O → 각 물품별로 개별 과세가격 결정 (제1방법 원칙 적용)
  - 구분 명기 X → [STEP 2]로

**[STEP 2]** 일괄가격을 **합리적 기준으로 안분** 가능한가?
  - 가능 O → 안분 후 제1방법 적용 (관세법 제30조 제1항)
  - 가능 X → 조건·사정 금액 계산 불가로 제1방법 배제 검토

#### 4-3. 안분 방법

```
물품 A의 과세가격 = 일괄가격 × (물품 A의 개별가격 / 전체 물품의 합산 개별가격)
```

> ※ 안분 기준: ① 개별 물품의 정상가격 비율, ② 무게·수량·부피 비율 등 객관적 기준 적용 가능.

#### 4-4. 기출 쟁점: 일괄거래에서 가격 미구분 물품

- **2019년 기출 (제36회)**: 측정장비(USD 400,000)는 개별 가격 구분 O, 검사장비+자동자료처리기계(USD 600,000)는 일괄 가격 → 기능단위기기(functional unit) 판단 시 일괄 취급하여 동일 세율(8%) 적용. 개별 안분 불요.
- **핵심**: 하나의 기능단위로 결합되어 단일 물품으로 분류되는 경우, 일괄가격 그 자체가 과세가격의 기초가 됨.

### 5. 수입국 도착 전 판매(연속 매매) 처리

#### 5-1. 의의

- 물품이 **수입국에 도착하기 전 보세구역에서 제3자에게 재판매(B/L 양수도)** 되는 경우, 최종 매수인이 수입 통관하는 가격을 과세가격으로 인정할 수 있는가의 문제.

#### 5-2. 판단 기준

- **원칙**: 관세법상 과세가격은 **수입신고를 하는 자(최종 수입자)가 실제 지급한 금액** 기준.
- **WTO협정 주해**: 수입국으로의 수출 목적으로 판매된 것이라면, 수입국 입항 전 연속 매매가 이루어진 경우에도 최종 판매가격을 거래가격으로 채택할 수 있다.
- **국내 처리**: 세관은 최종 구매자(양수인)의 실제 지급가격을 과세가격의 기초로 삼되, 구매자가 제시한 가격의 신뢰성·진정성을 검토함.

> → 상세: **제3장 제1절 B/L 양수도 부분** 및 **제6장 제5절 합리적 기준 방법** 참조

---

### 6. 과세가격 결정의 기준 시점

- 과세가격은 **수입신고일** 현재의 가격을 기준으로 결정한다. (관세법 제30조)
- 잠정가격 신고의 경우: 잠정가격으로 신고 → 확정 후 재신고. (→ 제8장 제2절 참조)

## [3] 판단 흐름도

```
[STEP 0] 수출판매 성립 여부 확인
        │
        ├─ 수출판매 아님 (증여·임차·내부이전 등)
        │         └→ 제1방법 적용 불가 → 제6방법 또는 별도 규정 적용
        │
        └─ 수출판매 성립
                  ↓
[STEP 1] 거래가격 배제요건 해당 여부 (관세법 제30조 제3항)
        │
        ├─ ① 처분·사용 제한 해당 → 단서 적용 가능 여부 검토
        │         └→ 단서 미적용 → 제1방법 배제
        │
        ├─ ② 조건·사정 영향
        │         ├─ 금액 계산 가능 → 조정 후 제1방법 적용
        │         └─ 금액 계산 불가 → 제1방법 배제
        │
        ├─ ③ 사후귀속이익 (적절 조정 불가) → 제1방법 배제
        │
        └─ ④ 특수관계 → 가격 영향 여부 검토 (시행령 제23조)
                  ├─ 영향 없음 입증 → 제1방법 적용
                  └─ 영향 있음 → 제1방법 배제
                            ↓
[STEP 2] 배제요건 비해당 → 실제지급가격 + 가산요소 확정
                  ↓
[STEP 3] 일괄거래 여부 확인
        ├─ 물품별 개별가격 구분 O → 각각 제1방법 적용
        └─ 물품별 개별가격 구분 X → 합리적 안분 후 적용
                  ↓
[STEP 4] 과세가격 = 실제지급가격 + 가산요소 (관세법 제30조 제1항)
```

## [4] 계산 공식 및 실수 포인트

### ■ 제1방법 과세가격 기본 공식

```
┌────────────────────────────────────────────────────────────┐
│  과세가격 = 실제지급가격                                       │
│          + 가산요소 (수수료·생산지원·권리사용료·운임·보험료 등)  │
│          - 공제요소 (수입항 이후 발생 비용, 연불이자 등)         │
└────────────────────────────────────────────────────────────┘
```

- **실제지급가격**: 구매자가 판매자에게 실제로 지급하였거나 지급할 총금액 (관세법 제30조 제2항) → 제3장 제2절 상세
- **가산요소**: 관세법 제30조 제1항 각호 열거 → 제5장 각 절 상세
- **공제요소**: 금액이 명백히 구분 가능한 경우, 수입 후 발생 비용 공제 가능 → 제3장 제4절 상세

### ■ 일괄거래 안분 공식

```
┌──────────────────────────────────────────────────────────────┐
│  물품 A 과세가격 = 일괄가격 × (A의 참조가격 / 전체 참조가격 합계) │
└──────────────────────────────────────────────────────────────┘
```

### ■ ⚠ 실수하기 쉬운 포인트 체크리스트

- □ **임차수입**을 수출판매로 혼동하지 말 것: 소유권 이전 없으면 제1방법 적용 불가 → 시행규칙 제7조의4 적용
- □ **위탁판매수입**의 경우: 수입 시점에 매매 미성립 → 제1방법 배제, 나중에 판매 성립 후 과세가격 재결정 문제로 이어짐
- □ **배제요건 단서** 반드시 확인: ① 처분제한이 있더라도 가격 영향 없으면 제1방법 적용 가능
- □ **조건·사정의 금액 계산 가능 여부**: 계산 가능 → 제1방법 유지(조정), 계산 불가 → 제1방법 배제
- □ **일괄거래**에서 기능단위기기 해당 여부: 기능단위기기이면 전체를 하나의 물품으로 분류 → 안분 불필요
- □ **과세가격 기준 시점**: 수입신고일 현재 기준 (계약일·결제일 아님)
- □ 특수관계 영향 판단 → **비교가격(3가지)** 또는 **판매주변상황** 검토 방법 연결 (→ 제4장 제6절)

`📌 최근 심화 경향 (2022~2025)`
- 복합 거래 종합문제(30~50점)에서 "수출판매 성립 여부"를 전제 논점으로 먼저 쓰고 이후 가산요소 계산으로 연결하는 구성이 표준화됨.
- WTO협정 제1조 주해 직접 인용("수입국으로의 수출을 위한 판매") 요구 증가.
- 2022년 이후 배제요건 4가지를 단순 열거하는 것이 아니라 **구체적 거래사실에 적용하여 해당 여부 판단**하는 심화 문제 출제 강화.
- 특수관계 배제요건(④)의 경우 → 이전가격 연구(Transfer Pricing Study) 활용 가능성 연계 (→ 제4장 제6절).

## [7] 단권화 암기 키워드 & 체크리스트

`✏ 고득점 키워드 (8개)` — 답안에 반드시 포함해야 할 핵심 표현

1. **"우리나라에 수출하기 위하여 판매되는 물품"** — 수출판매의 법적 정의 (관세법 제30조 제1항)
2. **"실제로 지급하였거나 지급하여야 할 가격"** — 실제지급가격의 법적 정의
3. **"처분 또는 사용에 대한 제한"** — 배제요건 ①
4. **"금액으로 계산할 수 없는 조건 또는 사정"** — 배제요건 ②
5. **"사후귀속이익"** — 배제요건 ③
6. **"특수관계가 가격에 영향을 미친 경우"** — 배제요건 ④
7. **"소유권(title)의 이전"** — 수출판매 성립의 핵심 판단 기준 (관세평가 운영에 관한 고시 제15조 제1항)
8. **"합리적 안분"** — 일괄거래 시 물품별 과세가격 산출 방법

`★ 절 핵심 체크리스트` — 완전히 이해했다면 답할 수 있어야 할 질문

- ★★★ 1. 수출판매가 성립하지 않아 제1방법을 적용할 수 없는 거래 유형을 3가지 이상 열거하라.
- ★★★ 2. 관세법 제30조 제3항의 거래가격 배제요건 4가지를 법 조문 수준으로 서술하라.
- ★★★ 3. 배제요건 ②(조건·사정)에서 '금액 계산 가능한 경우'와 '불가능한 경우'의 처리 방법 차이를 설명하라.
- ★★  4. 일괄거래에서 물품별 가격이 구분되지 않을 때 과세가격을 어떻게 결정하는가?
- ★★  5. 특수관계가 있는 경우 제1방법을 적용하기 위한 요건은 무엇인가?
- ★   6. 임차수입물품에 제1방법이 적용되지 않는 이유를 수출판매 개념과 연결하여 설명하라.
- ★   7. WTO 관세평가협정 제1조 주해에서 규정하는 수출판매의 의미를 국내법 조문과 비교하여 서술하라.
