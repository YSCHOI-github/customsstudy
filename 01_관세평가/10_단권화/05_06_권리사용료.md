
# 관세평가 단권화 교과서

---

```
┌──────────────────────────────────────────────────────────────────┐
│ 제5장 제6절  권리사용료 (Royalties and Licence Fees)              │
│ 출제 티어: ★★★ A  (필수 완전 정복)                               │
│ 출제율: 91%  (제30~42회 중 10/11개년 출제)                        │
│ 대표 출제 유형: 특허권(30·42회), 저작권(31·32·40회), 상표권(35·40회) │
│ 핵심 법령: 관세법 제30조 제1항 제4호, 시행령 제26조               │
│ WTO협정: 제8조 제1항(c) 및 주해(Annex I, Note to Art. 8.1(c))    │

└──────────────────────────────────────────────────────────────────┘
```

## [1] 핵심 요약 (3줄 이내)

권리사용료는 **①물품관련성 + ②거래조건성 + ③직·간접 지급** 3요건을 모두 충족해야 실제지급가격에 가산한다. 권리에 국내 생산·마케팅 등 활동 대가가 포함된 경우 안분하여 수입물품 해당분만 가산한다. 출제 빈도 1위(91%) 절로서 유형별 물품관련성 암기, 거래조건성 3요건 서술, 안분 산출식 계산이 반드시 출제된다.

---

## [2] 구조화된 이론 분석

### 1. 권리사용료의 의의 및 법적 근거

**의의**: 수입물품과 관련하여 지식재산권을 사용하는 대가로 구매자가 판매자 또는 제3자에게 지급하는 금액 (관세법 제30조 제1항 제4호)

**가산 원칙**: 실제지급가격에 포함되지 않더라도, 아래 요건 충족 시 실제지급가격에 가산하여 과세가격 결정 (관세법 제30조 제1항)

**대상 권리 (관세법 제30조 제1항 제4호)**:
  - 특허권 / 실용신안권 / 디자인권 / 저작권 / 상표권
  - **이와 유사한 권리** (영업비밀·know-how 포함 여부 → 개별 물품관련성으로 판단)

> [WTO협정 제8조 제1항(c)]: 수입물품과 관련하여 특허권·상표권·저작권 등의 사용 대가로서 구매자가 직접 또는 간접으로 지급하는 권리사용료. 단, **수입물품의 재판매·처분·사용으로 발생하는 수익의 환급분은 제외**.

---

### 2. 가산의 3대 요건 (관세법 시행령 제19조)

#### 요건 ① — 물품관련성 (시행령 제19조 제3항)

수입물품에 해당 권리가 *체화*되거나, 해당 권리를 *실시하기 위해 고안*된 물품이어야 한다.

| 권리 유형 | 물품관련성 충족 요건 |
|-----------|-------------------|
| **특허권·실용신안권** | ① 수입물품에 특허·실용신안이 **체화**되어 있는 경우 / ② **제조방법** 특허를 실시하기 위해 고안된 설비 등 수입물품 |
| **디자인권** | 수입물품에 해당 디자인이 **포함**되어 있는 경우 |
| **상표권** | 수입물품에 해당 상표가 **부착**되어 있는 경우 |
| **저작권** | 수입물품에 저작물이 **복제**되어 있거나 **표현**되어 있는 경우 |
| ⚠ 저작권 **예외** | **저작물을 복제하는 권리(재현생산권)** → WTO협정 주해 적용 (아래 별도 서술) |

> [WTO협정 제8조 제1항(c) 주해 — 재현생산권]:
> "수입물품에 복제되어 있는 저작물에 관한 권리사용료는 과세가격에 포함되나, 수출국에서 수입물품을 **재현생산하는 권리**에 대한 권리사용료는 과세가격에 포함되지 아니한다."
>
> - **포함**: 음반·CD에 복제된 음악의 저작권 사용료 → 물품관련성 O → 가산
> - **불포함**: 마스터테이프에서 음반 복제·제조할 권리 → 재현생산권 → 가산 불가 (그러나 마스터테이프 자체는 생산지원비용으로 별도 가산 검토)

---

#### 요건 ② — 거래조건성 (시행령 제19조 제2항)

권리사용료가 **수입물품 구매의 거래조건**으로서 지급되어야 한다.

**직접 충족 기준** (시행령 제19조 제5항):
1. 구매자가 권리사용료를 지급하지 않으면 **해당 수입물품을 구매할 수 없는** 경우
2. 권리사용료가 **판매자 또는 판매자가 지정하는 자**에게 지급되는 경우

**거래조건 해당 여부 판단 시 고려사항** (시행규칙 제4조의2 — 2023년 출제):
1. 수입물품 구매계약과 권리사용계약의 **연계 여부** — 계약서상 상호 참조 또는 연동 조건 존재
2. 권리사용료 책정방식이 수입물품의 **가격 결정방식과 연계**되어 있는지 여부
3. 권리사용료 지급 없이는 판매자가 **물품을 생산하거나 판매할 수 없는** 여부
4. 수입물품 가격이 권리사용료 지급 의무의 **이행을 반영하여 결정**되는지 여부
5. 권리사용료의 **지급 시기·조건**이 수입물품 인도조건과 연계되어 있는지 여부
6. 권리사용자가 권리사용료를 지급하지 않으면 **라이선서가 계약을 해지**할 수 있는지 여부

---

#### 요건 ③ — 직·간접 지급 (관세법 제30조 제1항 제4호)

구매자가 **직접 또는 간접으로** 지급 — 판매자 이외 제3자(특허권자)에게 지급하는 경우도 포함

**원천징수세 처리** (2020년 기출):
- 구매자가 권리사용료 소득에 대해 원천징수 납부 → 권리사용료 총액(원천징수 전 금액) 기준 가산
- 이중과세방지협약에 따라 원천징수 영수증 제공 여부는 가산 여부와 무관
- 원천징수세액도 구매자가 권리사용료의 일부로 **간접적으로 지급**한 것으로 봄

---

### 3. 권리사용료의 가산 범위 조정 — 국내 활동 대가 포함 시 (시행령 제19조 제6항)

**원칙**: 권리사용료에 국내 생산, 마케팅, 그 밖의 사업 활동에 대한 대가가 포함된 경우 → 해당 금액 제외 후 가산

**안분방법**: 제외할 금액이 불명확한 경우 **객관적이고 수량화할 수 있는 자료에 의해 안분계산**

```
안분의 논리:
  권리사용료 총액 = [수입물품 관련분] + [국내 활동 대가]
  가산 대상       = 권리사용료 총액 × (수입부분품 등의 가격 / 완제품가격(세금·권리사용료 제외))
```

## [3] 판단 흐름도

```
[STEP 1] 권리 유형 확인 (관세법 제30조 제1항 제4호)
         특허·실용신안·디자인·저작권·상표권·유사권리?
              ↓ 해당 없음 → 가산 불가
              ↓ 해당 있음

[STEP 2] 물품관련성 검토 (시행령 제19조 제3항)
         → 수입물품에 권리가 체화/부착/포함/복제되어 있는가?
         → 특허: 제조방법 특허 실시를 위해 고안된 설비인가?
              ↓ 미충족 → 가산 불가
              ↓ 충족

[STEP 3] 거래조건성 검토 (시행령 제19조 제5항)
         → 지급 않으면 물품 구매 불가? 또는 판매자(지정자)에게 지급?
         → 6가지 고려사항으로 간접 판단
              ↓ 미충족 → 가산 불가
              ↓ 충족

[STEP 4] 직·간접 지급 확인 (관세법 제30조 제1항 제4호)
         → 구매자가 직접 또는 간접 지급 여부?
              ↓ 미충족 → 가산 불가
              ↓ 충족 (원천징수세 포함 총액 기준)

[STEP 5] 국내 활동 대가 포함 여부 (시행령 제19조 제6항)
         → 포함 없음: 권리사용료 전액 가산
         → 포함 있음 (명확): 해당 금액 제외 후 가산
         → 포함 있음 (불명확): 안분방법으로 조정액 산출
              ↓
[STEP 6] 과세가격에 가산 → 과세가격 = 실제지급가격 + 조정액
```

---

### 비교표: 물품관련성 有 vs 無

| 구분 | 가산 O | 가산 X |
|------|--------|--------|
| 특허권 | 수입물품에 특허가 체화 / 특허 제조방법 실시용 설비 수입 | 수입 후 국내공정에만 사용되는 별도 특허 |
| 저작권 | CD·음반에 **복제**된 저작물의 저작권 (표현 체화) | 음반 **제조(재현생산)권** → WTO주해 제외 |
| 상표권 | 수입물품에 상표가 **부착** | 수입 후 국내 부착 예정인 상표 사용료 |
| 국제마케팅비 | 상표권 사용과 실질적으로 동일 성격이면 가산 (2016년 국제마케팅비 사례) | 단순 광고 서비스 대가 (물품 무관) |

## [4] 계산 공식 및 실수 포인트

### ■ 공식 A — 조정액 산출식 (장기반복 수입 — 국내 활동 대가 포함 시)

```
┌──────────────────────────────────────────────────────────────────┐
│                                                                  │
│           수입부분품 등의 가격 합계                               │
│  조정액 = ─────────────────────────── × 권리사용료 총액           │
│           완제품가격(세금·권리사용료 제외)                        │
│                                                                  │
│  ※ 완제품가격 = 수입부분품 등의 가격 + 국내 생산 등 활동 금액    │
│                                                                  │
└──────────────────────────────────────────────────────────────────┘
```

### ■ 공식 B — 가산율 산출식 (장기반복 수입)

```
┌──────────────────────────────────────────────────────────────────┐
│                                                                  │
│            조정액                                                │
│  가산율 = ──────────────────── × 100 (%)                         │
│            수입물품 CIF가액 합계                                  │
│                                                                  │
│  ※ 가산율이 산출되면, 이후 수입 건마다:                          │
│     가산액 = 해당 수입건 CIF가액 × 가산율(%)                      │
│                                                                  │
└──────────────────────────────────────────────────────────────────┘
```

### ■ 공식 C — 단회(일회성) 수입 — 국내 활동 대가 없는 경우

```
┌──────────────────────────────────────────────────────────────────┐
│  국내 활동 대가 미포함 시:                                        │
│                                                                  │
│  조정액 = 권리사용료 총액 (전액 가산)                             │
│                                                                  │
│  과세가격 = 실제지급가격(CIF) + 권리사용료 전액                   │
│                                                                  │
└──────────────────────────────────────────────────────────────────┘
```

---

### ■ 계산 절차

**STEP 1.** 권리사용료 가산 요건 충족 여부 확인 (물품관련성·거래조건성·직간접 지급)

**STEP 2.** 국내 활동 대가 포함 여부 확인
  - 포함 있음(명확): 해당 금액 직접 제외
  - 포함 있음(불명확): 안분 필요 → 수입물품 CIF가액 / 특허물품 생산가액 계산

**STEP 3.** 조정액 산출 (안분 또는 전액)

**STEP 4.** 가산율 산출 (장기반복 수입 시) = 조정액 ÷ CIF가액 × 100

**STEP 5.** 과세가격 = 실제지급가격(CIF) + 조정액

---

### ■ ⚠ 실수하기 쉬운 포인트 체크리스트

- □ **원천징수세 포함 여부**: 구매자가 원천징수 납부한 세액 → 권리사용료 **총액** 기준으로 가산 (실제 송금액 아님)
- □ **재현생산권 구분**: 음반에 복제된 저작물 = 가산, 음반 제조(재현생산)권 = 가산 불가
- □ **안분 분모**: "수입물품 가격"이 아니라 "**완제품가격**(세금·권리사용료 제외)" (고시 제22조 제1항 제2호 법정 용어)
- □ **가산율 적용 대상**: 조정액 산출 후 "해당 수입건 CIF가액 × 가산율"로 각 수입건별 가산
- □ **영업비밀(know-how)**: 단독 지급 시 물품관련성 별도 검토 필요; 특허와 묶인 경우에도 유사 권리 해당 여부 판단
- □ **사후귀속이익 가산**: 수입 후 국내 판매수익의 일부를 판매자에게 귀속하는 구조 → **제5호 사후귀속이익으로 과세가격에 가산 O** (관세법 제30조 제1항 제5호; 제4호 권리사용료와 구별; WTO협정 제8조 제1항(d))
- □ **가산율 소수점**: 문제에서 지정하는 반올림 자리수 엄수 (2013: 소수점 셋째자리에서 반올림→둘째자리, 2025: 첫째자리에서 반올림→둘째자리)

### 계산 예제 — 제30회 (2013년) 재현

**거래사실**:
- 특허권 사용대가(국내 활동 포함): 4억원
- 수입원재료 CIF가액: 8억 9,000만원
- 특허물품 생산가격(세금·권리사용료 제외): 14억 9,000만원

**풀이**:
```
조정액 = 4억 × (8.9억 / 14.9억)
       = 4억 × 0.59732...
       = 약 2.3893억원

가산율 = (2.3893억 / 8.9억) × 100
       = 26.8461...%
       → 소수점 셋째자리 반올림 → 26.85%

과세가격 = 8.9억 + 2.3893억 = 약 11.29억원
```

---

### 계산 예제 — 제42회 (2025년) 재현

**거래사실**:
- 특허권(조립+도장공장 생산방법): 8억원
- 영업비밀: 2억원 (별도 쟁점)
- 수입설비 CIF: 조립공장 45억 + 도장공장 15억 = **60억원**
- 전제: 과세당국이 물품관련성·거래조건성 충족 인정; 국내 활동 대가 없음

**풀이**:
```
물음2: 과세대상 수입물품
  → 특허 = "조립 및 도장공장 자동화 라인" 전체에 관한 것
  → 조립공장 컨베이어 + 도장공장 컨베이어 양 설비 모두 해당

물음3: 조정액 및 가산율
  국내 활동 대가 미포함 → 특허 8억 전액 가산

  조정액 = 8억원 (전액)

  가산율 = (8억 / 60억) × 100
         = 13.333...%
         → 소수점 셋째 자리에서 반올림 → 둘째 자리까지: 13.33%

  과세가격 = 60억 + 8억 = 68억원

  ※ 영업비밀 2억: 관세법상 "이와 유사한 권리" 해당 여부
    → 특허와 묶인 라이센스 계약으로 지급 → 별도 물품관련성 검토
    → 수입물품에 영업비밀이 구현된 경우 → 유사권리로 추가 가산 가능
```

---

`📌 최근 심화 경향 (2021~2025)`

- **WTO협정 조문 직접 인용 강화**: 2023년에 WTO 제8조 제1항(c) 주해 "재현생산권" 개념을 협정 원문 기준으로 출제 → 단순 국내법 암기 부족, 협정 원문과 국내법 비교 필수
- **거래조건 판단 고려사항 6가지 상술 요구**: 2023년 40회 물음2에서 시행규칙 제4조의2 고려사항을 별도 배점(6점)으로 출제 → 조문 수준 암기 필요
- **안분방법 이론+계산 분리 출제**: 2021년 38회에서 안분원칙·조정액·가산율 산출식을 각각 독립 물음으로 분리해 총 10점 배점 → 공식 암기 필수
- **원천징수세 처리**: 2020년 37회에서 처음 명시적 출제 → 2025년 42회에도 동일 구조 재등장 → 반복 출제 확인

## [7] 단권화 암기 키워드 & 체크리스트

### ✏ 고득점 키워드 (답안에 반드시 포함)

1. **물품관련성** — 수입물품에 체화/부착/포함/복제
2. **거래조건성** — 지급하지 않으면 구매 불가; 판매자·지정자에게 지급
3. **직접 또는 간접으로 지급** — 원천징수세 포함 총액 기준
4. **재현생산권** — WTO협정 주해상 가산 제외 대상
5. **안분계산** — 국내 활동 대가 포함 시 수입물품 해당분만 가산
6. **조정액** = 권리사용료 총액 × (수입부분품 등의 가격 / 완제품가격(세금·권리사용료 제외))
7. **가산율(%)** = 조정액 / 수입부분품 등의 가격 × 100
8. **거래조건 판단 고려사항** — 시행규칙 제4조의2 6개 항목
9. **사후귀속이익 가산** — 판매 후 수익을 판매자에게 귀속하는 구조 → 관세법 제30조 제1항 제5호로 과세가격에 **가산 O**
10. **이와 유사한 권리** — 영업비밀·know-how 포함 여부 개별 판단

---

### ★ 절 핵심 체크리스트

- ★★★ 1. 권리사용료 3대 가산 요건(물품관련성·거래조건성·직간접 지급)과 각 요건의 세부 판단기준은? (시행령 제19조 제3항·제2항·제5항, 시행규칙 제4조의2)
- ★★★ 2. 권리 유형별 물품관련성 충족 요건(특허·디자인·상표·저작권)을 각각 서술할 수 있는가?
- ★★★ 3. 국내 활동 대가 포함 시 조정액 산출식과 가산율 산출식을 쓰고 계산할 수 있는가? (장기반복 수입)
- ★★  4. WTO협정상 재현생산권이 가산 제외되는 이유와 저작물 복제품의 가산과의 구별은?
- ★★  5. 거래조건 판단 시 고려해야 할 6가지 사항을 모두 열거할 수 있는가? (시행규칙 제4조의2)
- ★★  6. 원천징수세가 포함된 경우 가산 기준금액(총액 vs 실제 송금액)을 올바르게 결정할 수 있는가?
- ★    7. 수입물품 판매 후 수익을 판매자에게 귀속하는 구조(사후귀속이익, Subsequent Proceeds)는 제4호 권리사용료와 구별되며 제5호에 따라 과세가격에 가산됨을 설명할 수 있는가?
