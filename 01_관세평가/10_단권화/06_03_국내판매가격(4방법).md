┌─────────────────────────────────────────────────────────────────┐
│ 제6장 제3절  국내판매가격을 기초로 한 과세가격 결정방법           │
│             (제4방법 / Deductive Method)                         │
│                                                                 │
│ 출제 티어: ★★ B  (중요)                                          │
│ 출제율: 55% (제31·32·35·36·37·38·42회 관련 출제)                │
│ 핵심 법령: 관세법 제33조 / 시행령 제27조                          │
│ 연계 절: → 제2장 제2절 평가방법 적용순위                          │

└─────────────────────────────────────────────────────────────────┘
```

## [1] 핵심 요약 (3줄 이내)

- 제4방법은 수입물품을 **국내에서 판매한 단위가격**에서 이윤·일반경비·국내비용·관세 등을 역산하여 과세가격을 결정하는 방법이다 (관세법 제33조).
- 출제의 핵심은 **단위가격 결정**(최대수량 + 판매기간 + 비특수관계 필터링)과 **이윤 및 일반경비 비율 선택**(납세의무자 비율 vs 세관장 동종동류 비율)의 두 논점이다.
- 계산 문제로 반드시 출제되므로 공식·단계·실수 포인트를 완전히 숙지해야 한다.

---

## [2] 구조화된 이론 분석

### 1. 적용 전제 및 요건

1. **순차 적용의 원칙** (관세법 제30조~제35조)
   - 제1방법(거래가격) → 제2방법(동종동질) → 제3방법(유사물품) 순서로 적용 불가 판정 후 제4방법 적용
   - **예외**: 납세의무자 신청 시 제4방법과 제5방법(산정가격)의 순서를 교환할 수 있음 (관세법 제33조 제1항 단서)

2. **적용 요건 개요** (관세법 제33조 제1항)
   - 수입물품 또는 동종·동류물품이 **우리나라에서** 판매될 것
   - 수입신고일 또는 거의 동시에(±10일, 시행령 제27조 제1항) 또는 수입신고일 후 90일 이내(시행령 제27조 제2항)에 판매될 것
   - 비특수관계 구매자에게 판매된 것
   - 수입 상태 그대로(추가 가공·변형 없이) 판매된 것

---

### 2. 단위가격 결정 — 4단계 필터링 (핵심 논점)

| 단계 | 필터링 기준 | 비고 |
|------|-----------|------|
| ① 판매기간 | 수입신고일과 거의 동시(가격변동이 거의 없다고 인정되는 기간, 시행령 제27조 제3항), 단 수입신고일부터 90일이 지난 후의 가격은 제외 (제27조 제3항 단서) | '거의 동시' = 90일 이내 가격변동 없는 기간 |
| ② 특수관계 | 특수관계 없는 구매자에게 판매된 것만 사용 | 특수관계자 판매 전부 제외 |
| ③ 수입 상태 | 추가 가공·변형 없이 수입된 상태로 판매된 것 | 이 조건 불충족 시 별도 조정 |
| ④ 최대수량 | 동일한 단위가격으로 판매된 총수량이 **가장 많은** 단위가격 선택 | 단일 거래 기준 아닌 합산 수량 |

> **[WTO협정 제5조 주해]**: "최대수량으로 판매된"이란 단일 가격수준별로 합산한 수량 중 최다 수량을 의미하며, 판매일자·건수가 아닌 **단위가격별 합산 수량**이 기준이다.

#### 판매기간 적용 기준

```
  [원칙] 수입신고일과 거의 동시(수입신고일의 가격과 가격변동이 거의 없다고
         인정되는 기간) 중 판매 — 단, 수입신고일부터 90일이 지난 후의 가격은 제외
         (시행령 제27조 제3항)
         ↓ 해당 판매 없음
         → 제4방법 적용 불가 → 제5방법으로 이동
```

> ⚠ 법령(시행령 제27조 제3항)에는 '±10일'이라는 수치가 없다. '거의 동시'는 물품의 종류·특성에 따라 판단하며, 90일이 지난 판매가격은 제외한다.

---

### 3. 공제항목 3가지 (관세법 제33조 제1항 각호)

| 공제항목 | 내용 | 비고 |
|---------|------|------|
| ① 이윤 및 일반경비 | 국내판매 시 동종·동류물품에 통상 부가되는 이윤 + 판매관리비 | 비율 결정 방법 별도 설명 |
| ② 수입항 도착 후 국내비용 | 통상의 운임·보험료·하역비·창고료 등 | 수입항 이후에 발생한 비용만 |
| ③ 조세 및 공과금 | 수입 및 국내판매 관련 관세·내국세·부가가치세·기타 공과금 | 관세율이 주어지면 연립방정식 |

> **핵심 판단**: 수입항 **이전** 비용 → 과세가격에 포함(공제 불가) / 수입항 **이후** 비용 → 공제 대상

---

### 4. 이윤 및 일반경비 비율 결정 (핵심 쟁점)

#### (1) 두 가지 비율

| 구분 | 근거 | 산출 방법 |
|------|------|---------|
| **납세의무자 비율** | 관세법 제33조 제1항 제1호 | 납세의무자가 제출한 일반적으로 인정된 회계보고서 기준 |
| **세관장 동종동류 비율** | 시행령 제27조 제5항·제6항 | 세관장이 동종·동류 수입물품 국내판매 통계를 기초로 산출 |

#### (2) 적용 원칙 — 120% 룰 (시행령 제27조 제5항)

```
  납세의무자 비율 ≤ 동종동류 비율 × 120%
         ↓ YES
  → 납세의무자 비율 적용 ✔
         ↓ NO (납세의무자 비율 > 동종동류 비율 × 120%)
  → 세관장 동종동류 비율 적용 ✔
```

> **이유**: 납세의무자가 제출한 이윤·경비 비율이 동종·동류 업계 평균의 **1.2배(120%)를 초과**하면 지나친 공제를 인정하지 않고 업계 평균을 적용한다 (시행령 제27조 제5항). 납세의무자 비율이 동종동류 비율보다 낮으면 120% 기준을 자동 충족하므로 납세의무자 비율을 그대로 사용한다.

> ⚠ **판단 예시**: 납세의무자 25%, 동종동류 20% → 20%×120%=24%, 25%>24% → **동종동류 20% 적용**  
> 납세의무자 44%, 동종동류 40% → 40%×120%=48%, 44%≤48% → **납세의무자 44% 적용**

#### (3) 이윤 및 일반경비 계산식

```
이윤 및 일반경비 비율(r) = (판매비 및 관리비 + 영업이익) / 매출액

즉, r = (매출액 - 매출원가) / 매출액 = 매출총이익률
```

> ⚠ 납세의무자 비율 계산 시 소수점 이하 절사 여부 확인 (문제에서 명시)

---

### 5. 동종·동류물품의 개념 (제4방법에서의 사용)

- 제4방법에서 '동종·동류물품'은 **이윤 및 일반경비 기준 비율 산출** 목적으로 사용 (제2방법의 동종·동질물품과 다른 개념)
- **동종·동류물품** (Goods of the same class or kind): 특정 산업 또는 산업부문에 속하는 물품군으로서, 동종 또는 동류에 해당하는 수입물품 포함 (WTO협정 제5조 제1항(b))
- 수입국의 생산자가 생산한 물품은 포함되지 않음

---

### 6. 위탁판매 수입물품의 특칙

- 위탁판매인 경우: 수입자(수탁자)는 판매 주체이나 소유권은 위탁자(수출자)에게 있음
- 이 경우 **이윤 및 일반경비** 대신 **위탁판매 수수료**(통상적으로 인정된 금액)를 공제 (관세법 제33조 제1항 제2호)
- 수수료가 통상적 수수료인지 확인 필요

---

### 7. 동종·동류물품이 수입된 상태 그대로 판매되지 않은 경우 (특례)

수입물품이 추가 가공·변형 후 판매된 경우에는 적용 불가가 원칙이나, 납세의무자 요청 시 가공에 따른 가치증가분을 적절히 공제하여 사용 가능 (관세법 제33조 제3항).

## [3] 판단 흐름도

### A. 단위가격 결정 흐름

```
  ┌──────────────────────────────────────────────────────┐
  │             제4방법 단위가격 결정 흐름                  │
  └──────────────────────────────────────────────────────┘

  [STEP 1] 판매 데이터 수집
           수입신고일과 거의 동시(가격변동이 거의 없는 기간) 판매 목록 추출
           단, 수입신고일부터 90일이 지난 후 판매가격은 제외 (시행령 제27조 제3항)
               ↓ 해당 판매 없음
           → 제4방법 적용 불가

  [STEP 2] 특수관계자 판매 제거
           → 특수관계 구매자에게 판매된 것 전부 제외

  [STEP 3] 동일한 수입상태 확인
           → 추가 가공·변형이 있는 경우 제외 (또는 별도 조정)

  [STEP 4] 가격별 수량 합산
           → 동일 단위가격으로 판매된 건들 수량 합산

  [STEP 5] 최대수량 가격 선택
           → 합산 수량이 가장 많은 단위가격 = 과세가격 결정 기준 단위가격
```

### B. 과세가격 계산 흐름

```
  [단위가격 P 확정]
       ↓
  [이윤 및 일반경비 비율 r 결정] — 120% 룰 (시행령 제27조 제5항)
  납세의무자 r ≤ 동종동류 r × 120% → 납세의무자 r 적용
  납세의무자 r > 동종동류 r × 120% → 동종동류 r 적용
       ↓
  [공제항목 C 산정]
  C = 수입항 후 국내비용(per unit) + 조세·공과금(per unit)
       ↓
  [과세가격 X 산출]
  조세가 고정금액으로 주어진 경우 → 단순 공제
  조세가 세율(t)로 주어진 경우   → 연립방정식
       ↓
  [총 과세가격 = X × 수입수량]
```

## [4] 계산 공식 및 실수 포인트

### ■ 기본 공식 (과세가격 X)

**[유형 A] 조세가 고정 금액으로 주어진 경우 (단순 공제)**

```
┌───────────────────────────────────────────────────────────┐
│  X = P × (1 − r) − C_국내비용(per unit) − C_조세(per unit) │
│                                                           │
│  P: 국내판매 단위가격                                       │
│  r: 이윤 및 일반경비 비율                                   │
│  C_국내비용: 수입항 후 운임·보험료 등 (총액÷수량)            │
│  C_조세: 관세·내국세 등 총액 ÷ 수량                        │
└───────────────────────────────────────────────────────────┘
```

**[유형 B] 관세가 세율(t)로만 주어진 경우 (연립방정식)**

```
┌───────────────────────────────────────────────────────────┐
│  X + X·t = P·(1−r) − C_국내비용                           │
│  X·(1+t) = P·(1−r) − C_국내비용                           │
│                                                           │
│  ∴  X = [ P·(1−r) − C_국내비용 ] / (1+t)                 │
└───────────────────────────────────────────────────────────┘
```

> ⚠ 부가가치세가 별도로 세율로 주어진 경우:
> X·(1+t_관세)·(1+t_부가세) 항을 공제 항목으로 추가 → 더 복잡한 연립방정식

---

### ■ 계산 절차 (STEP 분해)

**STEP 1.** 판매 데이터에서 유효 판매 필터링
- 판매기간 (수입신고일부터 90일 이내, 가격변동이 거의 없는 기간) ✔/✘
- 특수관계 제거 ✔/✘
- 수입 상태 동일 여부 ✔/✘

**STEP 2.** 가격별 합산 수량 계산 후 최대수량 단위가격(P) 선택

**STEP 3.** 이윤 및 일반경비 비율(r) 결정 — **120% 룰 적용** (시행령 제27조 제5항)
- 납세의무자 비율 계산 = (판매비관리비 + 영업이익) / 매출액 (소수점 셋째 자리 계산 후 반올림 → 둘째 자리, 시행규칙 제7조의9 제4항)
- 납세의무자 비율 ≤ 동종동류 비율 × 120% → 납세의무자 비율 적용
- 납세의무자 비율 > 동종동류 비율 × 120% → 세관장 동종동류 비율 적용

**STEP 4.** 수입항 후 국내비용(C) 산정 (총액 ÷ 수량 = 단위당)

**STEP 5.** 조세·공과금(T) 산정
- 고정 금액 → T = 총액 ÷ 수량
- 세율만 제시 → 연립방정식으로 풀기

**STEP 6.** 단위 과세가격(X) 계산 → 총 과세가격(X × 수량)

---

### ■ ⚠ 실수하기 쉬운 포인트 체크리스트

- □ **최대수량** = 단위가격별 **합산** 수량 기준 (단일 거래 건 기준이 아님)
  - 예: 100원에 50개, 100원에 70개 → 100원 기준 수량은 120개
- □ **판매기간 상한**: 수입신고일부터 **90일 이내** — 90일이 지난 후 가격은 제외 (법령상 '±10일' 수치는 없음)
- □ **특수관계자 판매는 반드시 제외** — 가격이 비슷해도 관계없음
- □ 납세의무자 이윤 비율 계산: 소수점 **셋째 자리까지 계산 후 반올림 → 둘째 자리** (시행규칙 제7조의9 제4항, 임의 절사 불가)
- □ **납세의무자 비율 vs 동종동류 비율** — **120% 룰**: 납세의무자 비율이 동종동류 비율의 120% 이하이면 납세의무자 비율 적용, 초과하면 동종동류 비율 적용 (단순 '더 높은 쪽' 아님)
- □ 수입항 **도착 후** 발생 비용만 공제 — 수입항 이전(선적전 검사비, 체선료 등)은 공제 불가
- □ 연립방정식 유형에서 **부가가치세가 추가** 세율로 주어지면 방정식이 더 복잡해짐
- □ 위탁판매: 이윤 및 일반경비 → **위탁판매 수수료**로 대체 (관세법 제33조 제1항 제2호)

### 【제37회(2020) 문제4】 전기면도기 — 기본 유형

**거래 데이터**

| 순서 | 판매일자 | 수량 | 단위가격 | 유효 여부 |
|------|---------|------|---------|---------|
| 1 | 02.15 | 300개 | 90,000원 | ✔ 90일 이내 |
| 2 | 03.20 | 300개 | 100,000원 | ✔ 90일 이내 |
| 3 | 04.10 | 100개 | 90,000원 | ✔ 90일 이내 |
| 4 | 05.10 | 450개 | 80,000원 | ✘ 90일 초과 (기준일 02.02 + 90일 = 05.02) |
| 5 | 06.08 | 200개 | 90,000원 | ✘ 90일 초과 |

수입신고일 2020.02.02 → 90일 기준일: 2020.05.02

유효 판매별 합산:
- 90,000원: 300 + 100 = **400개** ← 최대
- 100,000원: 300개

→ **단위가격 P = 90,000원**

이윤 및 일반경비 비율 (120% 룰 적용):
- 납세의무자 비율: 25%
- 세관장 통보: 20%
- 동종동류 비율 × 120% = 20% × 1.2 = **24%**
- **25% > 24% → 세관장 동종동류 비율 20% 적용** (시행령 제27조 제5항)

총 수입물량: 1,350개 (과세가격 산출 대상)

조세 및 공과금: 11,150,000원 (고정 금액) → 단위당 = 11,150,000 / 1,350 ≈ 8,259원
국내비용: 1,600,000원 / 1,350 ≈ 1,185원

단위 과세가격:
```
X = 90,000 × (1 − 0.20) − 1,185 − 8,259
  = 72,000 − 9,444
  = 62,556원 (단위)

총 과세가격 = 62,556 × 1,350 = 84,450,600원
```
*(정확한 소수점 처리는 문제 지시에 따름)*

| 2 | 3/8 | 1,000개 | 110 c.u. | ✔ |

| 7 | 3/30 | 1,100개 | 100 c.u. | ✘ 기간 초과(3/25 이후) |

가격별 합산:
- 100 c.u.: 2,500개
- 110 c.u.: 1,000 + 1,000 + 1,500 = **3,500개** ← 최대
- 120 c.u.: 1,200개

→ **단위가격 P = 110 c.u.**

X = 110 − 110×10% − 5 − 10 − X×20%
X + 0.2X = 110 − 11 − 15
1.2X = 84
X = 70.00 c.u.
```
*(소수점 3째자리 절사하여 2째자리까지: 70.00 c.u.)*

### 【제33회(2016) 문제3】 카메라 — 이윤 비율 + 연립방정식

가격별 합산:
- 1,050원: 25개
- 1,000원: 40 + 15 = 55개
- 950원: 35 + 50 = **85개** ← 최대
- 900원: 50개

→ **단위가격 P = 950원**

납세의무자 비율 (시행규칙 제7조의9 제4항: 셋째 자리 계산 후 반올림 → 둘째 자리):
```
= (판매관리비 10,000 + 이윤 5,000) / 매출액 95,000
= 15,000 / 95,000 = 15.789...% → 반올림 → 15.79%
```

세관장 동종동류 비율: 12%
120% 룰 적용: 12% × 120% = **14.40%**
→ **15.79% > 14.40% → 세관장 동종동류 비율 12% 적용** (시행령 제27조 제5항)

국내비용: 1,000원 / 100개 = 10원/개, 관세율: 10%

연립방정식:
```
X = 950 − 950×0.12 − 10 − X×0.10
X + 0.1X = 950 − 114 − 10
1.1X = 826
X = 750.909... → 750원 (소수점 절사, 문제 지시에 따름)
```

총 과세가격 = 750 × 100 = **75,000원**

### 【제36회(2019) 문제2】 반도체 소모성 자재 — 권리사용료 가산 논점 포함

*제4방법 요건 서술 (6점) + 과세가격 산정 (4점)*

단위가격 × 최대수량 총금액: 40,000,000 c.u. (부가가치세 제외)

> ⚠ 쟁점: 국내구매자가 수입자가 지정한 제3자에게 지급한 **권리사용료 10,000,000 c.u.**는 단위가격에 포함되는가?

이윤 및 일반경비 (120% 룰 적용):
- 납세의무자 25% vs 동종동류 18%
- 18% × 120% = **21.6%**
- **25% > 21.6% → 세관장 동종동류 비율 18% 적용** (시행령 제27조 제5항)

```

= 32,800,000 − 1,000,000 − X×0.25
X + X×0.25 = 31,800,000
1.25X = 31,800,000
X = 25,440,000 c.u.
```

*(실행관세 8% + 덤핑방지관세 17% = 25%를 합산하여 적용)*
