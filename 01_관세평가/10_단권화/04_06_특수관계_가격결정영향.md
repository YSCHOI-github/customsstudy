```
┌──────────────────────────────────────────────────────────────────────┐
│ 제4장 제6절  특수관계와 가격결정에의 영향                                  │
│ 출제 티어: ★★★ A (필수)                                                │
│ 출제율: 64% (7/11개년) — 2023·2024·2025 3년 연속 출제                  │
│ 핵심 법령: 관세법 제30조 제3항 제4호·단서, 시행령 제23조                  │
│ 연계 절: → 제4장 제3~5절 거래가격 배제요건 (논리 흐름 공유)               │

│           → 제8장 제2절 잠정가격 신고 (특수관계 심사 중 잠정 신고)         │
```

## [1] 핵심 요약 (3줄 이내)

- 구매자와 판매자가 **특수관계(시행령 제23조 제1항 8가지)** 에 있고, 그 특수관계가 **가격에 영향을 미친 경우** 거래가격을 과세가격으로 적용할 수 없다 (관세법 제30조 제3항 제4호).
- 단, 수입자가 **① 판매주변상황 검토 또는 ② 비교가격 입증** 중 하나를 통해 특수관계가 가격에 영향을 미치지 않음을 증명하면 제1방법 적용이 가능하다 (시행령 제23조 제2·3항).
- **출제 핵심 패턴**: ① 특수관계 8가지 유형 조문 암기, ② 판매주변상황 검토 5가지·비교가격 3가지 서술, ③ 이전가격 연구(TP Study)·APA와의 관계 논술(2023 심화 경향).

## [2] 구조화된 이론 분석

### 1. 특수관계의 범위 — 시행령 제23조 제1항 (8가지) ★★★ 반드시 암기

| 호 | 특수관계 유형 | 핵심 키워드 |
|----|------------|-----------|
| **1호** | 구매자와 판매자가 상호 **사업상의 임원 또는 관리직**에 있는 경우 | 임원·관리직 |
| **2호** | 구매자와 판매자가 상호 **법률상의 동업자**인 경우 | 동업 |
| **3호** | 구매자와 판매자가 **고용관계**에 있는 경우 | 고용 |
| **4호** | 특정인이 구매자와 판매자 **쌍방의 의결권 있는 주식을 직·간접으로 5% 이상** 소유·관리 | 제3자 5% 주식 |
| **5호** | 구매자와 판매자 중 어느 한쪽이 상대방을 **직접 또는 간접으로 지배**하는 경우 | 일방 지배 |
| **6호** | 구매자와 판매자가 **동일한 제3자에 의하여** 직접 또는 간접으로 **지배를 받는** 경우 | 공동 피지배 |
| **7호** | 구매자와 판매자가 **동일한 제3자를** 직접 또는 간접으로 **공동지배**하는 경우 | 공동지배 |
| **8호** | 구매자와 판매자가 **「국세기본법 시행령」 제1조의2제1항의 친족관계**에 있는 경우 | 친족 |

> **친족관계의 범위 (국세기본법 시행령 제1조의2제1항)**: 6촌 이내의 혈족, 4촌 이내의 인척, 배우자(사실혼 포함)
>
> ⚠️ **2025년 기출 포인트**: "6촌 혈족"은 시행령 제23조 제1항 **제8호** + 국세기본법 시행령 제1조의2제1항에 의해 특수관계 해당. "6촌 혈족 = 6호 특수관계" 혼동 주의 — 6촌 혈족은 제**8**호에 해당. 제6호는 "공동 피지배"임에 유의.

### 2. 특수관계 배제요건의 법적 구조 — 관세법 제30조 제3항 제4호

```
관세법 제30조 제3항 제4호 [본문]
"구매자와 판매자 사이의 특수관계가 해당 물품의 가격에
 영향을 미친 경우 → 거래가격을 과세가격으로 하지 아니한다"

        ↓ 단서 조항 (아래 두 경우는 배제요건에서 제외)

[단서 ①] 수입자가 비교가격으로 입증한 경우
  → "해당 물품의 거래가격이 대통령령으로 정하는 바에 따라
     특수관계로 인한 영향이 없는 가격임을 증명"
  → 시행령 제23조 제2항 제3호(가목·나목) 비교가격으로 입증

[단서 ②] 대통령령으로 정하는 경우 (판매주변상황 검토)
  → "해당 산업부문의 정상적인 가격결정 관행에 부합하는
     방법으로 결정된 경우 등 대통령령으로 정하는 경우"
  → 시행령 제23조 제2항 제1호·제2호 검토
```

> **[WTO협정 제1조 제2항]**: 구매자와 판매자가 특수관계에 있는 경우라도 (a) 판매의 주변상황 검토 및 (b) 수입자의 비교가격 입증을 통해 거래가격의 수용 가능성을 입증할 수 있다.

### 3. 특수관계 영향 판단 방법 — 2가지 경로 ★★★

#### 경로 A: 판매주변상황 검토 (시행령 제23조 제2항)
**세관장이 주도**하여 판매의 주변상황을 검토하는 방법

아래 5가지 중 하나에 해당하면 특수관계가 가격에 **영향을 미치지 않은 것**으로 봄:

| 번호 | 내용 | 근거 |
|------|------|------|
| **① 독립거래 통상 방법 일치** | 가격이 특수관계 없는 구매자와 판매자 간에 **통상적으로 이루어지는 가격결정방법**으로 결정 | 시행령 제23조 제2항 제1호 |
| **② 정상적 가격결정 관행 부합** | 가격이 해당 산업부문의 **정상적인 가격결정 관행**에 부합하는 방법으로 결정 | 시행령 제23조 제2항 제2호 |
| **③ 비특수관계자 판매가격과 비교** | 판매자가 **특수관계 없는 구매자**에게 판매하는 동종·동질·유사물품 가격과 비교하여 **충분한 이유** 있음 | 관세평가 운영에 관한 고시 제28조 |
| **④ 동일 판매자의 다른 물품 가격과 비교** | 구매자가 **동일한 판매자**로부터 구매하는 동종·동질·유사물품 가격과 비교하여 **충분한 이유** 있음 | 관세평가 운영에 관한 고시 제28조 |
| **⑤ 생산비용 + 이윤 충당** | 해당 물품 가격이 **모든 생산비용 및 이윤**을 충당하기에 충분한 수준 | 관세평가 운영에 관한 고시 제28조 |

> ⚠️ **5가지 암기 구조**: 시행령 §23②의 2가지(①②) + 고시 제28조의 3가지(③④⑤)

---

#### 경로 B: 비교가격에 의한 입증 (시행령 제23조 제2항 제3호)
**수입자(구매자)가 주도**하여 아래 비교가격 중 하나에 근접함을 입증하는 방법

| 번호 | 비교가격의 종류 | 법적 근거 |
|------|-------------|---------|
| **① 독립거래 거래가격** | 특수관계 없는 구매자에게 수출 판매되는 동종·동질·유사물품의 **거래가격** | 시행령 제23조 제2항 제3호 **가목** |
| **② 제4방법에 의한 가격** | 제33조(국내판매가격)에 따라 결정된 **과세가격** | 시행령 제23조 제2항 제3호 **나목** (§33) |
| **③ 제5방법에 의한 가격** | 제34조(산정가격)에 따라 결정된 **과세가격** | 시행령 제23조 제2항 제3호 **나목** (§34) |

> **비교가격 기준시점 (시행규칙 제5조 제3항)**: 제1방법(선적 시점), 제4방법(국내판매 시점), 제5방법(수입신고 시점) 기준으로 비교.

> ⚠️ **비교가격은 정확한 일치 불요**: 비교가격에 "**매우 근접(closely approximate)**"함을 증명하면 족함. 정확히 동일할 필요 없으며, 수량·거래단계 차이 등은 **적절히 조정** 후 비교.

### 4. 판매주변상황 검토 vs 비교가격 입증 비교

| 구분 | 판매주변상황 검토 (경로 A) | 비교가격 입증 (경로 B) |
|------|------------------------|--------------------|
| **주도자** | 세관장 | 수입자(구매자) |
| **방법** | 가격결정 방식·수준 종합 검토 | 구체적 비교가격에 근접 증명 |
| **근거 조문** | 시행령 제23조 제2항 제1호·제2호 | 시행령 제23조 제2항 제3호 (가목·나목) |
| **WTO 근거** | 제1조 제2항(a) | 제1조 제2항(b) |
| **실무 역할** | 세관이 가격 수용 여부 결정 | 수입자가 소명 자료로 제출 |
| **결론** | 2가지(시행령) + 3가지(고시) 중 하나 충족 → 영향 없음 | 3가지 비교가격에 근접 → 영향 없음 |

### 5. 이전가격 연구(TP Study)와의 관계 — 2023년 심화 경향 ★★★

> **[WTO 관세평가협정 예해 23.1]** 은 2023년 기출에서 지문으로 직접 인용됨.

#### (1) 이전가격 연구의 활용 가능성

- **OECD 이전가격 지침(Transfer Pricing Guidelines)** 에 따라 작성된 이전가격 연구(TP Study)는 WTO 관세평가협정 제1조 제2항(a)에 따른 **"판매 주변상황" 검토의 기초자료**로 활용 가능
- 수입자가 TP Study를 제출하면 세관은 이를 참고하여 판매주변상황을 검토할 수 있음
- **단, TP Study 자체가 과세가격을 결정하는 것은 아님** (기초자료에 불과)

#### (2) WTO 관세평가 vs OECD 이전가격 지침의 차이

| 구분 | WTO 관세평가협정 | OECD 이전가격 지침 |
|------|--------------|-----------------|
| **목적** | 수입물품의 **과세가격** 결정 | 국제거래 소득에 대한 **이전가격** 결정 |
| **기준** | 수입 시 물품의 **거래가격**(실제지급가격) | 정상가격원칙(Arm's Length Principle) |
| **시점** | **수입 시점** 기준 | 과세연도 기준 |
| **가격 방향** | 일방적 (수출국 → 수입국) | 양방향 (국제거래 전반) |
| **조정 방법** | 협정 제8조 가산요소 | OECD 5가지 이전가격 결정 방법 |
| **관계** | 서로 다른 목적, 결과가 다를 수 있음 | TP 연구가 관세평가 참고자료로 활용 가능 |

> ⚠️ **핵심**: TP Study가 정상가격으로 인정받았더라도 그것이 자동으로 관세평가상 과세가격이 되지는 않는다. 관세평가는 별도의 요건과 방법으로 독립적으로 결정된다.

---

### 6. APA(사전가격승인)와 관세평가

- **APA(Advance Pricing Agreement/사전가격승인협약)**: 납세의무자와 과세당국이 이전가격 결정방법을 사전에 합의하는 제도 (국조법상 제도)
- **관세평가와의 관계**: APA에 의해 정상가격으로 승인된 가격이라도 관세평가 목적상 거래가격으로 **자동 인정되지 않음**
- 관세당국은 APA를 **판매주변상황 검토의 참고자료**로 활용할 수 있으나, 독립적으로 관세평가를 수행해야 함

> **2016년 기출 연계**: B사가 미국·한국 내국세당국으로부터 쌍방 APA 승인을 받았더라도, 세관장은 독립적으로 특수관계가 가격에 영향을 미쳤는지를 검토해야 한다.

## [3] 판단 흐름도

```
  [STEP 1] 구매자와 판매자 사이에 시행령 제23조 제1항의
           특수관계 8가지 중 하나가 해당하는가?
       ↓ 없음 → 특수관계 없음 → 배제요건 해당 없음 → 제1방법 적용
       ↓ 있음 (특수관계 성립)
  [STEP 2] 특수관계가 해당 물품의 가격에 영향을 미쳤는가?
       ↓ 영향 없음 → 제1방법 적용 가능
       ↓ 영향 있음 (또는 의심 있음)
  [STEP 3-A] 세관장: 판매주변상황 검토 (시행령 제23조 제2항 제1호·제2호 + 고시 제28조)
       5가지 기준 중 하나 충족?
       ↓ 충족 → 가격 영향 없음 → 제1방법 적용 가능
       ↓ 미충족
  [STEP 3-B] 수입자: 비교가격 입증 (시행령 제23조 제2항 제3호)
       3가지 비교가격 중 하나에 근접함을 입증?
       ↓ 입증 성공 → 가격 영향 없음 → 제1방법 적용 가능
       ↓ 입증 실패
  [최종] 관세법 제30조 제3항 제4호 배제요건 해당
       → 제1방법 배제 → 제31조(제2방법)부터 순차 적용
```

## [4] 특수관계 8가지 암기 도표

```
  특수관계 (시행령 제23조 제1항)
  │
  ├─ [조직·역할 관계]
  │    ├─ 1호: 상호 임원·관리직
  │    ├─ 2호: 상호 동업자
  │    └─ 3호: 고용관계
  │
  ├─ [지분·지배 관계]
  │    ├─ 4호: 제3자가 쌍방 5% 이상 주식 소유
  │    ├─ 5호: 일방이 상대방을 직·간접 지배
  │    ├─ 6호: 동일 제3자에 의한 공동 피지배 (지배 받음)
  │    └─ 7호: 동일 제3자를 공동지배 (지배 함)
  │
  └─ [신분 관계]
       └─ 8호: 친족관계 (6촌 혈족·4촌 인척·배우자)
```
