# 관세평가 단권화 교과서

---

```
┌──────────────────────────────────────────────────────────────────────┐
│ 제6장 제4절  산정가격을 기초로 한 과세가격 결정방법 (제5방법)               │
│ 출제 티어: ★ C  (2019·2020년 10점 독립 문제, 2021년 단편 언급)           │
│ 출제율: 약 27% (제36회·제37회·제38회 확인)                               │
│ 핵심 법령: 관세법 제34조, 시행령 제28조                                   │
│ 연계 절: → 제6장 제3절 [제4방법, 국내판매가격] (순서 전환 관계)             │
│          → 제5장 제5절 [생산지원비용] (산정가격 구성요소 포함 여부 교차)     │
│          → 제5장 제6절 [권리사용료] (제34조 제1호 포함 여부 판단)           │
│          → 제6장 제5절 [제6방법] (제5방법 불가 시 다음 순위)               │
└──────────────────────────────────────────────────────────────────────┘
```

---

## [1] 핵심 요약 (3줄 이내)

- **산정가격(제5방법)**은 수출국 생산자의 생산원가에 **이윤·일반경비**와 **수입항까지의 운임·보험료**를 합산하여 과세가격을 결정하는 방법이다. (관세법 제34조)
- 생산자가 자료 제공을 **거부하거나 자료 입수가 불가능**한 경우 적용할 수 없으며, 납세의무자의 신청이 있으면 제4방법보다 먼저 적용할 수 있다. (관세법 제33조 제1항 단서)
- 시험 핵심: ① 산정가격 3대 구성요소 암기, ② **수입국 개발 생산지원요소의 제외** 처리, ③ 이윤·일반경비 결정 기준(동종·동류 기준)이 반복 출제된다.

---

## [2] 구조화된 이론 분석

### 2-1. 법적 근거 및 의의

1. **법적 근거**: 관세법 제34조
   - 제1~4방법으로 과세가격을 결정할 수 없는 경우 적용 (제5순위)
   - 예외적으로 납세의무자의 요청 시 제4방법보다 먼저 적용 가능

2. **WTO 관세평가협정 근거**: 제6조 (Computed Value)
   - [WTO협정 제6조 제1항]: 산정가격은 다음의 합계액을 기초로 결정한다:
     - (a) 수입물품 생산에 사용된 재료비 및 제조·가공비
     - (b) 수입국에 수출하기 위해 판매하는 동종·동류 물품 생산자가 통상 반영하는 이윤 및 일반경비에 상당하는 금액
     - (c) 제8조 제2항에 따른 운임·보험료 등

3. **적용상 한계 (이론적 특성)**
   - 수출국 **생산자의 회계 자료** 접근을 전제로 하는 방법
   - 생산자가 수입국 세관에 자료 제공 의무를 부담하지 않으므로, 실무상 적용이 어려움
   - 생산자가 협조하는 경우에만 실질적 적용 가능

---

### 2-2. 적용 요건

1. **전제 조건**: 제1방법 ~ 제4방법으로 과세가격 결정 불가
2. **자료 입수 가능성**: 수출국 생산자로부터 생산원가 등 관련 자료 제공 또는 입수 가능
3. **순서 전환 요건** (관세법 제33조 제1항 단서):
   - 납세의무자(수입자)가 세관장에게 요청하여 **제4방법보다 제5방법을 먼저 적용**받을 수 있음
   - 승인 시: 제1방법 → 제2방법 → 제3방법 → **제5방법(산정가격)** → 제4방법(국내판매가격) → 제6방법 순으로 적용
   - 신청 이유: 수입자가 생산자와 특수관계에 있거나 긴밀한 협력관계에 있어 생산원가 자료 입수가 용이한 경우

---

### 2-3. 산정가격의 3대 구성요소 (관세법 제34조)

```
┌────────────────────────────────────────────────────────────────┐
│           산정가격 = ① + ② + ③                                  │
│                                                                │
│  ① 생산비용 (원자재비 + 제조비용 + 기타)      관세법 제34조 제1호   │
│  ② 이윤 및 일반경비 (동종·동류 기준)          관세법 제34조 제2호   │
│  ③ 수입항까지의 운임·보험료 등                관세법 제34조 제3호   │
└────────────────────────────────────────────────────────────────┘
```

---

### 2-4. ① 생산비용 (관세법 제34조 제1호, 시행령 제28조 제1항)

**포함 항목:**

| 항목 | 내용 | 주의사항 |
|------|------|---------|
| 재료비 | 수입물품 생산에 사용된 원자재, 부품, 조립용 재료비 | 손모분 포함 |
| 제조비 | 직접 인건비, 조립비, 기타 가공비 | 직접 제조 관련 비용 전부 |
| 간접비 | 생산 감독, 설비 유지, 시간외 근무 등 간접 제조원가 | 통상적 간접비 인정 |
| 용기·포장비용 | 관세법 제30조 제1항 제2호 해당 금액 | 시행령 제28조 제2항에 명시된 가산항목 |
| 권리사용료 | 생산자의 회계장부에 제조원가로 반영된 경우에 한해 포함 | 제1방법과 달리 법 제30조 제1항 제4호를 직접 가산하는 구조 아님 |
| 생산지원 해당분 | 구매자가 생산자에게 무상 또는 인하 가격으로 제공한 물품·용역 가치 (시행령 제18조) | ※ 단, 수입국(한국) 개발분 제외 (시행령 제18조 제4호) |

**⚠ 핵심 예외 — 수입국에서 개발된 생산지원요소는 제외**

- [WTO협정 제8조 제1항 (b)(iv) 주해]: *"수입국에서 개발된 엔지니어링, 개발, 예술 및 디자인 작업의 가치는 관세가격에 포함될 필요가 없다"*
- **관세법 시행령 제18조 제4호**: 구매자가 수입국에서 개발한 디자인·설계·설계도 등은 생산지원비용에서 제외
- → 산정가격 계산 시 **구매자가 수입국에서 개발하여 무상 제공한 설계도·기술·디자인 비용은 생산비용에서 제외**
- ⚠ **단, 생산자 스스로 그 비용을 부담하여 원가에 반영한 경우에는 포함** (시행령 제28조 제2항)

```
예시:
  한국 수입자(A)가 한국에서 개발한 설계도를 해외 생산자(B)에게 제공
  → 설계도 개발비용은 산정가격(제34조 제1호) 구성에서 제외
  (이미 수입국에서 부담한 비용이므로 이중 계산 방지)
```

### 2-5. ② 이윤 및 일반경비 (관세법 제34조 제2호, 시행령 제28조 제2항)

1. **기준**: 수입국에 수출하기 위해 판매하는 **동종·동류 물품(goods of the same class or kind)** 생산자가 통상적으로 반영하는 금액
   - [WTO협정 제6조 제1항 (b)]: 동종·동류는 수출국 내에서 해당 물품과 동일한 산업부문의 생산자가 수입국에 수출판매할 때 반영하는 이윤·경비 기준

2. **결정 방법**:

| 우선 순위 | 방법 | 요건 |
|---------|------|------|
| 1순위 | 납세의무자가 제공한 **자료**에 기초한 생산자의 이윤·경비 | 동종·동류 생산자의 통상적 수치와 **일치**하는 경우 |
| 2순위 | 세관장이 산출한 동종·동류 **통상 비율** | 납세의무자 제공 자료와 일치하지 않거나 자료 없는 경우 |

> **핵심 판단**: 납세의무자 제출 비율이 **"당해 산업부문에서 통상적으로 반영되는 수치와 일치됨이 확인"**되면 납세의무자 자료 우선 적용

3. **동종·동류물품(same class or kind)의 개념**:
   - [WTO협정 제15조 제1항 (d)]: 동종·동류물품은 해당 물품이 속하는 그룹 또는 범위의 물품으로서, 동일 산업부문에서 생산된 것
   - 수출국과 수입국 모두에서 제조하는 생산자 포함
   - ⚠ **제2·3방법의 "동종·동질물품·유사물품"과는 다른 개념** → 더 넓은 범위

---

### 2-6. ③ 수입항까지의 운임·보험료 (관세법 제34조 제3호)

- **내용**: 수출항에서 수입항까지의 운임·보험료 및 관련 비용
- **적용 기준**: 관세법 제30조 제1항 제6호의 운임·보험료 규정과 동일하게 적용 (시행령 제28조 준용)
- → 상세 내용은 **제5장 제9절 [운임·보험료] 참조**

---

### 2-7. 비교: 제4방법 vs 제5방법

| 구분 | 제4방법 (국내판매가격) | 제5방법 (산정가격) |
|------|---------------------|-----------------|
| 법적 근거 | 관세법 제33조 | 관세법 제34조 |
| WTO협정 | 제5조 (Deductive Value) | 제6조 (Computed Value) |
| 출발점 | **수입국 국내판매가격**에서 역산 | **수출국 생산원가**에서 합산 |
| 자료 출처 | 수입국 내 판매 자료 (입수 용이) | 수출국 생산자 원가 자료 (입수 어려움) |
| 적용 순서 | 원칙적으로 제5방법 앞 (제4순위) | 원칙적으로 제5방법 (제5순위) |
| 순서 전환 | 납세의무자 요청 시 제5방법 먼저 가능 | 동일 |
| 실무 빈도 | 상대적으로 적용 多 | 생산자 비협조 시 적용 불가 |

## [3] 판단 흐름도

```
[제5방법 적용 가능 여부 판단]
          │
          ▼
[STEP 1] 제1~3방법으로 과세가격 결정 가능한가?
          │ YES → 해당 방법 적용, 제5방법 불필요
          │ NO
          ▼
[STEP 2] 납세의무자가 제4방법보다 제5방법을 먼저 적용하도록
          세관장에게 요청하였는가?
          │ YES → [바로 STEP 4로]
          │ NO
          ▼
[STEP 3] 제4방법(국내판매가격)으로 과세가격 결정 가능한가?
          │ YES → 제4방법 적용
          │ NO
          ▼
[STEP 4] 수출국 생산자로부터 생산원가 등 필요 자료를 입수할 수 있는가?
          │ YES → 제5방법 적용 진행
          │ NO  → 제6방법(합리적 기준) 적용
          ▼
[STEP 5] 산정가격 3대 구성요소 산출
          │
          ├─ ① 생산비용 = 재료비 + 제조비 + 간접비 + 권리사용료
          │              + 생산지원 해당분 (단, 수입국 개발분 제외)
          │
          ├─ ② 이윤·일반경비 = 동종·동류 통상 비율 기준
          │              (납세의무자 자료 일치 시 우선, 불일치 시 세관장 산정)
          │
          └─ ③ 수입항까지 운임·보험료
          ▼
[STEP 6] 산정가격 = ① + ② + ③ → 과세가격 확정
```

## [4] 계산 공식 및 실수 포인트

### ■ 산정가격 계산 공식

```
┌──────────────────────────────────────────────────────────────────┐
│  산정가격 =                                                        │
│    [재료비 + 직접인건비 + 조립비 + 간접제조비]                         │
│    + [구매자 제공 생산지원 가치 (수입국 개발·무상제공분 제외)]              │
│    + [용기·포장비용 (법 제30조 제1항 제2호)]                            │
│    + [권리사용료 (생산자 장부상 제조원가 반영분에 한함)]                   │
│    + [동종·동류 기준 이윤 및 일반경비]                                 │
│    + [수입항까지의 운임·보험료]                                        │
└──────────────────────────────────────────────────────────────────┘
```

### ■ 2020년 기출 계산 구조 (제37회 문제5)

**주어진 자료:**

| 항목 | 금액 | 포함 여부 |
|------|------|---------|
| 원자재 비용 | $1,000,000 | ✅ 포함 (재료비) |
| 직접 인건비 | $700,000 | ✅ 포함 (제조비) |
| 조립비용 | $300,000 | ✅ 포함 (제조비) |
| 간접비용 (생산 감독, 설비 유지 등) | $200,000 | ✅ 포함 (간접 제조원가) |
| 설계도 구입비용 (한국에서 개발, **A사가 B사에게 무상 제공**) | $50,000 | ❌ **제외** (구매자 수입국 개발·무상제공 생산지원요소 → 시행령 제18조 제4호) |
| A사가 B사에게 지급하는 권리사용료 | $20,000 | ✅ 포함 (생산자 회계장부에 제조원가로 반영된 전제) |
| 수입항까지 운임·보험료 | $30,000 | ✅ 포함 (제34조 제3호) |
| 이윤 및 일반경비 | $100,000 | ✅ 포함 (동종·동류 통상 수치와 일치 확인) |

**산정가격 계산:**
```
  생산비용:  $1,000,000 + $700,000 + $300,000 + $200,000 + $20,000
           = $2,220,000

  이윤·일반경비: $100,000

  운임·보험료: $30,000

  산정가격 = $2,220,000 + $100,000 + $30,000 = $2,350,000
```

---

### ■ 계산 절차

- **STEP 1.** 주어진 항목을 ① 생산비용 / ② 이윤·일반경비 / ③ 운임·보험료 / ④ 제외 항목으로 분류
- **STEP 2.** **수입국 개발 항목**(설계도, 디자인 등)은 제외 처리, 그 근거 명시
- **STEP 3.** 이윤·일반경비: 납세의무자 자료가 동종·동류 통상 수치와 일치하는지 확인 후 적용
- **STEP 4.** ① + ② + ③ 합산하여 산정가격 결정

---

### ■ ⚠ 실수하기 쉬운 포인트 체크리스트

- □ **구매자가 수입국(한국)에서 개발하여 무상 제공한 설계도·디자인 비용**은 생산비용에 포함하지 말 것 → 시행령 제18조 제4호
  (단, 생산자가 스스로 부담한 수입국 개발 기술비용은 포함됨 — 시행령 제28조 제2항)
- □ **이윤·일반경비**를 "납세의무자 회계 자료"와 "세관장 산정 동종·동류 비율" 중 어느 것을 적용할지 **근거 명시** 필수
- □ **동종·동류물품** 개념을 제2·3방법의 "동종·동질물품·유사물품"과 혼동하지 말 것 → 더 넓은 산업부문 범위
- □ 제4방법보다 제5방법을 먼저 적용하려면 **"납세의무자의 요청"** 으로 충분 (세관장 별도 승인 불필요 — 관세법 제33조 제1항 단서)
- □ 생산자가 **자료 제공을 거부**하면 제5방법 적용 불가 → 제6방법으로 넘어감을 명시
- □ 권리사용료는 **물품관련성·거래조건성** 충족 여부를 먼저 확인 후 포함 여부 결정

- WTO협정 제6조·제8조 주해의 직접 인용 요구 가능 (최근 심화 경향)
