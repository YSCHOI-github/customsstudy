# 제1장. 관세평가 과목 전체 구조

## 1-1. 과목 성격 및 학문적 위상 ✅ [공통합의]

4인 전문가 모두 관세평가를 "**관세법 + WTO 협정 + 실무 고시**가 유기적으로 결합된 법·실무 융합 과목"으로 정의한다.

- **본질**: 수입물품에 관세를 부과하기 위한 **과세가격(과세표준)**을 결정하는 법·경제적 프로세스
- **공식**: `관세액 = 과세가격 × 관세율` → 관세평가는 좌측 변수(과세가격) 결정 담당 *(NotebookLM)*
- **국제적 정합성**: WTO 관세평가협정(GATT VII 이행협정) 준수 — 협정문 주해(Notes)·예해(Explanatory Notes)·권고의견(Advisory Opinions)·사례연구(Case Studies) 학습 필수 *(NotebookLM 강조)*

## 1-2. 시험 기본 정보 ✅ [공통합의]

| 항목 | 내용 |
|---|---|
| 시험 유형 | 주관식 논술형 |
| 문항 수 | 4문제 |
| 시험시간 | 80분 |
| 배점 구조 | **30점 × 2문항 + 20점 × 2문항 = 100점** |
| 답안지 분량 | A4사이즈 22행 × 16면 |
| 과락 기준 | 40점 미만 |

**⚠️ [상이의견] — 답안 분량 가이드**
- **Perplexity**: 1점당 3~5줄, 30점 문제 4~5페이지, 20점 문제 2~3페이지
- **Claude**: 합격자 평균 14~20쪽 작성
- **NotebookLM·GPT**: 구체 분량 가이드 미제시
- 👉 **권고**: Perplexity의 "1점당 3~5줄" 기준이 가장 실용적 가이드 — 30점 = 15~25줄, 20점 = 10~15줄 확보

## 1-3. 법령 3단 체계 ✅ [공통합의]

```
┌─────────────────────────────────────────────┐
│  1단계. 국제규범                              │
│  WTO 관세평가협정 (GATT VII 이행협정)           │
│  - 협정문, 주해, 예해, 권고의견, 사례연구         │
└─────────────────────────────────────────────┘
                     ↓ 국내 수용
┌─────────────────────────────────────────────┐
│  2단계. 국내법                                │
│  관세법 제30조~제35조 (6가지 평가방법)            │
│  + 관세법 시행령·시행규칙                       │
└─────────────────────────────────────────────┘
                     ↓ 행정 운영
┌─────────────────────────────────────────────┐
│  3단계. 행정규칙                              │
│  관세평가 운영에 관한 고시 (2021 전면개정)         │
│  + 관세청 예규·질의회신                         │
│  + 조세심판원 심판례·법원 판례                   │
└─────────────────────────────────────────────┘
```

**📌 [중요 배경]**: 2020~2021년 개정으로 과거 『수입물품 과세가격 결정에 관한 고시』의 실체 규정이 **시행령·시행규칙으로 상향 입법**되고, 『**관세평가 운영에 관한 고시**』가 새로 제정되어 행정 절차를 정리했다. → **조세법률주의·위임입법 한계 대응** *(Claude·Perplexity 공통 지적)*

## 1-4. 과세가격 결정의 계층적 메커니즘 ✅ [공통합의]

관세법은 **6가지 평가방법의 순차적 적용 원칙**을 규정한다. 제1방법 불가 시 제2→제3→…→제6방법으로 진행.

| 방법 | 근거조문 | 핵심 원리 |
|---|---|---|
| **제1방법** (거래가격) | 관세법 §30 | 구매자가 **실제로 지급했거나 지급할 가격** + 가산요소 |
| **제2방법** (동종·동질물품) | 관세법 §31 | 기능·품질·명성이 동일한 물품의 거래가격 기초 |
| **제3방법** (유사물품) | 관세법 §32 | 기능·성분이 유사한 물품의 거래가격 기초 |
| **제4방법** (국내판매가격) | 관세법 §33 | 국내판매가격에서 이윤·경비·세금 **역산(공제)** |
| **제5방법** (산정가격) | 관세법 §34 | 제조원가 + 통상이윤·일반경비 **가산** |
| **제6방법** (합리적 기준) | 관세법 §35 | 1~5방법 불가 시 **신축적** 적용 |

**📌 [Claude 단독주장]**: 제4·5방법은 수입자 신청 시 **순서 교환**이 가능(시행령 제27조). 이 '**신청권**'의 존재 자체를 모르는 수험생이 많음 → 출제 가능성 높은 단서.

# 제3장. 실무 미경험·비전공자 취약점 분석

## 3-1. 4인 전문가 공통 지적 취약점 ✅ [공통합의]

**① 거래가격의 법적 개념 혼동**
- 일반인이 생각하는 '구매가격 = 인보이스 금액'과 관세법상 거래가격의 차이
- **함정**: FOB vs CIF 조건별 가산항목 차이, 미래 지급분도 '지급할 가격'에 포함 *(Claude·GPT)*

**② 권리사용료 2요건(관련성 + 거래조건성) 판단 미숙**
- 관련성: 물품에 상표·특허 등이 **체화(體化)**되어 있는지
- 거래조건성: 권리사용료 미지급 시 **구매 자체가 불가능한가** — 구매자의 선택권 여부가 핵심
- 2021년 고시 개정으로 거래조건성 인정 예시 구체화됨 *(Claude·NotebookLM 강조)*

**③ 특수관계 2단계 판단 혼동**
- `특수관계 존재 → 즉시 제1방법 배제`로 오해하는 경우 다수
- 정답: **특수관계 인정(1단계)** → **가격영향 여부 판단(2단계)** → 배제 결정
- **합리적 의심(Reasonable Doubt)** 구조 이해 필수 *(Claude 상세 도식)*

**④ 간접지급액 개념 불명확** *(Perplexity·NotebookLM·Claude)*
- 구매자가 판매자에게 **직접 지급하지 않는** 광고비·판촉비·장려금 등의 과세 포함 범위
- 판매자 부채 대위변제, 제3자 대납 등 이면 거래 파악 능력 요구

**⑤ 회계·원가 개념 부담** *(Perplexity·NotebookLM)*
- 제4방법(국내판매가격) 이윤·일반경비 산출
- 제5방법(산정가격) 제조원가 + 통상이윤·일반경비 합산
- 재무제표·원가표 독해 능력 필요

## 3-2. 전문가별 독자 지적 취약점 ⚠️ [상이의견]

| 전문가 | 추가 지적 취약점 |
|---|---|
| **NotebookLM** | 수출판매 정의 범위 혼란 (보세구역 전매, 무상·임차 수입 등 특수상황), '금액으로 계산할 수 없는 조건·사정'의 추상성 |
| **Claude** | 제4방법 **이윤·경비 기준비율** 존재 자체 모름, 4·5방법 **순서 교환 신청권** 존재 자체 모름, 사전심사(ACV) vs 특수관계 사전심사(ACVA) 구별 혼동 |

## 3-3. 답안 작성 서술 능력 취약점 ✅ [공통합의]

# 제7장. 비전공자 맞춤 학습 전략

## 7-1. 학습 원칙 ✅ [공통합의]

관세평가는 **"암기 과목이 아닌 판단 과목"**. 단순 법조문 암기로는 합격 불가 — **판단 근거를 논리적으로 서술**하는 훈련이 핵심.

## 7-2. 3단계 학습 프로세스 ✅ [공통합의 종합]

### 1단계: 이론 정립 & 취지 파악 (1~2개월)
- 관세법 제30조~제35조 **원문 정독**
- WTO 협정 구조 파악 (6방법 체계)
- 법령 용어 정의 숙지
- **취지 이해**: 왜 이 규정이 만들어졌는가? 예) 생산지원 가산 = 구매자가 원재료 무상공급했다면 그만큼 물품가격이 낮아졌을 것이므로 *(NotebookLM)*

### 2단계: 사례 분석 & 법리 접목 (2~3개월)
- 기출문제·WTO 권고의견·조세심판원 심판례 분석
- **쟁점 → 조문 역방향 접근** *(Perplexity 강조)*: 사례→조문 순으로 접근하면 비전공자가 이해 용이
- **쟁점별 서브노트** 작성 (권리사용료 노트, 연속거래 노트, 특수관계 노트 등)

### 3단계: 답안 현출 & 기술 숙달 (2~3개월)
- **3단 답안구조** 반복 훈련: 쟁점파악 → 법적근거 → 적용·판단
- 손으로 직접 답안 작성 (타이핑 X, 시간 측정 포함)
- 모의고사 후 첨삭 → **개인별 함정 체크리스트** 작성

## 7-3. 비전공자 특화 팁 ⚠️ [전문가별 차이]

### 공통 팁 ✅

- **무역거래 흐름 도식화**: 수출자 → 선적 → 수입자 → 세관신고 → 과세가격결정
- **인코텀즈(FOB/CIF) 기본 개념**: 운임·보험료 포함여부가 가산요소 계산에 직결
- **거래관계도 그리기**: 구매자·판매자·제조자·대리인 간 물품·자금 흐름을 화살표 표시 *(NotebookLM·Claude)*

# 🎯 최종 통합 조언

> **관세평가는 '아는 것을 쓰는 시험'이 아니라 '묻는 것에 정확히 답하는 시험'이다.** *(Claude)*
>
> 출제자의 의도를 파악하고, **법령 근거 → 판단 기준 → 사례 적용 → 결론**의 4단계 답안 구조를 숙달하면, 비전공자도 충분히 합격선을 넘길 수 있다.

**4인 전문가 공통 메시지 종합**:
1. **제1방법에 집중하라** — 배점 78.5%, 학습시간 60~70% 투입
2. **권리사용료·특수관계·생산지원을 마스터하라** — 매년 출제 확실
3. **2024년 이후 사례판단형 대비하라** — 단순암기는 불가
4. **답안은 3단 구조로 작성하라** — 쟁점→근거→판단
5. **법조문 번호를 정확히 인용하라** — 채점위원 감점 1순위
6. **2021년 고시·2025년 세제개편을 체크하라** — 개정법령은 필출
7. **사례→조문 역방향 접근하라** *(비전공자 특화)* — 실무 그림 먼저
8. **시각화하라** — 거래관계도·체계도·이미지화
