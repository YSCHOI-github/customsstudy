## 1. 무역의 정의와 특성

**무역(Trade)**: 국가와 국가 간에 재화·서비스·기술 등을 교환하는 경제 행위

| 구분 | 국내 거래 | 무역 (국제 거래) |
|---|---|---|
| **언어·관습** | 동일 | 상이 → 오해·분쟁 위험 |
| **통화** | 단일 | 복수 → 환위험(Exchange Risk) 발생 |
| **법률** | 국내법 단일 | 복수 법체계 → 준거법 지정 필요 |
| **결제** | 즉시 | 장기·원격 → 신용장(L/C) 등 필요 |
| **운송** | 단거리 | 장거리 → 해상·항공운송·보험 필요 |
| **위험** | 낮음 | 비상위험(정치·전쟁) + 신용위험 추가 |

---

## 2. 무역의 종류

### (1) 형태별 분류

| 종류 | 내용 |
|---|---|
| **유형무역** | 물품(Goods) 수출입 — 가장 일반적 |
| **무형무역** | 용역·서비스·기술·지식재산권 거래 |

### (2) 결제 방식별 분류

| 종류 | 내용 |
|---|---|
| **유환무역(有換)** | 대금 결제가 수반되는 일반적 무역 |
| **무환무역(無換)** | 대금 결제 없이 물품만 이동 (증여·견본·위탁판매 등) |

### (3) 연계무역 (Counter Trade)

수출과 수입을 **하나의 계약으로 연계**하는 방식:

| 유형 | 내용 |
|---|---|
| **구상무역(Barter)** | 물물교환 — 대금 없이 물품 ↔ 물품 |
| **대응구매(Counter Purchase)** | 수출 후 상대국으로부터 일정 비율 수입 약정 |
| **환매(Buy-back)** | 플랜트·기계 수출 후 생산된 제품으로 대금 회수 |
| **상계무역(Offset)** | 방산·항공 분야에서 기술이전·부품조달 조건부 수출 |

### (4) 가공무역

| 유형 | 내용 |
|---|---|
| **위탁가공무역** | 원료를 외국으로 보내 가공 후 수입 (임가공비 지급) |
| **수탁가공무역** | 외국에서 원료를 받아 가공 후 수출 (임가공비 수취) |

### (5) 지역별 분류

| 종류 | 내용 |
|---|---|
| **중계무역(Intermediary Trade)** | 외국 물품을 수입 후 원형 그대로 제3국에 재수출 |
| **통과무역(Transit Trade)** | 물품이 제3국을 단순 경유 |
| **중개무역(Merchandising Trade)** | 제3국에 소재하면서 양국 간 거래를 주선 (물품 직접 소유 없음) |

> ⚠️ **혼동 주의**:
> - **중계무역** = 물품 소유권 취득 후 재수출
> - **중개무역** = 소유권 취득 없이 거래 주선만
> - **통과무역** = 단순 경유, 거래 당사자 아님

## 3. 무역거래의 절차 (수출 플로우)

```
① 해외시장조사 → ② 거래처 발굴(신용조사) → ③ 거래 제의(Offer)
       ↓
④ 무역계약 체결 (품질·수량·가격·선적·결제·보험 조건)
       ↓
⑤ 수출승인 (해당 시) → ⑥ 신용장 수취 (L/C 방식)
       ↓
⑦ 원자재 조달·생산 → ⑧ 수출통관·선적
       ↓
⑨ 선하증권(B/L) 수취 → ⑩ 선적서류 제출(네고)
       ↓
⑪ 대금 회수 → ⑫ 사후관리(클레임 대응)
```

**수입 플로우 (역방향 요약)**
1. 수입계획 수립 → 2. 수입승인 → 3. L/C 개설 → 4. 선적서류 수취
5. 수입통관 → 6. 화물 인수 → 7. 대금 결제

## 4. 무역거래의 주체와 거래형태

### (1) 무역업자 유형

| 유형 | 내용 | 특징 |
|---|---|---|
| **직접 무역업자** | 자기 명의·자기 계산으로 수출입 | 이익·위험 직접 부담 |
| **무역대리업자** | 본인(Principal)을 위해 대리·중개 | 수수료(Commission) 수취 |
| **종합상사** | 광범위한 품목을 대규모로 취급 | 금융·정보·물류 기능 포함 |

### (2) 수출입 대행

| 유형 | 내용 |
|---|---|
| **수출대행** | 수출능력 없는 제조업자 대신 무역업자가 수출 대행 (위탁자 명의 또는 대행업자 명의) |
| **수입대행** | 수입자를 위해 무역업자가 수입 대행 |

### (3) 대리상(Agency)의 종류

| 유형 | 내용 |
|---|---|
| **판매대리상(Sales Agent)** | 본인 명의로 거래 체결, 대금은 본인에게 직접 지급 |
| **구매대리상(Buying Agent)** | 수입자를 위해 물품 구매 대행 |
| **독점대리상(Sole/Exclusive Agent)** | 특정 지역 내 독점 판매권 보유 |
| **중개인(Broker)** | 매도인·매수인 연결만 하고 계약 당사자 아님 |

---

## 5. 핵심 무역 용어 정리

| 용어 | 정의 |
|---|---|
| **Offer (청약)** | 계약 성립을 위한 일방의 의사 표시 (확정적·구체적·도달 시 효력) |
| **Acceptance (승낙)** | 청약의 조건을 무조건 수락하는 의사 표시 |
| **L/C (신용장)** | 발행은행이 수익자에게 지급을 확약하는 조건부 지급 확약서 |
| **B/L (선하증권)** | 선박회사가 화물 수취를 증명하고 운송을 약정하는 유가증권 |
| **Invoice (송장)** | 매도인이 발행하는 물품 명세·금액 기재 서류 |
| **Packing List** | 포장 명세서 (수량·중량·용적 등) |
| **C/O (원산지증명서)** | 물품의 원산지를 증명하는 공식 서류 |
