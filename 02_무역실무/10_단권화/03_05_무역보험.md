## 1. 무역보험의 의의와 법적 근거

### 1) 무역보험의 정의 (무역보험법 제2조 제1호)

무역보험이란 **수출·수입·해외투자·해외자원개발** 등 대외거래와 관련하여 발생하는 위험 중 **일반 보험회사가 인수하기 어려운 비상위험(Political Risk) 및 신용위험(Credit Risk)을 국가가 인수**하여 무역거래의 안정과 국민경제 발전에 기여하는 **정책성 보험**을 말함

### 2) 핵심 비교 — 무역보험 vs 해상보험

| 구분 | 무역보험 (K-SURE) | 해상보험 (민간보험) |
|---|---|---|
| **법적 근거** | 무역보험법 | 영국 MIA 1906 / 상법 보험편 |
| **운영 주체** | 한국무역보험공사(K-SURE) — 정책기관 | 민간보험사 (삼성화재 등) |
| **담보 위험** | 비상위험(전쟁·수입금지 등) + 신용위험(바이어 파산 등) | 물리적 손해위험 (침몰·화재·도난 등) |
| **보험 목적물** | 수출대금 채권, 투자금, 대출채권 | 운송 중의 화물 (물품 자체) |
| **피보험이익** | 수출대금 회수불능에 대한 경제적 이익 | 화물의 안전 도착에 대한 경제적 이익 |
| **보험사고** | 바이어 파산, 수입금지, 전쟁, 환거래 제한 등 | 해상 고유의 위험(침몰, 좌초, 화재 등) |
| **보상 시기** | 손실 확정 후 → 통상 **2년 이상** 소요 | 사고 후 비교적 신속 보상 |
| **보험료 성격** | 정책적 수준 (시장가보다 저렴) | 시장 원리 |
| **강제성** | 임의 (일부 수출금융 시 담보로 요구) | 임의 (CIF·CIP 조건 시 매도인 의무) |

### 3) 무역보험의 2대 위험

**① 비상위험 (Political Risk)**
- 수입국의 **전쟁·내란·혁명** 등 정치적 사변
- 수입국 정부의 **수입금지·외환거래 제한·수입허가 취소**
- 수입국의 **채무불이행 선언** 또는 모라토리엄
- 수입국의 **천재지변** (지진, 홍수 등)

**② 신용위험 (Credit Risk)**
- 바이어(수입자)의 **파산·지급불능·지급거절**
- 바이어의 **계약 일방적 파기**
- 바이어의 **물품 인수 거부** (부당한 경우)

## 2. 한국무역보험공사 (K-SURE) 개요

### 1) 설립 근거 및 목적 (무역보험법 제37조)

- **설립 근거**: 무역보험법에 의해 설립된 **특수법인**
- **주무부처**: 산업통상자원부
- **설립 목적**: 무역·해외투자·해외자원개발 등 대외거래에서 발생하는 위험 인수 → 국가 수출 진흥 및 대외거래 지원

### 2) K-SURE의 재원

1. **정부 출연금** (기획재정부·산업통상자원부)
2. **보험료 수입**
3. **운용 수익금**
4. **차입금** (재정융자특별회계, 수출입은행 등)

## 3. 무역보험의 종류 — 전체 체계도

```
무역보험
├── [수출보험]
│   ├── 단기수출보험 ← 핵심 ★★
│   │   ├── (선적전) 단기수출보험
│   │   └── (선적후) 단기수출보험
│   ├── 중장기수출보험
│   ├── 수출어음보험
│   ├── 농수산물패키지보험
│   └── 서비스종합보험
├── [수출보증]
│   ├── 수출신용보증(선적전)
│   └── 수출신용보증(선적후) / 해외공사이행보증
├── [수입보험]
│   └── 수입보험 (Buyer's Credit 등)
├── [해외투자보험]
│   └── 해외투자보험
├── [환변동보험] ← 핵심 ★★
│   ├── 수출환변동보험
│   └── 수입환변동보험
└── [기타]
    └── 이자율변동보험, 해외자원개발펀드보험 등
```

## 4. 단기수출보험 (Short-term Export Insurance)

### 1) 정의 및 대상

**정의**: 수출대금의 결제기간이 **2년 이하**인 수출거래에서 발생하는 비상위험·신용위험을 담보하는 보험

**대상 결제방식**: D/P, D/A, O/A(Open Account), 분할 L/C 등

### 2) 종류

| 구분 | 선적전 단기수출보험 | 선적후 단기수출보험 |
|---|---|---|
| **보험 시기** | 수출계약 체결 후 ~ 선적 전 | 선적 후 ~ 대금 수령 전 |
| **보험사고** | ① 수입자의 계약 파기 ② 비상위험 발생 | ① 수입자 파산·지급불능 ② 비상위험 발생 |
| **부보비율** | 손실액의 **95%** 이내 | 손실액의 **95%** 이내 |
| **주요 기능** | 생산 단계의 자금 회수 위험 방지 | 수출대금 회수 불능 위험 방지 |

### 3) 담보위험 (무역보험법 제37조의2)

**비상위험**:
1. 수입국의 **전쟁·내란·혁명**
2. 수입국의 **외환거래 금지·제한** 또는 송금 중단
3. 수입국 정부의 **수입금지·수입허가 취소**
4. 수입국에서의 **천재지변**으로 인한 계약 이행 불능
5. **수입국 이외 제3국**의 사정으로 결제 불능

**신용위험**:
1. 수입자의 **파산·화의·지급불능** 선언
2. 수입자의 **물품 인수 거절** (정당한 사유 없이)
3. 수입자의 **대금 지급 거절** (4개월 이상)
4. 수입자에 의한 **계약의 일방적 파기**

### 4) 면책사유 (무역보험법 제39조)

1. 수출자의 **고의 또는 중대한 과실**로 인한 손실
2. 수출자가 **수출계약의 이행을 지체·불이행**하여 발생한 손실
3. **물품의 하자·품질 불량**으로 인한 수입자의 수령 거부
4. **신용장 조건 불일치** 등 수출자 귀책에 의한 대금 미결제
5. **가격 하락·수요 감소** 등 순수 상업적 위험
6. 보험계약자의 **고지의무 위반** (중요 사항 미고지·허위 고지)

## 5. 중장기수출보험 (Medium & Long-term Export Insurance)

### 1) 정의 및 대상

**정의**: 결제기간이 **2년 초과**인 플랜트·선박·방산물자 등 자본재 수출거래에서 발생하는 위험 담보

**대상**: 플랜트 수출, 선박 수출, 대형 인프라 프로젝트, 방산물자 수출

### 2) 구조

| 보험 유형 | 내용 |
|---|---|
| **공급자신용보험** | 수출자(공급자)가 직접 신용을 공여하고 수출대금을 분할 회수하는 방식 담보 |
| **구매자신용보험** | 국내 은행이 해외 수입자에게 융자 제공 시, 은행의 대출채권 담보 |
| **은행융자보험** | 국내 은행이 수출자에게 융자 시 은행의 채권 담보 |

### 3) 특징 (단기수출보험과 비교)

| 구분 | 단기수출보험 | 중장기수출보험 |
|---|---|---|
| 결제기간 | **2년 이하** | **2년 초과** |
| 대상 품목 | 일반 소비재·원자재 | 플랜트·선박·방산·자본재 |
| 부보비율 | 95% 이내 | **85~98%** (개별 협의) |
| 보험료율 | 상대적 저율 | 위험도에 따라 고율 |
| OECD 규율 | 미해당 | **OECD 공식지원 가이드라인** 적용 |

## 6. 수출신용보증 (Export Credit Guarantee)

### 1) 정의

수출자(기업)가 **수출금융**을 위해 금융기관으로부터 대출받을 때, K-SURE가 보증인이 되어 대출금 상환 불능 시 대위변제를 약속하는 보증

### 2) 종류

| 구분 | 내용 |
|---|---|
| **선적전 수출신용보증** | 생산·제조 자금 조달을 위한 은행 대출에 대한 보증 |
| **선적후 수출신용보증** | 수출어음 매입·할인을 위한 은행 대출에 대한 보증 |
| **외화대출보증** | 수출기업의 외화 대출에 대한 보증 |

### 3) 보증 구조

```
수출자 ──────────────── 금융기관
   │  (보증 신청)        │  (대출 실행)
   ▼                    │
K-SURE ←────────────────┘
   (보증서 발급 → 채무불이행 시 대위변제)
```

## 7. 환변동보험 (Exchange Rate Fluctuation Insurance)

### 1) 정의

수출입거래에서 **환율 변동**으로 인해 발생하는 손실을 보상하는 정책보험. K-SURE가 **약정환율**을 제공하여 수출기업의 환위험 헷지 수단으로 기능

### 2) 종류 및 구조

| 구분 | 수출환변동보험 | 수입환변동보험 |
|---|---|---|
| **이용자** | 수출기업 | 수입기업 |
| **환율 방향** | 수출 후 **환율 하락** 시 손실 보전 | 수입 후 **환율 상승** 시 손실 보전 |
| **약정환율** | 보험 체결 시 K-SURE가 제시한 기준 환율 | 좌동 |
| **보상** | 실제 환율 < 약정환율 → 차액 보상 | 실제 환율 > 약정환율 → 차액 보상 |
| **이익 반환** | 실제 환율 > 약정환율 → 차액을 K-SURE에 납입 | 실제 환율 < 약정환율 → 차액 K-SURE에 납입 |

> ⚠️ **핵심**: 환변동보험은 **양방향** 구조 — 이익 발생 시에는 반대로 K-SURE에 납입 (순수 보험이 아닌 **위험 헷징 계약**의 성격)

### 3) 환변동보험 vs 선물환 비교

| 구분 | 환변동보험 | 선물환 계약 |
|---|---|---|
| **취급기관** | K-SURE (정책기관) | 외국환은행 (시중은행) |
| **법적 성격** | 보험 계약 | 금융 계약 |
| **중도 해지** | 가능 (위약금 발생) | 원칙적 불가 |
| **이익 반환** | 있음 | 없음 (정산으로 처리) |
| **대상** | 중소기업 중심 정책 지원 | 일반 기업 |
| **보험료** | 있음 | 없음 (환율 차이로 내재) |

## 8. 해외투자보험 (Overseas Investment Insurance)

### 1) 정의 (무역보험법 제2조)

거주자가 **외국법인에 출자·대부**하거나 **해외 부동산을 취득**하는 등 해외투자를 한 경우, **비상위험**으로 인해 투자 원금 또는 과실을 회수할 수 없게 됨으로써 입은 손실을 보상하는 보험

### 2) 담보위험 (비상위험 한정)

1. 투자 대상국의 **전쟁·내란·혁명** 등
2. 투자 대상국 정부의 **수용·국유화·몰수**
3. **외환거래 제한**으로 인한 과실·원금의 국내 송금 불능
4. **계약의 일방적 파기** (투자 대상국 정부 기관 상대 계약)

> ⚠️ **신용위험은 담보하지 않음** — 해외투자보험의 가장 중요한 제한

### 3) 피보험자 및 보험 목적

| 항목 | 내용 |
|---|---|
| **피보험자** | 해외직접투자를 한 **거주자** |
| **보험 목적** | 투자 원금 및 과실(배당·이자) |
| **부보비율** | 투자금액의 **90%** 이내 |
| **보험기간** | 최장 **20년** (프로젝트 기간에 연동) |

## 9. 수입보험 (Import Insurance)

### 1) 정의

국내 수입자(기업)가 **선급금을 지급**하고 수입하는 경우, 수출자(외국)의 **계약 불이행·파산** 등으로 선급금을 반환받지 못할 위험을 담보하는 보험

### 2) 담보 구조

| 구분 | 내용 |
|---|---|
| **보험계약자** | 국내 수입자(Buyer) |
| **보험사고** | ① 수출자(외국)의 파산 ② 수출자의 수출계약 불이행 ③ 비상위험 발생 |
| **보상 내용** | 선급금 반환 불능 손실액의 **95% 이내** |
| **주요 활용** | 원자재·에너지 장기 공급계약, 선박·플랜트 선급금 |

## 10. 무역보험 주요 용어 정리

| 용어 | 정의 |
|---|---|
| **부보비율 (Coverage Ratio)** | 손실액 중 보험으로 담보되는 비율 (단기수출보험: 원칙 95%) |
| **자기부담비율 (Deductible)** | 피보험자가 스스로 부담하는 손실 비율 (100% - 부보비율) |
| **약정환율** | 환변동보험 체결 시 K-SURE가 고시하는 기준 환율 |
| **대위변제** | K-SURE가 보험금 지급 후 피보험자의 수입자에 대한 채권을 대위 취득 |
| **구상권 (Subrogation)** | 보험금 지급 후 K-SURE가 수입자에게 구상권을 행사하는 권리 |
| **OECD 공적수출신용 가이드라인** | 중장기수출보험 운용 시 준수해야 하는 국제규범 |
| **모라토리엄 (Moratorium)** | 수입국 정부의 대외채무 상환 일시 정지 선언 — 비상위험 사유 |

## 11. 무역보험 vs 해상보험 — 핵심 비교표 (총정리)

| 비교 항목 | 무역보험 | 해상보험 |
|---|---|---|
| 근거 법률 | **무역보험법** | 영국 **MIA 1906** / 상법 제3편 |
| 운영 주체 | **K-SURE** (국가 정책기관) | **민간 보험사** |
| 담보 위험 | 비상위험 + 신용위험 | 해상 고유 위험 (물리적 손해) |
| 보험 목적 | **수출대금 채권** (금전 채권) | **운송 중 화물** (유체물) |
| 피보험이익 | 수출대금 회수 가능성 | 화물 안전 도착 가능성 |
| 보험계약자 | 수출자·금융기관·투자자 | 화물 이해관계자 (매도인·매수인) |
| 보험사고 | 지급불능·계약파기·수입금지 | 침몰·좌초·화재·투하 등 |
| 보상 속도 | **느림** (손실 확정까지 장기간) | **비교적 신속** |
| 국제규범 | OECD 공적수출신용 가이드라인 | **ICC(A/B/C)**, MIA, Incoterms |
| Incoterms 연동 | 없음 | CIF·CIP — 매도인 보험 부보 의무 |

## 🧠 암기 속성 정리

| 주제 | 항목 수 | 핵심 키워드 |
|---|---|---|
| 무역보험 2대 위험 | **2가지** | 비상위험(Political), 신용위험(Credit) |
| 비상위험 사례 | **5가지** | 전쟁·내란, 외환제한, 수입금지, 천재지변, 제3국사정 |
| 신용위험 사례 | **4가지** | 파산, 인수거절, 지급거절, 계약파기 |
| 단기수출보험 면책사유 | **6가지** | 고의·중과실, 이행지체, 하자, L/C불일치, 상업위험, 고지의무위반 |
| 해외투자보험 담보위험 | **4가지** | 전쟁·내란, 수용·국유화, 외환제한, 계약파기 |
| 무역보험 종류 | **6가지** | 단기수출·중장기수출·수출신용보증·수입보험·해외투자·환변동 |
| 환변동보험 특징 | 양방향 | 환율 불리 → 보상, 유리 → K-SURE 납입 |
