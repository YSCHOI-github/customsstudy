# PART 3-1 — 이행법리
**등급: 🟠A | 출제빈도: 7/13년 | 평균배점: 28점**

> 📌 **2026 출제 예측**: CISG 위험이전(제67~70조) + Incoterms 2020 비용분담(A9/B9) 복합 소물음 유력. 2023년 CPT A9/B9 비용 분설 출제 → 인도·비용·위험 연계 심화 예상

---

## 1. 인도(Delivery) — CISG 제31~34조

### 1-1. 인도 장소 (CISG 제31조)

**① 요건/정의**

운송 수반 계약의 경우:
1. 운송인에게 교부함으로써 인도 완료
2. 특정 장소에서 인도 시 → 해당 장소에서 교부
3. 무역계약상 특정 장소 미지정 시 → 최초 운송인에게 교부

운송 비수반 계약의 경우:
1. 특정물 또는 특정 재고에서 인출해야 할 경우 → 물품 소재지
2. 기타의 경우 → 계약 체결 시 매도인의 영업소

**② 예외**

- 당사자 간 합의로 인도 장소 변경 가능함
- 계약 체결 후 매도인 영업소 변경 시 → 변경으로 인한 추가비용은 매도인 부담 (제31조 단서)

**③ 법적 효과**

- 인도 완료 시점 = **위험이전 시점** (제67조와 연동됨)
- 인도 지체 시 → 손해배상청구권 발생 (제61조)

---

### 1-2. 인도 시기 (CISG 제33조) ★ 2025년 제42회 직접 출제

**① 요건/정의**

매도인의 인도 시기 의무:

| 구분 | 인도 시기 |
|---|---|
| 기일이 정해진 경우 | 해당 기일 |
| 기간이 정해진 경우 | 그 기간 내 어느 날 (선택권: 매도인) |
| 기일·기간 미정의 경우 | 계약 체결 후 합리적인 기간 내 |

**② 법적 효과**

- 인도 기일 미준수 → 본질적 위반(Fundamental Breach) 가능성 → 계약해제권 발생 (제49조)

---

### 1-3. 서류 인도 (CISG 제34조)

**① 요건/정의**

1. 매도인이 물품 관련 서류 인도 의무 부담하는 경우 → 계약에서 정한 시기·장소·형식에 따라 인도
2. 인도 기일 이전에 서류 인도 시 → 매수인에게 손해를 야기하지 않는 한 인도 시까지 하자 치유 가능함

**② 예외**

- 하자 치유 시 매수인의 손해배상청구권은 소멸하지 않음

---

## 2. 위험이전(Risk Transfer) — CISG 제66~70조

> ⚠️ **★★ 핵심 반복 출제**: 2015년(제32회, 50점), 2017년(제34회, 50점) 동일 주제 출제. 2026 재출제 유력

### 2-1. 위험이전의 효과 (CISG 제66조)

**① 요건/정의**

- 위험이전 **후** 물품 멸실·훼손 → 매수인의 대금지급의무 소멸하지 않음
- 단, 매도인의 작위·부작위로 인한 멸실·훼손은 예외

**② 법적 효과**

- 위험이전 후 → 매수인은 대금 전액 지급 의무 부담함
- 위험이전 전 → 매도인이 위험 부담함 (매수인 대금 감액 또는 해제 가능)

---

### 2-2. 운송 수반 계약에서의 위험이전 (CISG 제67조)

**① 요건/정의**

**[원칙]** 매도인이 특정 장소에서 운송인에게 교부를 요구받지 않는 경우:
- 위험 = **최초 운송인에게 교부한 시점**에 이전

**[예외 1]** 매도인이 특정 장소에서 교부를 요구받는 경우:
- 위험 = **해당 장소에서 운송인에게 교부한 시점**에 이전

**② 중요 조건 — 물품의 특정화**

- 화물운송장(B/L)·화물인도지시서 등에 물품 명시, 매수인에게 통지, 또는 기타 방법으로 **물품이 특정화된 때까지는 위험이전 불가함** (제67조 제2항)

**③ 예외 — 매도인의 본질적 위반 시** (제70조)

- 매도인이 본질적 위반을 한 경우 → 위 위험이전 규정은 매수인의 구제수단(해제·대체물 청구)에 영향을 미치지 아니함

---

### 2-3. 운송 중 매도 물품의 위험이전 (CISG 제68조)

**① 요건/정의**

- 원칙: 계약 체결 시점에 위험이전
- 예외: 계약 체결 당시 운송인이 멸실·훼손 사실을 알았거나 알았어야 했고 매도인에게 이를 공개하지 않은 경우 → **위험은 매도인에게 귀속**됨

**② 중요 쟁점 — 보험증권(Insurance Policy) 연동**

- 상황에 따라 물품을 운송인에게 교부한 때로 소급하여 이전 가능함
- 보험증권이 운송 중 멸실을 커버하는 경우 → 매수인이 위험 부담함

---

### 2-4. 기타 경우의 위험이전 (CISG 제69조)

**① 요건/정의**

| 상황 | 위험이전 시점 |
|---|---|
| 매수인이 물품을 인수하는 경우 | 물품을 인수한 때 |
| 매수인이 제때 물품 인수하지 않는 경우 | **물품이 매수인의 처분에 맡겨진 때** (인도기일 도래 + 불인수) |
| 매도인 영업소 외 장소에서 인도하는 경우 | 인도 기일 도래 + 매수인이 그 장소에 있음을 인식한 때 |

**② 물품의 특정화 요건**

- 제67조와 동일: 물품이 특정화되지 않은 경우 위험이전 불가함 (제69조 제3항)

## 3. CISG vs. Incoterms 2020 — 위험이전 비교표

> ⚠️ **★★ 반드시 암기**: 2015·2017년 50점 출제 논제의 핵심 쟁점

| 구분 | CISG | Incoterms 2020 |
|---|---|---|
| **법적 성격** | 국제조약 (자동 적용, 배제 가능) | 임의규칙 (계약으로 편입 시 적용) |
| **적용 우선순위** | 당사자 합의·관습·관례 → CISG (제9조) | CISG보다 우선 적용 (CISG 제9조 근거) |
| **위험이전 기준** | 물품의 교부(Delivery) 시점 | 정형거래조건별 인도 지점 |
| **운송 수반 계약** | 최초 운송인 교부 시 (제67조) | FOB: 본선 선적 완료 시 / FCA: 운송인 인도 시 등 |
| **물품 특정화 요건** | 명시적 요건 있음 (제67조 제2항, 제69조 제3항) | 별도 규정 없음 (묵시적 전제) |
| **본질적 위반과의 관계** | 본질적 위반 시 구제수단 보장 (제70조) | 별도 규정 없음 |
| **운송 중 매도** | 제68조 별도 규정 (계약체결 시 또는 소급) | 별도 규정 없음 |
| **불인수에 따른 위험** | 인도기일 도래 + 처분 가능 상태 시 이전 (제69조) | 별도 규정 없음 |
| **유사점** | 물품의 교부를 위험이전 기준으로 삼음 | — |
| **차이점** | 추상적·보충적 규정 | 구체적·정형화된 인도 지점 |

---

## 4. 비용분담(Cost Allocation) — Incoterms 2020 A9/B9

> 📌 2023년 제40회 CPT 규칙 A9/B9 비용 각 5가지 출제 (10점)

### 4-1. 비용분담의 일반 원칙

**① 요건/정의**

- 인도 완료 전 비용 → **매도인 부담**
- 인도 완료 후 비용 → **매수인 부담**
- 단, 정형거래조건별로 예외 다수 존재함 (특히 C-Terms)

---

### 4-2. 주요 정형거래조건별 비용분담 비교표

| 구분 | 매도인 부담 비용 | 매수인 부담 비용 |
|---|---|---|
| **EXW** | 물품 준비비용만 | 공장 이후 전 비용 |
| **FCA** | 지정인도장소까지 운송·수출통관비 | 수입통관·이후 전 비용 |
| **CPT** | 지정목적지까지 운송비 | 양하 후 비용 + 수입통관 |
| **CIP** | 운송비 + 보험료(ICC A 최소) | 양하 후 비용 + 수입통관 |
| **DAP** | 지정목적지 도착까지 전 비용 | 양하비 + 수입통관 |
| **DPU** | 지정목적지 양하까지 전 비용 | 수입통관 + 이후 비용 |
| **DDP** | 수입통관·관세 포함 전 비용 | 양하 후 비용만 |
| **FOB** | 본선 선적까지 전 비용 | 운임·보험료 + 이후 전 비용 |
| **CFR** | 운임 포함 지정목적항까지 | 양하비 + 보험료 + 수입통관 |
| **CIF** | 운임 + 보험료(ICC C 최소) | 양하비 + 수입통관 |

---

### 4-3. CPT A9/B9 비용 목록 ★ 2023년 제40회 출제

**매도인(A9) 부담 비용 — 5가지**

1. 계약에 따른 지정목적지까지의 운송에 관련된 모든 비용
2. 해당하는 경우 목적지에서의 양하비 (운송계약상 매도인 부담분)
3. 수출통관 비용 (관세·세금·기타 공과금 포함)
4. 제3국 통과에 필요한 통관 비용
5. 매수인에게 통지 비용

**매수인(B9) 부담 비용 — 5가지**

1. 인도 시점 이후 물품에 관한 모든 비용
2. 목적지에서의 양하비 (운송계약상 매수인 부담분)
3. 수입통관 비용 (관세·세금·기타 공과금 포함)
4. 매도인의 협조 요청에 따른 서류·정보 취득 비용
5. 물품 수령 통지 비용

## 5. 대금지급(Payment) — CISG 제54~59조

### 5-1. 대금지급의무의 내용 (CISG 제54~56조)

**① 요건/정의**

1. **CISG 제54조**: 매수인의 대금지급의무 = 계약·협약에 따른 조치 이행 + 절차 준수 의무 포함 (예: L/C 개설 의무)
2. **CISG 제55조**: 대금 미정 계약 → 계약 체결 시 해당 거래 분야 통상적으로 부과되는 가격 적용함
3. **CISG 제56조**: 중량 기준 대금 → 불명확 시 **순중량(Net Weight)** 기준으로 결정됨

---

### 5-2. 대금지급의 장소 (CISG 제57조)

**① 요건/정의**

| 상황 | 지급 장소 |
|---|---|
| 기타 특정 장소 미정 | **매도인의 영업소** |
| 물품 또는 서류 교부와 교환하는 경우 | 물품·서류 교부 장소 |

**② 예외**

- 계약 체결 후 매도인 영업소 변경 시 → 변경으로 인한 추가비용은 **매도인 부담**

---

### 5-3. 대금지급의 시기 (CISG 제58조)

**① 요건/정의**

1. 지급기일 미정 시 → 매도인이 계약에 따라 물품 또는 서류를 매수인 처분에 맡긴 때 지급
2. 운송 수반 계약 → 대금지급과 교환하는 조건으로 서류 발송 가능함
3. 매수인의 **물품 검사권**: 지급 전 물품 검사 가능 (단, 검사 절차와 모순되는 인도·지급조건 별도 약정 가능)

## 🧠 암기 속성 정리

**CISG 위험이전 4가지 경우 → ①②③④**

1. 운송 수반 계약 (제67조) → 최초 운송인 교부 시 (특정화 요건 병행)
2. 운송 중 매도 (제68조) → 계약 체결 시 (또는 소급)
3. 기타 인수 경우 (제69조) → 물품 인수 시 / 처분 가능 상태 시
4. 본질적 위반 시 (제70조) → 위험이전 불구 구제수단 유지

**CISG 인도 장소 3가지 → ①②③**

1. 운송 수반 + 특정 장소 없음 → 최초 운송인 교부
2. 운송 수반 + 특정 장소 있음 → 해당 장소에서 교부
3. 운송 비수반 → 물품 소재지 (또는 매도인 영업소)

**CISG 대금지급 장소 → 매도인 영업소 (원칙) / 물품·서류 교부 장소 (예외)**

**C-Terms 비용분담 핵심 암기**

- CPT/CFR: 운임 → 매도인 / 보험료 → 매수인
- CIP/CIF: 운임 + 보험료 → 매도인 (CIP는 ICC A, CIF는 ICC C 최소)
