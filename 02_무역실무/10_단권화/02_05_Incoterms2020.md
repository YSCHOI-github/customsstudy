## 1. Incoterms® 2020 소개문(Introduction) — 핵심

| **A2/B2** | 인도 / 인도수령 | Delivery / Taking delivery |
| **A3/B3** | 위험의 이전 | Transfer of risks |
| **A4/B4** | 운송 | Carriage |

### 1-2. Incoterms® 2020 소개문(Introduction 7) — 다루지 않는 사항 8가지 ★★★
*(2024·41회 직격 출제)*

1. 매매계약의 체결 여부 (Whether a contract of sale has been concluded)
2. 매매되는 물품의 명세 (Specifications of the goods sold)
3. 지급의 시기·장소·방법·통화 (Time, place, method and currency of payment)
4. 계약위반에 대해 청구 가능한 구제수단 (Remedies available for breach of contract)
5. 대부분의 규칙의 예외 (Most exceptions to the rules, e.g., force majeure)
6. 수입관세·세금·부과금의 부과 (Imposition of import duties, taxes or levies)
7. 수출 및 수입 금지 (Export and import prohibitions)
8. 물품에 대한 소유권·소유권 이전 (Transfer of title or ownership of the goods sold)

---

### 1-3. Incoterms® 규칙이 하지 않는 역할 10가지 ★★★
*(2020·37회 직격 출제 — "What the Incoterms® rules do NOT do")*

1. Incoterms® 규칙은 물품에 대한 **소유권이나 소유권 이전**을 규율하지 않음
2. **완전한 매매계약**을 구성하지 않음 (단지 매매계약의 한 조건)
3. **지급 조건** (시기·장소·방법·통화)을 결정하지 않음
4. **계약위반에 대한 구제수단**을 규정하지 않음
5. **불가항력(Force Majeure)**을 다루지 않음
6. **지식재산권**을 다루지 않음
7. **수출입 금지·제한**에 대해 다루지 않음
8. **수입관세·조세·부과금** 부과를 결정하지 않음
9. **준거법(Governing Law)**을 결정하지 않음
10. **제재(Sanctions)** 조치를 다루지 않음

### 1-4. Incoterms® 2020 주요 변경사항 5가지 (소개문 62) ★★★
*(2023·40회 직격 출제)*

1. **FCA에서 본선적재표시 선하증권(On-board B/L) 허용**
   - 구(2010): FCA 조건에서 본선적재 B/L 발급 불가
   - 신(2020): 매수인이 운송인에게 지시 → 매도인에게 본선적재 B/L 발급 가능 (FCA B6)

2. **CIP와 CIF의 보험담보 수준 이원화**
   - CIF: ICC(C) 이상 (종전과 동일)
   - CIP: **ICC(A) 이상** (담보 수준 상향, 신규 변경)

3. **DAT → DPU로 명칭 변경**
   - Delivered at Terminal → **Delivered at Place Unloaded** (양하된 장소)
   - 적용 범위 확대: 터미널 외 모든 양하 장소에서 인도 가능

4. **FCA·DAP·DPU·DDP에서 자체 운송수단 사용 허용**
   - 구(2010): 제3자 운송인을 통한 인도만 규정
   - 신(2020): 당사자 자신의 운송수단으로 인도 가능 명시

5. **보안관련 요건(Security requirements)을 A4·A7에 편입**
   - 운송 및 통관 의무 조항 내 보안요건 명시적 포함

## 4. F Terms — FCA, FAS, FOB

### 4-1. FCA (Free Carrier) ★★★

#### ① 인도와 위험 (FCA Explanatory Notes — 2023·40회 출제)

**[인도장소별 구분]**

| 인도장소 | 인도 방법 | 위험이전 시점 |
|---|---|---|
| 매도인 영업소 | 매도인의 운송수단에 적재 완료 시 | 매도인이 적재 완료한 때 |
| 기타 장소 | 매도인의 운송수단에서 양하 준비 완료 시 | 매수인의 지정 운송인에게 인도된 때 |

- 위험은 **운송인에게 인도된 때** 매수인에게 이전됨
- '운송인'이란 운송계약에 의하여 철도·도로·항공·해상·내수로 또는 복합운송으로 운송을 인수한 자

#### ② FCA B6 — 매수인의 운송서류 의무 (2023·40회 출제) ★★★

> **조항 내용**: 매수인은 합의된 경우, 운송인에게 본선적재표시(on-board notation)가 있는 운송서류(예: 유통 가능한 선하증권)를 매도인에게 발급하도록 지시할 것을 요구해야 함

**배경**: 신용장(L/C) 조건 하 FCA 사용 시 발생하는 문제 해결
- 신용장은 통상 **본선적재 선하증권(On-board B/L)** 요구
- FCA에서는 매도인이 운송계약 체결 주체가 아니므로 B/L 취득 불가
- **해결책(Incoterms 2020 신설)**: 매수인이 운송인에게 지시 → 매도인에게 on-board B/L 발급

**실무 흐름**:
1. 매도인이 운송인(CY)에게 물품 인도
2. 매수인이 운송인에게 "매도인에게 on-board B/L 발급" 지시
3. 운송인이 매도인에게 on-board B/L 발급
4. 매도인이 L/C 조건에 따라 B/L 은행 제시

## 5. C Terms — CPT, CIP, CFR, CIF

### C Terms 공통 특징 (중요!)

> **비용분기점 ≠ 위험분기점**
> - 매도인은 목적지까지 **운임 부담**하지만
> - 위험은 **출발지** (운송인 인도 시 or 선적 시)에서 매수인에게 이전됨

---

### 5-1. CPT (Carriage Paid To) ★★★

#### ① CPT A9 — 매도인이 부담하는 비용 (2023·40회 출제, 5가지)

1. 지정인도장소까지의 운송에 관한 **모든 비용** (운임 포함)
2. 지정목적지까지 계약된 운송에 필요한 **비용** (환적 포함)
3. **수출통관비용** (수출관세, 세금, 기타 공과금 포함)
4. **선적 전 검사비용** (수출국 당국이 강제한 경우)
5. **물품점검비용** (품질·규격·수량·중량 점검 등)

> ⚠️ 포장비, 화인(Marking)비도 원칙적으로 매도인 부담

#### ② CPT B9 — 매수인이 부담하는 비용 (2023·40회 출제, 5가지)

1. 물품이 운송인에게 인도된 이후 **모든 비용**
2. 지정목적지에서의 **양하 비용** (운임에 포함되지 않은 경우)
3. **수입통관비용** (수입관세, 세금, 기타 공과금)
4. 합의된 목적지를 벗어나 발생하는 **추가 운송비용**
5. 지정목적지에서 **통지된 시간 내 물품 수령 미이행** 시 발생하는 추가 비용

---

### 5-2. CIP (Carriage and Insurance Paid To)

- CPT에 **보험(Insurance)** 의무가 추가된 규칙
- **보험조건**: Incoterms® 2020 신규 — **ICC(A) 또는 동등한 약관 이상** (전위험 담보)
- 매도인이 매수인을 위해 보험 계약 체결 의무
- 보험금액: 최소 계약가격의 **110%**

---

### 5-3. CFR (Cost and Freight)

| 항목 | 내용 |
|---|---|
| **인도** | 지정 선적항에서 본선 상에 적재 |
| **위험이전** | 물품이 본선 상에 적재된 때 |
| **운임** | 매도인 부담 (지정목적항까지) |
| **보험** | 의무 없음 |

### 5-4. CIF (Cost, Insurance and Freight) ★★★

#### ① 기본 구조

| 항목 | 내용 |
|---|---|
| **인도** | 지정 선적항에서 본선 상에 적재 |
| **위험이전** | 물품이 본선 상에 적재된 때 |
| **운임** | 매도인 부담 (지정목적항까지) |
| **보험** | **매도인** 의무 — 최소 **ICC(C)** 이상, 보험금액 계약가격의 **110%** |

#### ② CIF와 소유권 이전 문제 (2013·30회 출제)

**문제점**:
- CIF는 매도인이 운송서류(B/L, 보험서류, 상업송장) 일체를 매수인에게 양도
- 운송 중 위험은 매수인에게 이전되었으나 **소유권 이전 시점이 불명확**
- 매수인이 서류를 수리하기 전까지 물품의 소유권 귀속 분쟁 발생 가능

**대응방안**:
1. 계약서에 **소유권 유보 조항(Retention of Title Clause)** 명시
2. 지시식 B/L(Order B/L) 발급 → B/L 배서·교부로 소유권 이전 명확화
3. **신용장(L/C) 결제** 활용 → 서류 인도와 대금 지급 동시 이행

#### ③ CIF 복수 운송인 문제 ★★★
*(2021·38회 직격 출제)*

**[선적항에 대한 합의가 있는 경우]**

| 항목 | 내용 |
|---|---|
| 복수 운송인이 존재하는 구간 | 지정 선적항 → 지정 목적항 전 구간 |
| **위험이전시기** | 물품이 **지정 선적항**에서 본선에 적재된 때 |

**[선적항에 대한 합의가 없는 경우]**

| 항목 | 내용 |
|---|---|
| 복수 운송인이 존재하는 구간 | 매도인 영업소(또는 기타 장소) → 지정 목적항 전 구간 |
| **위험이전시기** | 물품이 **최초 운송인에게 인도된 때** |
| **위험부담시점 연장방법** | ① 전 구간을 커버하는 적하보험 계약 체결 ② 전 구간 통선하증권(Through B/L) 취득 |

## 6. D Terms — DAP, DPU, DDP

### 6-1. DAP (Delivered at Place)

| 항목 | 내용 |
|---|---|
| **인도** | 지정목적지에서 도착 운송수단에서 **양하 준비 완료** 상태로 인도 |
| **위험이전** | 물품이 목적지에서 양하 준비 완료된 때 |
| **수입통관·관세** | **매수인** 부담 |
| **양하비** | **매수인** 부담 |

---

### 6-2. DPU (Delivered at Place Unloaded) ★ (2020년 신설 명칭)

| 항목 | 내용 |
|---|---|
| **인도** | 지정목적지에서 도착 운송수단에서 **양하 완료 후** 인도 |
| **위험이전** | 물품이 목적지에서 **양하된 때** |
| **구DAT와 차이** | DAT: 터미널만 인도 가능 → DPU: **모든 장소**에서 양하 후 인도 가능 |
| **양하비** | **매도인** 부담 (DAP과의 핵심 차이) |
| **수입통관** | **매수인** 부담 |

> ⚠️ **DAP vs DPU 핵심 차이**: 양하비 부담 주체 — DAP는 매수인, DPU는 **매도인**

---

### 6-3. DDP (Delivered Duty Paid)

| 항목 | 내용 |
|---|---|
| **인도** | 지정목적지에서 도착 운송수단에서 양하 준비 완료 상태로 인도 |
| **위험이전** | 물품이 목적지에서 양하 준비 완료된 때 |
| **수출통관** | **매도인** 부담 |
| **수입통관·관세** | **매도인** 부담 (DDP의 특징) |
| **보험** | 의무 없음 |

> ⚡ **DDP**: 매도인 최대 의무. 수입관세까지 매도인 부담 — VAT 제외 가능하나 명시 필요

## 10. CISG와 Incoterms® 2020의 관계 ★★★

*(2024·41회, 2017·34회, 2015·32회 출제)*

### 10-1. CISG 제9조 — 관습과 관례의 구속력

**① 요건/정의**
- 제9조 제1항: 당사자들이 합의한 **관례(practices)** 와 그들 사이에 확립된 **관행(usages)** 에 구속됨
- 제9조 제2항: 당사자들이 알거나 알았어야 하며 국제무역에서 **광범위하게 알려져** 있고 **정기적으로 준수**되는 관습(usages)은 계약에 묵시적으로 적용됨

**② Incoterms® 2020의 우선 적용 근거 (CISG 제9조)**
- 당사자들이 계약서에 Incoterms® 2020 규칙을 명시적으로 원용한 경우
  → CISG 제9조 제1항에 의해 **합의된 관례**로서 우선 적용됨
- Incoterms® 규칙이 국제무역에서 광범위하게 알려지고 정기적으로 준수되는 경우
  → CISG 제9조 제2항에 의해 **묵시적 관습**으로서 적용될 수 있음

**③ 법적 효과**
- Incoterms® 규칙이 계약에 편입되면 → 매매계약의 일부를 구성
- CISG 조항과 Incoterms® 규칙 간 충돌 시 → **Incoterms® 규칙 우선 적용** (당사자 합의 우선)
- 단, CISG의 강행규정은 배제 불가

### 10-2. CISG vs Incoterms 2020 위험이전 비교 ★★★

*(2017·34회, 2015·32회 각 50점 출제 — 유사점·차이점 논술 필수)*

| 구분 | **CISG** | **Incoterms® 2020** |
|---|---|---|
| **근거** | 제66조~제70조 | A3/B3 (각 규칙별) |
| **원칙** | 물품 인도 시 위험이전 | 인도 시 위험이전 (동일) |
| **운송 중 매매 시** | 제68조: 계약 체결 시 소급 또는 운송인 인도 시 | 규정 없음 |
| **물품 특정** | 물품이 특정된 때부터 위험이전 가능 | 묵시적 전제 |
| **매도인 위반 시** | 제70조: 본질적 위반 시 위험이전 불인정 | 규정 없음 |
| **선적항** | 명시적 규정 없음 | 각 규칙별 명확히 지정 |

**[유사점]**
1. 물품의 인도(Delivery) 시점을 기준으로 위험이 이전됨
2. 물품이 특정(Identification)되어야 위험이전이 가능함
3. 당사자 합의에 의해 변경 가능

**[차이점]**
1. CISG는 운송 중 매매(제68조)에 관한 특별규정 존재 → Incoterms® 에는 없음
2. CISG는 매도인의 본질적 위반 시 위험이전 불인정(제70조) → Incoterms® 에는 없음
3. Incoterms® 는 구체적 인도 지점·방법을 규칙별로 명확히 지정 → CISG는 추상적 원칙만 규정
4. CISG는 매매계약의 준거규범 → Incoterms® 는 당사자 합의로 편입되는 임의 규범

## 🧠 암기 속성 정리

### 【필수 암기 1】 11개 규칙 그룹 & 위험이전 포인트

```
EXW → 매수인 수령 시
FCA → 운송인 인도 시 (★컨테이너 표준)
FAS → 선측 인도 시
FOB/CFR/CIF → 본선 적재 시 (★해상전용)
CPT/CIP → 최초 운송인 인도 시 (★비용은 목적지, 위험은 출발지)
DAP/DDP → 목적지 양하준비 시
DPU → 목적지 양하완료 시
```

### 【필수 암기 2】 보험 의무 조건

```
CIF = ICC(C) + 110%
CIP = ICC(A) + 110%  ← 2020 변경 핵심!
```

### 【필수 암기 3】 통관 예외 2개

```
EXW = 수출도 매수인
DDP = 수입도 매도인
(나머지 9개 = 수출매도인 / 수입매수인)
```

### 【필수 암기 4】 2020 변경사항 5개 약어

```
FCA B/L → CIP 보험UP → DPU 명칭 → 자체운송 → 보안편입
```

### 【필수 암기 5】 A1~A10 명칭 순서

```
일반(General) → 인도(Delivery) → 위험(Risk) → 운송(Carriage) → 보험(Insurance)
→ 서류(Document) → 통관(Clearance) → 점검(Checking) → 비용(Cost) → 통지(Notice)
```

### 【필수 암기 6】 Introduction 7 다루지 않는 사항 8가지 (영문 포함)

```
①계약체결여부 ②물품명세 ③지급조건(시기·장소·방법·통화) ④구제수단
⑤대부분의예외(불가항력포함) ⑥수입관세·세금 ⑦수출입금지 ⑧소유권이전
```
