## 1. 품질조건 (Quality Terms)

### 1-1. 품질결정방법 5종

| 방법 | 내용 | 비고 |
|---|---|---|
| 견본매매 (Sample) | 견본 일치 품질 인도 의무 | 가장 확실, 분쟁多 |
| 표준품매매 (Standard) | FAQ·GMQ 등 표준등급 기준 | 곡물·원면 多 |
| 명세서매매 (Specification/Description) | 계약서 명세서 기준 | 기계류·산업재 |
| 상표매매 (Brand/Trade Mark) | 특정 상표 기준 | 유명브랜드 |
| 규격매매 (Grade) | 공적규격(JIS·KS·ISO) 기준 | 공산품 |

### 1-2. 품질결정시기

1. **선적품질조건 (Shipped Quality Terms)**: 선적 시 품질 기준 — 매도인 유리
2. **양륙품질조건 (Landed Quality Terms)**: 양륙 시 품질 기준 — 매수인 유리

### 1-3. 정지조건 vs 해제조건 비교 ★2018년 제35회 10점 기출

| 구분 | 정지조건 (Condition Precedent) | 해제조건 (Condition Subsequent) |
|---|---|---|
| 개념 | 조건 성취 시 계약 효력 발생 | 조건 성취 시 계약 효력 소멸 |
| 효과 | 조건 성취 전: 권리의무 발생 정지 | 조건 성취 전: 완전 유효, 성취 시 소급 소멸 |
| 예시 | "수입허가 취득 시 계약 발효" | "품질검사 불합격 시 계약 해제" |
| 소급효 | 원칙상 소급효 있음 | 소급효 없음 (장래효) |

## 2. 수량조건 (Quantity Terms) ★★핵심 — M/L Clause 기출

### 2-1. 수량단위

| 단위 | 내용 |
|---|---|
| 중량 (Weight) | 총중량(Gross Weight)·순중량(Net Weight)·법정중량(Legal Weight)·실측중량 |
| 용적 (Measurement) | 입방미터(CBM), 입방피트(CFT) |
| 개수 (Number/Piece) | 개, 세트, 다스(Dozen), 그로스(Gross=144개) |
| 길이 (Length) | 미터, 야드, 피트 |

### 2-2. 수량결정시기

1. **선적수량조건 (Shipped Weight)**: 선적 시 중량 기준 — 매도인 유리
2. **양륙수량조건 (Landed Weight)**: 양륙 시 중량 기준 — 매수인 유리

---

### 2-3. 과부족용인조항 (More or Less Clause; M/L Clause) ★★★기출 2회 (2017·2015)

#### ① 정의 (Definition)
- 수량의 과부족을 일정 범위(%) 내에서 허용하는 조항임
- 원유·곡물·광석 등 산적화물(Bulk Cargo)에서 정확한 수량 계약 이행 곤란 시 활용됨
- 계약서 기재 방식 예시: "5% More or Less at Seller's Option"

#### ② UCP 600 제30조 — M/L Clause에 관한 신용장 해석기준 (조문 직접 적용)

**(a) 신용장 금액·수량·단가의 "약(About/Approximately)" 해석**
- "About" 또는 "Approximately" 표현 사용 시 → **10% 이내** 과부족 허용됨
- 금액·수량·단가 모두 적용됨

**(b) 수량 과부족 — M/L Clause 없는 경우 (UCP 600 제30조(b))**
- M/L Clause 미기재 + 수량이 포장단위 수 또는 개수로 표시되지 않은 경우 → **5% 이내** 과부족 허용
- 단, 신용장 금액을 초과하지 않아야 함

**(c) 금액 과부족 — M/L Clause 없는 경우 (UCP 600 제30조(c))**
- M/L Clause 미기재 + 전량 선적된 경우 → **5% 이내** 금액 부족 허용
- 단, 단가가 감액되지 않아야 함
- 단, 신용장이 분할선적 금지 규정이 없어야 함

#### ③ 핵심 비교 정리

| 구분 | 표현 | 허용범위 |
|---|---|---|
| "About/Approximately" 사용 | 금액·수량·단가 | ±10% |
| M/L Clause 없음 (포장·개수 단위 아닌 경우) | 수량 | ±5% |
| M/L Clause 없음 (전량 선적 완료 후) | 금액 | -5% (감액 허용) |

## 3. 가격조건 (Price Terms)

### 3-1. 가격의 구성요소

```
물품원가(Cost) + 수출포장비 + 국내운송비 + 수출통관비
+ 해상운임(Freight) + 해상보험료(Premium)
+ 수입통관비 + 수입지 내륙운송비
```

→ Incoterms 2020의 조건별로 매도인/매수인 비용부담 분기점 결정됨

### 3-2. 가격표시방법

| 방법 | 내용 |
|---|---|
| 단가 기준 | Per Metric Ton, Per Piece 등 |
| 통화 표시 | USD, EUR, JPY 등 계약통화 명시 |
| Incoterms 연계 | 예: USD 100 per MT CIF Busan, Incoterms® 2020 |

## 4. 선적조건 (Shipment Terms)

### 4-1. 선적시기 표현

| 표현 | 의미 |
|---|---|
| Immediate Shipment | 계약 후 즉시 (통상 30일 이내) |
| Prompt Shipment | 즉시에 준하는 조속 선적 |
| 특정 월 지정 | "Shipment during October" → 해당 월 내 선적 |
| "On or About" | UCP: 명시일 전후 각 5일 이내 |

### 4-2. 분할선적 (Partial Shipment) vs 환적 (Transshipment)

| 구분 | 분할선적 | 환적 |
|---|---|---|
| 개념 | 계약 수량을 수회로 나누어 선적 | 운송 도중 타 선박으로 화물 이환 |
| UCP 600 | 제31조 규정 | 제20조(b) 등 조건별 규정 |
| 원칙 | 신용장에 금지 명시 없으면 허용 | 신용장에 금지 명시 없으면 허용 |

### 4-3. 선적지연 관련

- **면책사유**: 천재지변(Act of God), 파업(Strike), 전쟁(War) 등 불가항력(Force Majeure)
- 불가항력 발생 시 → 통지의무 이행 + 증빙서류 제출 필요

## 5. 결제조건 (Payment Terms)

### 5-1. 주요 결제방식 개요 비교

| 결제방식 | 안전성(수출자) | 안전성(수입자) | 특징 |
|---|---|---|---|
| L/C (신용장) | ★★★ | ★★★ | 은행 지급보증 |
| D/P (지급도) | ★★ | ★★ | 대금지급 후 서류 인도 |
| D/A (인수도) | ★ | ★★★ | 환어음 인수 후 서류 인도 |
| T/T Advance | ★★★ | ★ | 선불 송금 |
| O/A (Open Account) | ★ | ★★★ | 외상거래 |

→ 상세 내용은 **PART 3-7 무역대금의 결제** 챕터에서 집중 서술

## 6. 보험조건 (Insurance Terms)

### 6-1. 부보조건 약칭 및 담보범위

| 약관 | 담보범위 | 특징 |
|---|---|---|
| ICC(A) | 포괄 담보 (All Risks 유사) | 가장 넓은 담보 |
| ICC(B) | 열거주의 + 주요 사고 | 중간 |
| ICC(C) | 열거주의 (최소) | 가장 좁은 담보 |

- CIF·CIP 계약 시 매도인 부보의무 발생
  - **CIF**: 최소 ICC(C) 이상 부보 의무
  - **CIP**: 최소 ICC(A) 이상 부보 의무 (Incoterms® 2020 변경사항)

→ 상세 내용은 **PART 3-4 해상보험** 챕터에서 집중 서술

## 7. 포장조건 (Packing Terms)

### 7-1. 포장의 종류

| 구분 | 내용 |
|---|---|
| 내장 (Inner Packing) | 물품 직접 보호 — 상업적 포장 |
| 외장 (Outer Packing) | 운송·보관용 — 운송포장 |
| 특수포장 | 냉동·위험물·액체 등 특수화물 |

### 7-2. 화인 (Shipping Mark)

- **주화인 (Main Mark)**: 수입자 약칭·목적지·번호
- **부화인 (Counter Mark)**: 수출자 약칭
- **주의 표시 (Care Mark)**: "FRAGILE", "KEEP DRY", "THIS SIDE UP" 등
- **원산지 표시 (Country of Origin Mark)**: "Made in Korea" 등 (대외무역법 의무사항)

## 8. 분쟁해결조건 (Dispute Settlement Terms)

### 8-1. 분쟁해결방법 4종

1. **협상 (Negotiation)**: 당사자 간 자율 해결
2. **조정 (Conciliation/Mediation)**: 제3자 중립 개입
3. **중재 (Arbitration)**: 중재인 판정 — **뉴욕협약으로 집행력 확보** → PART 4 집중
4. **소송 (Litigation)**: 법원 판결 — 비용·시간 多

### 8-2. 중재 vs 소송 비교

| 구분 | 중재 | 소송 |
|---|---|---|
| 종료방식 | 중재판정 (1심제) | 판결 (3심제) |
| 기밀성 | 비공개 원칙 | 공개 원칙 |
| 집행력 | 뉴욕협약 156개국 집행 | 국가별 상호주의 |
| 비용·속도 | 상대적 저비용·신속 | 고비용·장기 |

## 🧠 암기 속성 정리

### M/L Clause UCP 600 제30조 — 3줄 완전 정리

- **"About/Approximately"** 사용 → **±10%** 허용 (금액·수량·단가)
- **M/L Clause 없음 + 포장·개수 단위 아님** → 수량 **±5%** 허용 (신용장 금액 초과 불가)
- **M/L Clause 없음 + 전량 선적** → 금액 **-5%** 허용 (단가 감액 없을 것)

### 품질결정방법 5가지 → 견·표·명·상·규

1. 견(본매매)
2. 표(준품매매)
3. 명(세서매매)
4. 상(표매매)
5. 규(격매매)

### 정지·해제조건 — 2줄 구분

- **정지조건**: 조건 성취 → 효력 **발생** (조건 前 정지 상태)
- **해제조건**: 조건 성취 → 효력 **소멸** (조건 前 완전 유효)
